## よくある質問

### Q1: どのガイドから読むべきですか？

**A:** **ポートフォリオ作成ガイド**から始めてください。

理由：
- ポートフォリオは就職・フリーランス両方で必須
- 統合プロジェクトの活用方法がわかる
- 採用担当者の視点が理解できる

### Q2: 統合プロジェクトが1つしかありません。それでも就職できますか？

**A:** 可能ですが、2-3個あると有利です。

**1個の場合:**
- ジュニアエンジニア（新卒・第二新卒）として応募
- そのプロジェクトを深く説明できるようにする
- 学習意欲をアピール

**2-3個の場合:**
- 中堅ポジションも狙える
- 技術の幅をアピールできる
- 選考通過率が上がる

### Q3: フリーランスとして独立するには何個プロジェクトが必要ですか？

**A:** 最低2個、推奨3個以上です。

**理由:**
- クライアントは実績を重視
- 技術の幅を示せる
- 提案時の説得力が増す

**プロジェクトの種類:**
- フルスタック（React + Node.js）← 必須
- フロントエンド専門（React/Vue）
- バックエンド専門（API/DB）

### Q4: 未経験でもフリーランスになれますか？

**A:** 技術的には可能ですが、推奨しません。

**推奨ルート:**
1. まず就職してスキル・経験を積む（1-2年）
2. 副業で案件を受注して実績を作る
3. 独立の準備（貯金、人脈、案件確保）
4. フリーランスとして独立

**理由:**
- 実務経験がないと案件獲得が困難
- クライアント対応のスキルが必要
- 収入が不安定

### Q5: ポートフォリオサイトは必須ですか？

**A:** 必須ではありませんが、強く推奨します。

**優先順位:**
1. **GitHubプロフィール** ← 必須
2. **プロジェクトのREADME** ← 必須
3. **デモサイト** ← 推奨
4. **ポートフォリオサイト** ← あると有利

まずは1-3を完成させてください。

## 成功事例

### ケース1：未経験からジュニアエンジニアへ（26歳・文系出身）

**学習期間:** 6ヶ月

**作成物:**
- フルスタックタスク管理アプリ（統合プロジェクト）
- Reactポートフォリオサイト

**結果:**
- 応募企業: 30社
- 書類選考通過: 10社
- 面接: 7社
- 内定: 3社
- 年収: ¥3,200,000

**成功要因:**
- ポートフォリオが充実していた
- 技術面接で自信を持って説明できた
- 学習プロセスを詳しく説明

### ケース2：副業から独立（32歳・営業職からの転職）

**学習期間:** 12ヶ月

**作成物:**
- タスク管理アプリ（フルスタック）
- ECサイト（フロントエンド）
- 業務効率化ツール（バックエンド）

**結果:**
- 1年後にフリーランス独立
- 初月売上: ¥300,000
- 6ヶ月後売上: ¥600,000
- 1年後売上: ¥800,000

**成功要因:**
- 就職しながら副業で実績を作った
- 複数プロジェクトでスキルの幅をアピール
- クライアントとの信頼関係構築
