## Quick Reference

```
Regex Cheat Sheet:
  .       -- Any single character
  \d \w \s -- Digit / word character / whitespace
  [abc]   -- Character class
  ^  $    -- Start / end of line
  *  +  ? -- 0 or more / 1 or more / 0 or 1
  {n,m}   -- Between n and m times
  (...)   -- Capture group
  (?:...) -- Non-capturing group
  (?=...) -- Lookahead
  (?<=..) -- Lookbehind
  \1      -- Backreference
```
