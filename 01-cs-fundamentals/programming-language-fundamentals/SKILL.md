# Programming Language Fundamentals

プログラミング言語の本質を理解する包括的ガイド。型システム、メモリモデル、制御構造、関数、並行処理、言語比較まで、言語を「使う」だけでなく「理解して選ぶ」力を身につける。

## Learning Path
1. 導入 → 型システム → メモリモデル（基礎理解）
2. 制御構造 → 関数 → 並行処理（実装力）
3. 言語比較 → 言語の進化（選択力・応用力）
