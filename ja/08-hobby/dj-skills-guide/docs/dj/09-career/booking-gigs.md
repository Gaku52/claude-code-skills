## なぜ積極的なアプローチが必要か

**プロモーターは忙しい:**

```
現実:

プロモーターの元に:
毎週数十人のDJから連絡

受動的に待つ:
埋もれる、見つけてもらえない

積極的にアプローチ:
チャンスが生まれる

「仕事をください」ではなく:
「価値を提供します」の姿勢

行動 = 成功への近道
```

## ギグの種類

### 7つのギグタイプ

**それぞれの特徴:**

```
1. オープンデッキ

内容: 誰でも参加できる DJ イベント
時間: 15-30分
ギャラ: ¥0（無料）
難易度: ★☆☆☆☆

メリット:
- 経験を積める
- プロモーターに見てもらえる
- DJ仲間と交流
- 実践練習

デメリット:
- 無料
- 短時間

おすすめ:
初心者は必ず参加

─────────────────

2. バー/カフェ

内容: 背景音楽DJ
時間: 2-4時間
ギャラ: ¥5,000-15,000
難易度: ★★☆☆☆

メリット:
- 初ギグに最適
- プレッシャー少ない
- 長時間経験

デメリット:
- ギャラ低め
- 観客の反応薄い

選曲:
落ち着いた雰囲気
会話できる音量

─────────────────

3. クラブ（サポート/ウォームアップ）

内容: メインDJ前の時間帯
時間: 1-2時間
ギャラ: ¥10,000-30,000
難易度: ★★★☆☆

メリット:
- クラブ経験
- まだ人が少ない = 失敗しても
- メインDJ の技を学べる

デメリット:
- 観客少ない時間
- 盛り上げすぎNG

選曲:
徐々にエネルギー上げる

─────────────────

4. クラブ（メインタイム）

内容: ピークタイム DJ
時間: 2-3時間
ギャラ: ¥30,000-100,000
難易度: ★★★★☆

メリット:
- 満員のフロア
- 高ギャラ
- 実績になる

デメリット:
- プレッシャー大
- 経験必要

要求:
フロアを盛り上げる力

─────────────────

5. プライベートパーティ

内容: 誕生日、企業イベント等
時間: 2-4時間
ギャラ: ¥20,000-50,000
難易度: ★★☆☆☆

メリット:
- 高ギャラ
- 比較的気楽
- 平日昼間も

デメリット:
- リクエスト対応必須
- 幅広いジャンル必要

準備:
ポップ、定番曲必須

─────────────────

6. ウェディング

内容: 結婚式・披露宴 DJ
時間: 3-6時間
ギャラ: ¥50,000-150,000
難易度: ★★★★☆

メリット:
- 高ギャラ
- 定期的に依頼
- 昼間の仕事

デメリット:
- 専門知識必要
- ミス許されない
- 幅広いジャンル

準備:
ウェディング専用トレーニング

─────────────────

7. フェスティバル

内容: 野外/大型イベント
時間: 1-2時間
ギャラ: ¥50,000-数百万（レベルによる）
難易度: ★★★★★

メリット:
- 大観衆
- 知名度UP
- キャリアの節目

デメリット:
- ブッキング困難
- 実績必須

目標:
中級〜上級者
```

```
  #tokyodj #djevent

  → 投稿チェック
```

**足で稼ぐ:**

```

- ローカルバー
- カフェ
- 小さいクラブ

アプローチ:
「DJ 募集していますか？」

2. イベント参加

毎週末:

やること:
- 観察

3. オープンデッキ

最重要:
毎回参加

効果:
- 実践経験
- プロモーターに見てもらえる
- 他DJと交流
- ギグにつながる可能性大

4. DJ スクール/コミュニティ

イベント:
DJ スクール主催のイベント
→ 生徒が出演機会

ネットワーク:
先生、先輩からの紹介
```

## プロモーターへのアプローチ

### リサーチ

**知ってからアプローチ:**

```
Step 1: ターゲット選定

あなたのジャンルに合う:
□ Techno DJ なら Techno イベント
□ House DJ なら House ナイト

リスト作成:
10-20のイベント/会場

Step 2: 徹底リサーチ

各イベントについて:
□ 主催者は誰？
□ どんな雰囲気？
□ どんなDJ が出ている？
□ どんな音楽？
□ 観客層は？

実際に行く:
最低1回は参加

Step 3: 接点を見つける

SNS:
□ フォロー
□ 投稿にいいね、コメント
□ ストーリーズに反応

イベント:
□ 参加
□ 挨拶
□ 名刺交換

関係構築:
いきなり営業× NG
```

### アプローチメール

**テンプレート:**

```
件名:
「DJ出演のご相談 - DJ GAKU（Techno）」

本文:

「
[イベント名/主催者名] 様

はじめまして。
東京を拠点に Techno DJ として活動している
DJ GAKU と申します。

先日、[会場名] で開催された
[イベント名] に参加させていただき、
素晴らしい雰囲気と音楽に感動いたしました。

特に [メインDJ名] のセットが印象的で、
私も同じような方向性の Techno を
プレイしております。

もし今後、ウォームアップや
サポートDJ の機会がございましたら、
ぜひご検討いただけないでしょうか。

▼ Demo Mix (Techno, 60min)
https://soundcloud.com/djgaku/xxx

▼ Instagram
https://instagram.com/djgaku

▼ EPK (Electronic Press Kit)
https://drive.google.com/xxx

ご多忙中恐れ入りますが、
ご検討のほど、よろしくお願いいたします。

DJ GAKU
Email: dj@example.com
Tel: 090-XXXX-XXXX
」

ポイント:
1. 具体的（イベント名、DJ名）
   → ちゃんと来てる証明

2. 謙虚
   → 「チャンスください」

3. 簡潔（300文字程度）
   → 忙しい人向け

4. リンク明記
   → すぐ確認できる

5. 連絡先
   → 返信しやすく
```

### 対面アプローチ

**イベント後に:**

```
タイミング:
イベント終了後
DJ/プロモーターが片付け中

アプローチ:

「お疲れ様です。
今日のイベント、素晴らしかったです。

私も DJ をやっていて、
いつかこの会場でプレイできたらと
思っています。

もしよければ、デモを聴いて
いただけませんか？」

→ 名刺、USBメモリ渡す

注意:
- 忙しそうなら避ける
- 簡潔に（1-2分）
- しつこくしない
- 笑顔で

印象:
良ければ覚えてもらえる
```

## デモミックス戦略

### 複数パターン用意

**ターゲット別に:**

```
Pattern 1: メインデモ（60分）

目的: あなたの真骨頂
内容: ベストミックス
使用: メインタイム応募

Pattern 2: ウォームアップ（30分）

目的: 徐々に盛り上げる
内容: BPM低め → 徐々に↑
使用: サポートDJ応募

Pattern 3: ピークタイム（30分）

目的: フロアを沸かせる
内容: ハイエナジー
使用: メインタイム応募

Pattern 4: ラウンジ/バー（30分）

目的: 落ち着いた雰囲気
内容: Deep, Melodic
使用: バー/カフェ応募

最低2本:
メイン + ターゲット別1本
```

### デモの送り方

**プロフェッショナルに:**

```
形式:

SoundCloud リンク（推奨）:
- ストリーミング可能
- トラックリスト記載
- すぐ聴ける

Google Drive（オプション）:
- MP3 (320kbps)
- ファイル名: DJ_GAKU_Techno_Mix_60min.mp3
- 共有設定: 誰でも閲覧可

USBメモリ（対面時）:
- MP3
- ファイル名明記
- ラベル貼る

NG:
- 大容量メール添付
- WeTransfer（期限切れる）
- WAV（重すぎる）
```

## オープンデッキ攻略

### 準備

**完璧な準備:**

```
1週間前:

□ セットリスト作成（30分）
  - 15-20曲
  - 順番決定
  - キー分析

□ Hot Cue 設定
  - 全曲イントロ/アウトロ
  - ドロップポイント

□ リハーサル（3回以上）
  - 完璧に通せるまで

3日前:

□ USBメモリ準備
  - Rekordbox でエクスポート
  - バックアップUSB も

□ 服装決定
  - ブランドイメージに合う
  - 動きやすい

前日:

□ 十分な睡眠
□ USBメモリ確認
□ 会場の場所・時間確認
□ イメージトレーニング

当日:

□ 早めに到着（30分前）
□ 受付で登録
□ 会場の雰囲気確認
□ 他DJのプレイ観察
□ 深呼吸、リラックス
```

### プレイ中

**全力で楽しむ:**

```
最初の1分:

強い曲で始める:
観客の注目を集める

アイコンタクト:
フロアを見る、笑顔

10-20分:

エネルギーカーブ:
徐々に盛り上げる

フロアリーディング:
反応を見ながら調整

ミスしても:
止まらない、続ける

最後の5分:

ピークを作る:
印象に残るフィニッシュ

次のDJへ:
スムーズに引き継ぐ

終了後:

挨拶:
プロモーター、他DJ に感謝

片付け:
ケーブル、機材を整理

交流:
名刺配布、SNSフォロー

印象:
プレイ + 人柄が大事
```

## リピートを増やす方法

### 初ギグ後

**次につなげる:**

```
当日:

□ 全力でプレイ
□ 時間厳守
□ トラブル対応も冷静に
□ プロモーターに感謝

翌日:

□ お礼メール送信

「昨日はありがとうございました。
素晴らしい経験をさせていただきました。
またの機会がございましたら、
ぜひよろしくお願いいたします。」

□ SNS で投稿
  「@[会場名] でプレイさせていただきました！
   素晴らしい夜をありがとうございました」
  → タグ付け

1週間後:

□ 録音があれば SoundCloud にアップ
  → タグ付けして共有

1ヶ月後:

□ 軽くフォローアップ
  「その後いかがでしょうか。
   また機会があれば嬉しいです」

3ヶ月後:

□ 新作デモ送付
  「新しいミックスを作りました。
   もしよろしければ聴いてください」

リピート率を上げる:
- プロフェッショナル
- 時間厳守
- トラブルなし
- 良い印象
```

### プロとしての振る舞い

**信頼を築く:**

```
時間厳守:

□ セットアップ: 30分前到着
□ プレイ開始: 時間通り
□ プレイ終了: 時間通り
□ 片付け: 速やかに

遅刻:
二度と呼ばれない

コミュニケーション:

□ メール: 24時間以内返信
□ 確認事項: 事前に質問
□ 変更: すぐ連絡

柔軟性:

□ リクエスト: 可能なら対応
□ 時間変更: 柔軟に
□ トラブル: 冷静に

態度:

□ 謙虚
□ プロフェッショナル
□ 協力的
□ ポジティブ

評判:
業界は狭い
良い評判が次の仕事を呼ぶ
```

## ブッキングエージェント

### エージェントとは

**仕事を取ってくれる人:**

```
役割:
- ギグ探し
- 交渉
- スケジュール管理
- 契約書作成

手数料:
ギャラの 10-20%

メリット:
- 営業不要
- プロのネットワーク
- 良い条件交渉
- 時間節約

デメリット:
- 手数料
- 初心者は契約困難

タイミング:
月2-4本以上ギグがあるなら検討
```

### エージェント探し

**信頼できる相手:**

```
探し方:

1. リサーチ
   - 同ジャンルDJ が所属
   - 実績ある
   - 評判良い

2. アプローチ
   - デモ、EPK 送付
   - 実績を示す
   - 熱意を伝える

3. 面談
   - 条件確認
   - 手数料
   - 契約内容

注意:
- 契約書を熟読
- 専属か非専属か
- 解約条件
- 手数料率

最初は:
自分で営業
→ 実績作ってからエージェント
```

## よくある質問

### Q1: 初ギグまで何ヶ月？

**A:** 3-6ヶ月が平均

```
タイムライン:

積極的に動けば:
- オープンデッキ: 1ヶ月
- バー/カフェ: 3ヶ月
- クラブ: 6ヶ月-1年

受動的なら:
数年かかる可能性

行動量:
結果に直結
```

### Q2: 断られたらどうする？

**A:** 気にせず次へ

```
現実:
断られるのが普通
10件送って1件返信あれば良い

対処:
1. 落ち込まない
2. フィードバックあれば活用
3. 次のターゲットへ
4. 改善して再挑戦

諦めない:
継続が成功の鍵
```

### Q3: 機材は何を持参？

**A:** 基本は USB のみ

```
通常:

会場の機材使用:
- CDJ + DJM（クラブ）
- コントローラー（バー）

持参するもの:
□ USBメモリ（2本、バックアップ）
□ ヘッドフォン
□ ケーブル（念のため）

DDJ-FLX4:
小さい会場なら持参可能
事前に確認
```

## ブッキングの全体像と仕組み

### DJ業界のブッキング構造

**ギグが成立するまでの流れ:**

```
ブッキングのエコシステム全体像:

┌─────────────────────────────────────────────────┐
│                 ブッキングの流れ                    │
├─────────────────────────────────────────────────┤
│                                                   │
│  [アーティスト/DJ]                                  │
│       │                                           │
│       ├── 直接営業 ──→ [プロモーター]               │
│       │                    │                      │
│       ├── エージェント経由 → [プロモーター]          │
│       │       │            │                      │
│       │       │            ▼                      │
│       │       │       [イベント企画]                │
│       │       │            │                      │
│       │       │            ▼                      │
│       │       │       [会場/クラブ]                 │
│       │       │            │                      │
│       │       │            ▼                      │
│       │       └───→ [契約・ギャラ交渉]              │
│       │                    │                      │
│       │                    ▼                      │
│       └────────────→ [ギグ当日]                    │
│                            │                      │
│                            ▼                      │
│                     [フォローアップ]                 │
│                            │                      │
│                            ▼                      │
│                     [リピート/紹介]                  │
│                                                   │
└─────────────────────────────────────────────────┘

各関係者の役割:

プロモーター（主催者）:
├── イベントの企画・運営
├── DJのブッキング決定権
├── 予算管理
├── 集客戦略
└── 会場との交渉

ブッキングエージェント:
├── DJの代理人
├── ギグの営業活動
├── ギャラ交渉
├── スケジュール管理
└── 契約書作成・管理

会場（クラブ/バー）:
├── スペース提供
├── 音響・照明設備
├── ドリンク/フード販売
├── セキュリティ
└── 営業許可管理

DJ（アーティスト）:
├── 音楽パフォーマンス
├── ブランド構築
├── ファンベース拡大
├── コンテンツ制作
└── スキル向上
```

### ブッキングの種類と経路

**仕事が来る4つのルート:**

```
ルート1: 直接ブッキング（最も一般的）
━━━━━━━━━━━━━━━━━━━━━━━━━━━

プロモーター → 直接DJ に連絡
                → 条件交渉
                → 合意 → ギグ成立

特徴:
- 中間マージンなし
- 自分で全て管理
- 初心者〜中級者はこれがメイン
- 関係性が直接構築される

────────────────────────────

ルート2: エージェント経由
━━━━━━━━━━━━━━━━━━━

プロモーター → エージェント → DJ
               ↕ 交渉代行
               → 契約書作成
               → ギグ成立

特徴:
- プロフェッショナルな対応
- 手数料10-20%
- 中級〜上級者向け
- より大きなギグへアクセス

────────────────────────────

ルート3: 自主企画
━━━━━━━━━━━━━

DJ自身 → イベント企画
       → 会場確保
       → 他DJブッキング
       → 集客・宣伝
       → 当日運営

特徴:
- 完全な自由度
- リスクも利益も自分
- イベントプロモーターとしての経験
- 業界内での評価向上

────────────────────────────

ルート4: プラットフォーム経由
━━━━━━━━━━━━━━━━━━━━━━

オンラインプラットフォーム:
- GigSalad
- Bark
- The Bash
- Peatix（日本）

特徴:
- プライベートイベント中心
- 手数料あり
- 初心者でも参入しやすい
- ウェディング/企業イベント多め
```

### ブッキングのタイムライン

**1つのギグが成立するまで:**

```
一般的なタイムライン:

2-6ヶ月前: 大型イベント/フェスの企画開始
     │
     ▼
1-3ヶ月前: ヘッドライナー確定
     │
     ▼

     │         ← ここであなたにチャンス
     ▼

     │
     ▼

     │
     ▼
前日-当日: サウンドチェック・本番
     │
     ▼

- 2-4週間前にブッキング

- レギュラーなら月単位で決定

覚えておくこと:
┌────────────────────────────────┐
│ 大きなイベントほど            │

│                                │
```

## ブッキングエージェントとの関係構築

### エージェントの選び方（詳細版）

**信頼できるエージェントを見極める:**

```
良いエージェントの条件:

1. ジャンル専門性
   ├── あなたと同じジャンルのDJを多く抱えている
   ├── そのシーンに精通している
   ├── 適切な会場・イベントとのコネクション
   └── ジャンル特有の慣習を理解

2. 実績とネットワーク
   ├── 所属DJのギグ実績が確認できる
   ├── 主要なクラブ・フェスとの関係
   ├── 業界内での評判が良い
   └── 過去のクライアントからの推薦

3. コミュニケーション
   ├── レスポンスが早い（24-48時間以内）
   ├── 透明性がある（ギャラ、条件の開示）
   ├── 定期的な状況報告
   └── あなたのキャリアビジョンを理解

4. 契約条件の妥当性
   ├── 手数料率: 10-20%（一般的）
   ├── 契約期間: 1-2年（更新制）
   ├── 独占/非独占の区別が明確
   └── 解約条件が合理的

避けるべきサイン:
┌──────────────────────────────────────┐
│ ✗ 前払い手数料を要求                 │
│ ✗ 実績を示せない                     │
│ ✗ 契約書がない                       │
│ ✗ 他のDJからの評判が悪い             │
│ ✗ あなたのジャンルに無関心           │
│ ✗ 連絡が遅い・不規則                 │
│ ✗ ギャラの詳細を隠す                 │
└──────────────────────────────────────┘
```

### エージェント契約の種類

**3つの契約形態:**

```
1. 独占契約（Exclusive Agreement）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

内容:
- 全てのブッキングをエージェント経由
- 自分で取った仕事もエージェントに報告
- エージェントが全ての交渉を代行

メリット:
+ エージェントの本気度が高い
+ 優先的に営業してもらえる
+ 一貫したブランディング

デメリット:
- 自由度が低い
- エージェントが動かないとギグがない
- 手数料が全てのギグに適用

適するケース:
→ 月4本以上のギグがある中〜上級DJ

─────────────────────────────────

2. 非独占契約（Non-Exclusive Agreement）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

内容:
- エージェント経由 + 自分でも営業可能
- エージェント経由のみ手数料発生
- 複数のエージェントと契約可能

メリット:
+ 自由度が高い
+ リスク分散
+ 自分のネットワークも活用

デメリット:
- エージェントの優先度が低い可能性
- 管理が複雑
- スケジュール調整が必要

適するケース:
→ 初めてエージェントと契約する場合

─────────────────────────────────

3. 地域別契約（Territory-Based）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

内容:
- 特定地域のみエージェントが担当
- 例: 日本国内=エージェントA
-      海外=エージェントB

メリット:
+ 各地域の専門家に任せられる
+ グローバル展開に適する
+ 効率的

デメリット:
- 調整が必要
- 複数契約の管理

適するケース:
→ 国内外で活動するDJ
```

### エージェントへのアプローチ方法

**準備すべき資料:**

```
EPK（Electronic Press Kit）に含めるもの:

1. バイオグラフィー（3種類用意）
   ├── 長文版（500-1000文字）: 詳細な経歴
   ├── 中文版（200-300文字）: イベント告知用
   └── 短文版（50-100文字）: SNSプロフィール用

2. 写真素材
   ├── プレス用高解像度写真（3-5枚）
   ├── ライブ写真
   ├── アーティスト写真（白背景/黒背景）
   └── ファイル形式: JPEG, 300dpi以上

3. デモミックス
   ├── メインデモ（60分）: あなたのベスト
   ├── ライブ録音（あれば）: 実力証明
   └── SoundCloud/Mixcloud リンク

4. 実績一覧
   ├── 出演イベントリスト
   ├── 出演会場リスト
   ├── 共演DJ リスト
   └── メディア掲載歴

5. SNS・数字
   ├── フォロワー数
   ├── 月間リスナー数
   ├── 動画再生数
   └── メーリングリスト登録者数

6. 連絡先
   ├── メールアドレス
   ├── 電話番号
   ├── SNS リンク
   └── ウェブサイト
```

## デモミックスの制作（詳細ガイド）

### プロフェッショナルなデモの作り方

**録音から配信まで:**

```
Step 1: コンセプト決定
━━━━━━━━━━━━━━━━━━━

目的に応じた設計:

ウォームアップ用デモ:
├── BPM: 118-124（徐々に上昇）
├── 時間: 30-45分
├── 構成: ディープ → グルーヴィ
├── 選曲: 渋め、通好み
└── 印象: 「この人にオープニング任せたい」

メインタイム用デモ:
├── BPM: 126-135
├── 時間: 60分
├── 構成: エネルギッシュ、ピークあり
├── 選曲: アンセム含む
└── 印象: 「フロアを任せられる」

クロージング用デモ:
├── BPM: 130→120（徐々に低下）
├── 時間: 30-45分
├── 構成: エモーショナル → メロウ
├── 選曲: エモい曲、名曲
└── 印象: 「余韻を残せる」

─────────────────────────────────

Step 2: 選曲
━━━━━━━━━━━

重要なルール:
┌──────────────────────────────────┐
│ ・最初の3曲で勝負が決まる       │
│ ・プロモーターは平均90秒聴く    │
│ ・最初の5分で判断される         │
│ ・キラートラックを冒頭に配置   │
└──────────────────────────────────┘

選曲のバランス:
├── 30% 定番・認知度高い曲
├── 40% あなたらしい選曲
├── 20% 新しい/レアな曲
└── 10% 自作曲（あれば）

─────────────────────────────────

Step 3: ミックス構成
━━━━━━━━━━━━━━━━━━━

60分デモの構成例:

00:00-05:00  イントロ（強い掴み）
05:00-15:00  グルーヴ構築
15:00-25:00  テンション上昇
25:00-35:00  第1ピーク
35:00-40:00  ブレイクダウン
40:00-50:00  第2ピーク（最高潮）
50:00-55:00  クールダウン
55:00-60:00  エンディング

─────────────────────────────────

Step 4: 録音
━━━━━━━━━━━

環境:
├── 静かな部屋
├── モニタースピーカー使用
├── 録音レベル確認（クリッピングなし）
└── 一発録り推奨（編集は最小限）

録音ソフト:
├── Audacity（無料）
├── Ableton Live（有料）
├── rekordbox（録音機能）
└── ミキサーのREC OUT → オーディオIF

録音設定:
├── サンプルレート: 44.1kHz
├── ビット深度: 24bit
├── 形式: WAV（マスター）
└── 配信用: MP3 320kbps

─────────────────────────────────

Step 5: マスタリング
━━━━━━━━━━━━━━━━

最低限の処理:
├── ノーマライズ（-0.3dB）
├── 不要な無音カット
├── フェードイン/アウト（必要に応じて）
└── 全体の音量バランス確認

ツール:
├── Audacity: 無料で十分
├── iZotope Ozone: プロ仕様
└── LANDR: AI自動マスタリング

─────────────────────────────────

Step 6: アップロード・配信
━━━━━━━━━━━━━━━━━━━━━━━

SoundCloud 設定:
├── タイトル: DJ名 - ジャンル Mix（月 年）
│   例: DJ GAKU - Deep Techno Mix (March 2026)
├── 説明文: トラックリスト完全記載
├── タグ: ジャンル、BPM、用途
├── アートワーク: 高品質な画像
├── ダウンロード: 有効にする
└── プライベート設定: 必要に応じて

Mixcloud 設定:
├── 著作権問題なし（ライセンス済み）
├── トラックリスト自動認識
└── プロモーター向けに最適
```

```
重要な心構え:
┌─────────────────────────────────────┐
```

### 交渉のテクニック

**プロフェッショナルな交渉術:**

```
交渉の基本原則:

1. 自分の価値を知る
   ├── 経験年数
   ├── 過去の実績
   ├── SNSフォロワー数
   ├── 集客力
   └── 専門性・ユニークさ

2. 相手の予算を理解する
   ├── イベント規模
   ├── チケット価格
   ├── スポンサーの有無
   └── 他のDJのギャラ水準

3. Win-Winを目指す
   ├── 金額だけでなく条件全体で判断
   ├── 長期的関係を重視
   └── 柔軟性を持つ

────────────────────────────

交渉シナリオ例:

プロモーター:
「ギャラは¥20,000でお願いできますか？」

NG回答:
「もっとください」（根拠なし）

OK回答:
「ありがとうございます。
通常この規模のイベントでは
¥30,000-40,000でお受けしています。

今回は初めてのお取引ですので、
¥30,000でいかがでしょうか？

その代わり、SNSでのイベント告知や
当日の写真撮影にも協力させていただきます。」

ポイント:
├── 相場を提示（根拠がある）
├── 初回割引を示唆
├── 付加価値を提案
└── 具体的な金額を提示

────────────────────────────

交渉で使えるフレーズ集:

値上げ交渉:
「通常は¥○○でお受けしていますが...」
「前回の同規模イベントでは¥○○でした」
「交通費・機材費を考慮いただけますか」

柔軟性を示す:
「予算に応じてセット時間の調整も可能です」
「今回は特別価格でお受けします」
「次回以降の継続をお約束いただければ...」

断る場合:
「今回は条件が合いませんが、
また機会があればぜひお声がけください」

絶対NGなフレーズ:
├── 「安すぎます」（失礼）
├── 「他ではもっともらってます」（嘘くさい）
├── 「タダではやりません」（態度が悪い）
└── 「いくらでもいいです」（自分を安売り）
```

### 無料ギグの判断基準

**タダで出るべきか否か:**

```

□ 初めてのギグ（経験が最優先）

覚えておくこと:
┌──────────────────────────────────────┐
│ 無料ギグは投資                       │
```

```
   └── 「ロングミックスなら この人」

   └── 「レア盤なら この人」
   └── 「新譜の速さなら この人」

   └── 「VJ込みなら この人」

   └── 「ユーモアなら この人」

```

```
   └── オンラインだけでは不十分

   └── ギグは信頼から生まれる

   └── 時間厳守・約束を守る

   └── 「○○といえばあなた」を目指す

    └── 音楽への情熱が最大の武器
```
