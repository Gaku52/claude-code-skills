# 正規表現とテキスト処理

> 正規表現はテキスト処理の強力な武器。基本構文からルックアヘッド/ビハインド、名前付きキャプチャ、Unicode 対応、各言語での実装まで、正規表現の全てを解説する。

## クイックリファレンス

```
正規表現チートシート:
  .       — 任意の1文字
  \d \w \s — 数字/英数字/空白
  [abc]   — 文字クラス
  ^  $    — 行頭/行末
  *  +  ? — 0回以上/1回以上/0-1回
  {n,m}   — n〜m回
  (...)   — キャプチャグループ
  (?:...) — 非キャプチャグループ
  (?=...) — 先読み
  (?<=..) — 後読み
  \1      — 後方参照
```
