## クイックリファレンス

```
計算量チートシート:
  O(1)        — ハッシュテーブル lookup
  O(log n)    — 二分探索
  O(n)        — 線形探索
  O(n log n)  — マージソート、クイックソート（平均）
  O(n²)       — バブルソート、挿入ソート
  O(2^n)      — 部分集合列挙
  O(n!)       — 順列列挙
```
