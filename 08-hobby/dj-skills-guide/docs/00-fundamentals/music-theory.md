# 音楽理論基礎

DJ と楽曲制作に必要な音楽理論の基礎を、実践的な視点で体系的に学びます。本章では、音の物理的な性質から始まり、スケール、インターバル、コード、モード、そしてそれらを DJ プレイや楽曲制作でどのように活用するかまで、幅広くカバーします。

## この章で学ぶこと

- 音の物理的な性質と周波数の関係
- 音階（スケール）の構造と種類
- 音程（インターバル）の完全な理解
- メジャーとマイナーの違いと相互関係
- コード（和音）の構成と進行パターン
- モード（旋法）の特徴と活用法
- テンションノートとエクステンデッドコード
- 実際のDJ・制作での実践的な使い方
- ジャンル別の音楽理論的特徴

## なぜDJ・制作に音楽理論が必要なのか

音楽理論は「なんとなく良い」を「なぜ良いのか理解して再現できる」に変えるツールです。理論を知ることで、感覚だけに頼らない安定したパフォーマンスと制作が可能になります。

### DJの場合

**ハーモニックミキシングの実現:**
- キーの合う曲を選んでスムーズなトランジションができる
- 不協和音を避け、フロア全体が心地よいと感じるミックスを作れる
- Camelot Wheel や Circle of Fifths を活用した論理的な選曲ができる

**フロアコントロールの向上:**
- メジャーキーの曲でアッパーな雰囲気を演出できる
- マイナーキーの曲でダークでディープな空気感を作れる
- キーの変化を使って意図的にテンションを上下させられる

**トランジションの品質向上:**
- ベースラインが衝突しない曲の組み合わせを瞬時に判断できる
- ボーカル同士がぶつからないミックスポイントを選べる
- ループやエフェクトを音楽的に適切なタイミングで使える

**選曲の幅の拡大:**
- 同じキーだけでなく、相性の良いキーの曲も候補に入れられる
- ジャンルをまたいだミックスでも音楽的な整合性を保てる
- リクエストに対して、現在の流れを壊さない代替曲を提案できる

### 制作の場合

**メロディ・コード制作の自由度:**
- スケールを理解することで、間違いのないメロディが書ける
- コード進行のパターンを知ることで、狙った感情を表現できる
- テンションノートを使った洗練されたハーモニーが作れる

**アレンジメントの向上:**
- 転調を使ったドラマチックな展開が作れる
- セクションごとのコード進行の変化で物語性を持たせられる
- ブレイクダウンやビルドアップの効果を最大化できる

**ミックスダウンの品質:**
- 各楽器の周波数帯域と音程の関係を理解してEQ処理ができる
- ベースラインとキックの関係を音楽理論的に最適化できる
- ハーモニクス（倍音）を理解してサウンドデザインに活かせる

**ジャンルの理解と応用:**
- 各ジャンルで使われる典型的なスケールやコード進行がわかる
- ジャンルの「らしさ」を理論的に分析して再現できる
- ジャンルの枠を超えた実験的なアプローチの基盤になる

### プロの視点：なぜ理論を学ぶべきか

多くのプロDJ・プロデューサーが口を揃えて言うのは、「理論を知らなくても良い音楽は作れるが、理論を知っていれば作れる音楽の幅が格段に広がる」ということです。理論は創造性を制限するものではなく、むしろ創造性を加速させるツールです。

例えば、Carl Cox は自身のセットでハーモニックミキシングを意識していることで知られています。Deadmau5 はピアノの素養があり、コード理論を駆使して楽曲を制作しています。Flume は独自のコード感覚で知られますが、その基盤には確かな音楽理論の理解があります。

## 1. 音とは何か

### 音の物理的な性質

音は空気の振動（疎密波）として伝わります。この振動には以下の主要な特性があります。

**周波数（Frequency）:**
- 1秒間に振動する回数で、単位はヘルツ（Hz）
- 周波数が高いほど高い音、低いほど低い音として知覚される
- 人間の可聴範囲は約20Hz〜20,000Hz（20kHz）
- 年齢とともに高域の聴力は低下し、40代では16kHz程度が上限になることも多い

**振幅（Amplitude）:**
- 振動の大きさで、音の大きさ（音量）を決定する
- デシベル（dB）で測定される
- 音楽制作では dBFS（decibels relative to full scale）が基準

**波形（Waveform）:**
- 音の「形」で、音色（ティンバー）を決定する
- サイン波、矩形波、三角波、ノコギリ波などの基本波形がある
- 楽器の音は複数の周波数成分（倍音）の組み合わせ

**位相（Phase）:**
- 波の開始位置を表す
- DJ ミキシングでは、同じ周波数の音が逆位相になると打ち消し合う（位相キャンセル）
- ステレオ音場の広がりにも関係する

### 倍音（ハーモニクス）の仕組み

楽器の音が豊かに聞こえるのは、基本となる周波数（基音）に加えて、その整数倍の周波数を持つ**倍音（ハーモニクス）**が含まれているためです。

**倍音列の例（基音 A2 = 110Hz の場合）:**
```
第1倍音（基音）: 110Hz  = A2
第2倍音:        220Hz  = A3（1オクターブ上）
第3倍音:        330Hz  = E4（1オクターブ+完全5度上）
第4倍音:        440Hz  = A4（2オクターブ上）
第5倍音:        550Hz  = C#5（2オクターブ+長3度上）
第6倍音:        660Hz  = E5（2オクターブ+完全5度上）
第7倍音:        770Hz  ≈ G5（やや低い）（2オクターブ+短7度付近）
第8倍音:        880Hz  = A5（3オクターブ上）
```

**制作における倍音の重要性:**
- シンセサイザーの音作りは倍音の加減が基本
- サブトラクティブシンセ: 倍音が豊富な波形（ノコギリ波など）からフィルターで不要な倍音を除去
- アディティブシンセ: サイン波を重ね合わせて倍音を構築
- FM シンセ: 倍音構成をモジュレーションで動的に変化
- EQ処理の本質は倍音バランスの調整

**DJにおける倍音の知識:**
- ベースラインの基音がキックと近い周波数の場合、倍音も衝突する
- EQでベース帯域をカットしても、倍音が残るとベース感が維持される
- ハイパスフィルターをかけても、倍音のおかげで「ベースの存在」を感じさせることができる

### 音の高さ（ピッチ）の詳細

音の高さは**周波数（Hz）**で決まります。現代の西洋音楽では、A4（ラ）= 440Hz を基準としています。

- **高い音**: 周波数が高い（例: 女性の声 200-1000Hz、シンバル 3000-20000Hz）
- **低い音**: 周波数が低い（例: 男性の声 85-300Hz、キック 40-100Hz）

**DJ・制作で重要な周波数帯域:**

| 帯域名 | 周波数範囲 | 音楽的な役割 | 代表的な楽器 |
|-------|----------|-----------|-----------|
| サブベース | 20-60Hz | 体で感じる振動 | サブベース、キック |
| ベース | 60-250Hz | 曲の土台 | ベースライン、キック |
| ローミッド | 250-500Hz | 温かみ | ギター、ボーカル低域 |
| ミッド | 500Hz-2kHz | 存在感 | ボーカル、シンセ |
| ハイミッド | 2-4kHz | 明瞭さ | ボーカルの子音 |
| プレゼンス | 4-8kHz | きらめき | ハイハット、シンバル |
| エア | 8-20kHz | 空気感 | アンビエンス |

**チューニングの基準:**

歴史的に、チューニングの基準は変遷してきました。

- バロック時代: A4 = 415Hz（現代より約半音低い）
- クラシック: A4 = 442-443Hz（オーケストラによって異なる）
- 現代標準: A4 = 440Hz（ISO規格）
- 432Hz チューニング: 一部のプロデューサーが好む代替基準（より「自然な」響きとされるが科学的根拠は議論中）

DJ として知っておくべきことは、ピッチを変更するとキーも変わるということです。CDJ の Tempo スライダーでBPMを変えると、Master Tempo をオフにしている場合、ピッチ（キー）も一緒に変わります。+6%で約半音上がり、-6%で約半音下がります。

### 音名の体系

西洋音楽では、音に以下の名前がついています：

**英語式音名（DJ機材で最も一般的）:**
```
C - D - E - F - G - A - B (- C)
```

**イタリア式音名（日本の音楽教育で一般的）:**
```
ド - レ - ミ - ファ - ソ - ラ - シ (- ド)
```

**ドイツ式音名（クラシック音楽で使用）:**
```
C - D - E - F - G - A - H (- C)
※ B♭ を B と呼ぶ点が英語式と異なる
```

DJ機材（Rekordbox、Traktor、Serato）では英語式音名が標準です。楽譜や音楽教育ではイタリア式が使われることが多いですが、DJ・制作の文脈では英語式に慣れておくことを強く推奨します。

これらの音名は**オクターブ**（同じ音名の音から次の同じ音名の音まで）を形成します。オクターブは周波数が正確に2倍の関係です。

```
A3 = 220Hz → A4 = 440Hz → A5 = 880Hz
（1オクターブ上がると周波数が2倍になる）
```

### 半音と全音

1オクターブは12の等間隔の半音（semitone）に分割されます。これが**平均律（Equal Temperament）**と呼ばれるチューニングシステムです。

- **半音（Semitone / Half Step）**: 隣り合う鍵盤1つ分の距離（例: C → C#）
- **全音（Whole Tone / Whole Step）**: 半音2つ分の距離（例: C → D）

**ピアノの鍵盤で確認:**
```
│ │▓│ │▓│ │ │▓│ │▓│ │▓│ │
│ │▓│ │▓│ │ │▓│ │▓│ │▓│ │
│ C#│ D#│ │ F#│ G#│ A#│ │
│C  │D  │E │F  │G  │A  │B │C
└───┴───┴──┴───┴───┴───┴──┘
```

**12の半音の一覧:**
```
C → C#/D♭ → D → D#/E♭ → E → F → F#/G♭ → G → G#/A♭ → A → A#/B♭ → B → C
 半   半    半    半    半  半    半    半    半    半    半    半
```

### 異名同音（エンハーモニック）

同じ音に2つの名前がある場合があります。これを**異名同音（Enharmonic Equivalent）**と呼びます。

- C# = D♭（同じ音、名前が違う）
- D# = E♭
- F# = G♭
- G# = A♭
- A# = B♭

DJ ソフトウェアでは、同じキーが異なる表記で表示されることがあります。例えば、Rekordbox では「F#m」と表示される曲が、Traktor では「G♭m」と表示されることがあります。これらは同じキーです。

### セント（Cent）の概念

半音をさらに細かく分割した単位が**セント**です。1半音 = 100セント、1オクターブ = 1200セントです。

DJ にとってセントが重要な場面：
- ピッチフェーダーの微調整（1セント単位の調整が可能な機材もある）
- ボーカル曲のチューニング確認（人間の耳は約5-10セントのズレを感知できる）
- Master Tempo 使用時のアーティファクト確認

制作では：
- シンセサイザーのオシレーターのデチューン（数セントずらすと音に厚みが出る）
- サンプルのピッチ補正
- ボーカルのオートチューン設定

## 2. スケール（音階）

スケール（音階）とは、ある規則に従って音を並べたものです。スケールは楽曲のメロディとハーモニーの基盤となり、曲の「調性」を決定します。

### メジャースケール（長音階）

明るく、ポジティブな響きを持つスケールです。ポップスやハウスミュージックの多くがメジャースケールを基盤としています。

**構造（全音と半音の並び）:**
```
全-全-半-全-全-全-半
W-W-H-W-W-W-H
```

**C Major Scale（ハ長調）の例:**
```
C - D - E - F - G - A - B - C
 全  全  半  全  全  全  半
```

**すべてのメジャースケール一覧:**

| キー | 構成音 | シャープ/フラット数 |
|-----|-------|-----------------|
| C Major | C D E F G A B | なし |
| G Major | G A B C D E F# | #1 |
| D Major | D E F# G A B C# | #2 |
| A Major | A B C# D E F# G# | #3 |
| E Major | E F# G# A B C# D# | #4 |
| B Major | B C# D# E F# G# A# | #5 |
| F# Major | F# G# A# B C# D# E# | #6 |
| F Major | F G A B♭ C D E | ♭1 |
| B♭ Major | B♭ C D E♭ F G A | ♭2 |
| E♭ Major | E♭ F G A♭ B♭ C D | ♭3 |
| A♭ Major | A♭ B♭ C D♭ E♭ F G | ♭4 |
| D♭ Major | D♭ E♭ F G♭ A♭ B♭ C | ♭5 |

**DJ・制作での実際の使われ方:**
- House、Disco、Pop、Tranceなどの明るい曲全般
- アッパーなメロディやリフ
- サビ（コーラス）部分のハーモニー
- 太陽が昇る朝方のセットや、ピークタイムのアンセム

**メジャースケールを使ったDJテクニック:**

同じメジャーキーの曲同士を繋ぐと、非常に自然なトランジションが作れます。例えば、C Major の曲から G Major の曲に移行すると、1音だけが変化（F → F#）するため、フロアに違和感を与えずにキーを変更できます。これが五度圏（Circle of Fifths）の基本原理です。

### マイナースケール（短音階）

暗く、感情的な響きを持つスケールです。テクノやダブステップなど、ダークな雰囲気のジャンルで多用されます。マイナースケールには3つの主要なバリエーションがあります。

#### ナチュラルマイナースケール（自然的短音階）

最も基本的なマイナースケールです。

**構造:**
```
全-半-全-全-半-全-全
W-H-W-W-H-W-W
```

**A Minor Scale（イ短調）の例:**
```
A - B - C - D - E - F - G - A
 全  半  全  全  半  全  全
```

**すべてのナチュラルマイナースケール一覧:**

| キー | 構成音 | 相対メジャーキー |
|-----|-------|--------------|
| A minor | A B C D E F G | C Major |
| E minor | E F# G A B C D | G Major |
| B minor | B C# D E F# G A | D Major |
| F# minor | F# G# A B C# D E | A Major |
| C# minor | C# D# E F# G# A B | E Major |
| G# minor | G# A# B C# D# E F# | B Major |
| D minor | D E F G A B♭ C | F Major |
| G minor | G A B♭ C D E♭ F | B♭ Major |
| C minor | C D E♭ F G A♭ B♭ | E♭ Major |
| F minor | F G A♭ B♭ C D♭ E♭ | A♭ Major |
| B♭ minor | B♭ C D♭ E♭ F G♭ A♭ | D♭ Major |

#### ハーモニックマイナースケール（和声的短音階）

ナチュラルマイナーの第7音を半音上げたスケールです。エキゾチックでドラマチックな響きが特徴です。

**構造:**
```
全-半-全-全-半-増2-半
W-H-W-W-H-A2-H
```

**A Harmonic Minor の例:**
```
A - B - C - D - E - F - G# - A
 全  半  全  全  半  増2  半
```

第6音と第7音の間に増2度（3半音）の跳躍があり、これがエキゾチックな響きを生み出します。

**DJ・制作での活用:**
- Psy-Trance のメロディでは非常に頻繁に使われる
- Middle Eastern テイストのテクノやハウス
- フラメンコ風のサウンド
- 映画音楽的なドラマチックなブレイクダウン
- Armin van Buuren のトランスクラシックスに多い

#### メロディックマイナースケール（旋律的短音階）

上行時はナチュラルマイナーの第6音と第7音を半音上げ、下行時はナチュラルマイナーに戻るスケールです。ジャズでは上行形のみを使うことが多いです。

**上行形の構造:**
```
全-半-全-全-全-全-半
W-H-W-W-W-W-H
```

**A Melodic Minor（上行）の例:**
```
A - B - C - D - E - F# - G# - A
 全  半  全  全  全   全   半
```

**DJ・制作での活用:**
- ジャズハウス、ディープハウスのメロディ
- ニュアンスのある洗練されたハーモニー
- フュージョン系の楽曲制作
- Kaytranada や Tom Misch のような現代的サウンド

### メジャーとマイナーの関係

**同じ音を共有する関係（相対調 / Relative Key）:**
- C Major と A Minor は同じ音を使う（白鍵のみ）
- メジャーキーの6番目の音がマイナーキーのルートになる
- 例: C Major の6番目は A → A Minor が相対マイナー

**相対調の完全リスト:**

| メジャーキー | 相対マイナーキー | Camelot (Major) | Camelot (Minor) |
|------------|--------------|----------------|----------------|
| C Major | A minor | 8B | 8A |
| G Major | E minor | 9B | 9A |
| D Major | B minor | 10B | 10A |
| A Major | F# minor | 11B | 11A |
| E Major | C# minor | 12B | 12A |
| B Major | G# minor | 1B | 1A |
| F# Major | D# minor | 2B | 2A |
| F Major | D minor | 7B | 7A |
| B♭ Major | G minor | 6B | 6A |
| E♭ Major | C minor | 5B | 5A |
| A♭ Major | F minor | 4B | 4A |
| D♭ Major | B♭ minor | 3B | 3A |

**同主調（Parallel Key）:**
- 同じルート音から始まるメジャーとマイナー
- C Major と C Minor（同じルート C だが構成音が異なる）
- DJミックスでの「メジャー/マイナーチェンジ」に活用

これらの関係は**ハーモニックミキシング**で重要な概念です。Camelot Wheel（後述）ではこの関係が視覚的にわかりやすく表現されています。

### ペンタトニックスケール（五音音階）

5つの音で構成されるスケールで、世界中の音楽に見られる最も古いスケールの一つです。

**メジャーペンタトニック:**
```
C - D - E - G - A - C
（メジャースケールから4度と7度を抜いたもの）
```

**マイナーペンタトニック:**
```
A - C - D - E - G - A
（ナチュラルマイナーから2度と6度を抜いたもの）
```

**DJ・制作での重要性:**
- EDMのリフやフックに非常に多用される
- アジアンテイストのメロディ
- ブルースやR&Bベースのハウス
- シンプルで覚えやすいメロディを作りたい時に最適
- Avicii の多くのヒット曲はペンタトニックベースのメロディ

### ブルーススケール

マイナーペンタトニックに♭5（ブルーノート）を加えた6音のスケールです。

```
A - C - D - D#/E♭ - E - G - A
```

**DJ・制作での活用:**
- ファンキーなベースライン
- ブルージーなボーカルサンプル
- ヒップホップビートのメロディ
- ジャジーハウスのソロセクション
- Jamiroquai 風のアシッドジャズハウス

### クロマティックスケール（半音階）

12の半音すべてを含むスケールです。メロディとしてはあまり使いませんが、理論的な参照点として重要です。

```
C - C# - D - D# - E - F - F# - G - G# - A - A# - B - C
```

**制作での活用場面:**
- トランジション時のライザー（上昇音）
- テンションを高めるビルドアップ
- グリッサンド効果
- インダストリアルテクノのノイズ的要素

### ホールトーンスケール（全音音階）

すべての音が全音間隔で並ぶスケールです。

```
C - D - E - F# - G# - A# - C
```

**特徴と活用:**
- 浮遊感のある不思議な響き
- アンビエントやエクスペリメンタルな制作
- ドビュッシーが多用したことで知られる
- トランジション中のエフェクト的な使用
- Boards of Canada のようなサイケデリックなサウンド

## 3. インターバル（音程）

2つの音の距離を**インターバル（音程）**と呼びます。インターバルはメロディとハーモニーの両方を理解するための基礎的な概念です。

### 主要なインターバルの完全リスト

| インターバル名 | 半音数 | C からの例 | 響きの特徴 | 協和/不協和 |
|-------------|-------|----------|----------|----------|
| 完全1度 / ユニゾン (P1) | 0 | C - C | 同じ音 | 完全協和 |
| 短2度 (m2) | 1 | C - D♭ | 緊張、不安 | 不協和 |
| 長2度 (M2) | 2 | C - D | 開放的 | 緩い不協和 |
| 短3度 (m3) | 3 | C - E♭ | 悲しい、暗い | 不完全協和 |
| 長3度 (M3) | 4 | C - E | 明るい、幸福 | 不完全協和 |
| 完全4度 (P4) | 5 | C - F | 安定、荘厳 | 完全協和 |
| 増4度/減5度 (A4/d5) | 6 | C - F#/G♭ | 不安定、悪魔的 | 不協和 |
| 完全5度 (P5) | 7 | C - G | 非常に安定、力強い | 完全協和 |
| 短6度 (m6) | 8 | C - A♭ | 甘い、やや暗い | 不完全協和 |
| 長6度 (M6) | 9 | C - A | 温かい、穏やか | 不完全協和 |
| 短7度 (m7) | 10 | C - B♭ | ブルージー | 不協和 |
| 長7度 (M7) | 11 | C - B | 鋭い緊張 | 不協和 |
| 完全8度/オクターブ (P8) | 12 | C - C' | 同じ音（高さ違い） | 完全協和 |

### インターバルの転回

インターバルを**転回（Inversion）**すると、2つの音の上下関係が入れ替わります。

**転回の法則:**
- 元のインターバル + 転回 = 12半音（オクターブ）
- 完全音程は転回しても完全音程
- 長音程の転回は短音程（逆も同じ）
- 増音程の転回は減音程（逆も同じ）

| 元のインターバル | 転回後 |
|-------------|-------|
| 完全1度 (0) | 完全8度 (12) |
| 短2度 (1) | 長7度 (11) |
| 長2度 (2) | 短7度 (10) |
| 短3度 (3) | 長6度 (9) |
| 長3度 (4) | 短6度 (8) |
| 完全4度 (5) | 完全5度 (7) |
| 増4度 (6) | 減5度 (6) |

### トライトーン（増4度 / 減5度）

6半音のインターバルは**トライトーン（Tritone）**と呼ばれ、音楽理論において特別な位置を占めます。

- 中世ヨーロッパでは「悪魔の音程（Diabolus in Musica）」と呼ばれ忌避された
- オクターブのちょうど中間に位置する
- 非常に不安定で、解決（安定した音程への移行）を強く求める
- ドミナント7thコードの中に含まれ、トニックへの解決力を生む

**DJ・制作での活用:**
- テンションの高いビルドアップ
- ホラー映画的なサウンドデザイン
- インダストリアルテクノのリフ
- ジャズハウスの「裏コード（Tritone Substitution）」

### DJでのインターバルの実践的活用

**協和音程を活用したミキシング（推奨）:**

1. **完全1度（ユニゾン）= 同じキー:** 最も安全なミキシング。同じキーの曲同士は基本的に何をしても衝突しない。
2. **完全5度（7半音差）:** 非常に自然な移行。Camelot Wheel で1つ時計回りに移動。
3. **完全4度（5半音差）:** 5度の転回。Camelot Wheel で1つ反時計回りに移動。
4. **長3度/短3度:** 雰囲気を大きく変えたい時。感情的な効果が大きい。

**不協和音程を避けるべき場面:**

1. **短2度（1半音差）:** 隣り合うキー。ベースラインが完全に衝突する。
2. **長2度（2半音差）:** やや改善するが、まだ不協和が目立つ。
3. **トライトーン（6半音差）:** 最も避けるべき組み合わせ。

**プロのTips: キー差の許容範囲**

実際のDJプレイでは、以下の条件を満たせば多少のキー差は許容されます：
- ベースラインが同時に鳴らない（EQでカット）
- メロディ要素が少ない曲同士
- パーカッシブな要素が主体の曲
- トランジションが短い（2-4小節以内）

逆に、以下の場合はキーの一致が重要：
- ボーカル曲同士のミキシング
- メロディアスなトランスやプログレッシブハウス
- 長いブレンドミックス（16小節以上）
- ベースラインが同時に鳴る場面

## 4. コードの基礎

コード（和音）は、3つ以上の音を同時に鳴らしたものです。コードは楽曲のハーモニーの骨格を形成し、メロディの感情的な文脈を与えます。

### トライアド（3和音）

3つの音を重ねた最も基本的なコードです。

#### メジャートライアド（明るい響き）

```
C メジャー = C + E + G
（ルート + 長3度 + 完全5度）
```

構造: ルートから長3度（4半音）+ 短3度（3半音）
響き: 明るく、安定、幸福感

**すべてのメジャートライアド:**
```
C  = C  E  G       F  = F  A  C
C# = C# E# G#      F# = F# A# C#
D  = D  F# A       G  = G  B  D
E♭ = E♭ G  B♭      A♭ = A♭ C  E♭
E  = E  G# B       A  = A  C# E
                    B♭ = B♭ D  F
                    B  = B  D# F#
```

#### マイナートライアド（暗い響き）

```
C マイナー = C + E♭ + G
（ルート + 短3度 + 完全5度）
```

構造: ルートから短3度（3半音）+ 長3度（4半音）
響き: 暗く、悲しい、感情的

#### ディミニッシュトライアド（不安定な響き）

```
C ディミニッシュ = C + E♭ + G♭
（ルート + 短3度 + 減5度）
```

構造: ルートから短3度（3半音）+ 短3度（3半音）
響き: 非常に不安定、緊張感、ホラー的

#### オーギュメントトライアド（浮遊する響き）

```
C オーギュメント = C + E + G#
（ルート + 長3度 + 増5度）
```

構造: ルートから長3度（4半音）+ 長3度（4半音）
響き: 浮遊感、夢幻的、不思議

**制作での使い方:** ビルドアップで繰り返すと独特の浮遊感を演出できる。Radiohead が多用することで知られる。

### セブンスコード（4和音）

トライアドに7度の音を加えた4音のコードです。より豊かで複雑な響きを持ちます。

#### メジャーセブンス (Cmaj7)
```
C + E + G + B
（メジャートライアド + 長7度）
```
響き: 洗練、都会的、ジャジー
使用例: ディープハウス、ローファイヒップホップ、チルアウト

#### ドミナントセブンス (C7)
```
C + E + G + B♭
（メジャートライアド + 短7度）
```
響き: ブルージー、緊張感、解決を求める
使用例: ファンクハウス、ジャジーなテクノ、ソウルフルハウス

#### マイナーセブンス (Cm7)
```
C + E♭ + G + B♭
（マイナートライアド + 短7度）
```
響き: メランコリック、深い、内省的
使用例: ディープテクノ、アンビエント、ドラムンベース

#### マイナーセブンスフラットファイブ / ハーフディミニッシュ (Cm7♭5)
```
C + E♭ + G♭ + B♭
（ディミニッシュトライアド + 短7度）
```
響き: 都会的な緊張感
使用例: ジャズハウスの ii-V-I 進行

#### ディミニッシュセブンス (Cdim7)
```
C + E♭ + G♭ + B♭♭(= A)
（ディミニッシュトライアド + 減7度）
```
響き: 極度の緊張、映画音楽的
特徴: すべての音が短3度間隔。転回しても同じ響き。

### テンションノートとエクステンデッドコード

セブンスコードにさらに音を積み重ねると、**エクステンデッドコード（拡張和音）**になります。これらの追加される音を**テンションノート**と呼びます。

#### 9thコード
```
C9 = C + E + G + B♭ + D
Cmaj9 = C + E + G + B + D
Cm9 = C + E♭ + G + B♭ + D
```

**特徴と活用:**
- 9thはコードに広がりと奥行きを加える
- ディープハウスのパッドサウンドに最適
- Cmaj9 は「朝焼け」のような透明感のある響き
- R&B、ネオソウル、ローファイヒップホップの定番

#### 11thコード
```
C11 = C + E + G + B♭ + D + F
Cm11 = C + E♭ + G + B♭ + D + F
```

**特徴と活用:**
- サスペンデッド的な浮遊感
- アンビエントテクノのパッドに効果的
- 11thは3度との衝突に注意（実際にはE省略が多い）

#### 13thコード
```
C13 = C + E + G + B♭ + D + (F) + A
```

**特徴と活用:**
- 最も豊かで複雑な和音
- ジャズファンクのバッキングに最適
- ビッグバンドジャズ的な華やかさ
- ソウルフルハウスやガラージでの装飾的使用

#### サスペンデッドコード（Sus Chords）

3度を2度または4度に置き換えたコードで、メジャーでもマイナーでもない宙吊りの響きを持ちます。

```
Csus2 = C + D + G（3度→2度に置換）
Csus4 = C + F + G（3度→4度に置換）
C7sus4 = C + F + G + B♭（sus4 + 短7度）
```

**DJ・制作での活用:**
- EDMのビルドアップで sus4 → メジャーへの解決が定番
- アンビエントなパッドサウンド
- トランスのメインリフ
- 「解決感」を遅延させるテクニック

#### アッドコード（Add Chords）

トライアドに特定の音を追加するコードです。セブンスを経由しないのが特徴です。

```
Cadd9 = C + E + G + D（トライアド + 9度、7度なし）
Cadd11 = C + E + G + F（トライアド + 11度）
C6 = C + E + G + A（トライアド + 6度）
```

**Cadd9 はEDMの万能コード:** メジャーの明るさに9thの広がりを加えた、非常に使いやすいコード。Martin Garrix のようなメインストリームEDMで頻出。

### コードの転回形（Inversion）

コードの音の配置を変えることで、響きのニュアンスを変えられます。

**C メジャー（C-E-G）の転回形:**
```
基本形（Root Position）:  C - E - G  （ルートが最低音）
第1転回形（1st Inversion）: E - G - C  （3度が最低音）
第2転回形（2nd Inversion）: G - C - E  （5度が最低音）
```

**セブンスコード（Cmaj7 = C-E-G-B）の転回形:**
```
基本形:     C - E - G - B
第1転回形:  E - G - B - C
第2転回形:  G - B - C - E
第3転回形:  B - C - E - G
```

**制作での重要性:**
- ベースラインの動きを最小限にするために転回形を使う（ボイスリーディング）
- パッドサウンドでは転回形によって微妙に雰囲気が変わる
- ベースノートとコードトーンの関係を意識することでミックスがクリーンになる

**ボイスリーディングの実例:**
```
悪い例（ベースが大きくジャンプ）:
C(基本形) → F(基本形) → G(基本形) → C(基本形)
C-E-G → F-A-C → G-B-D → C-E-G
ベース: C → F → G → C（大きな跳躍）

良い例（ボイスリーディングを意識）:
C(基本形) → F(2nd inv) → G(1st inv) → C(基本形)
C-E-G → C-F-A → B-D-G → C-E-G
ベース: C → C → B → C（最小限の動き）
```

### ダイアトニックコード

スケール上の各音から作られるコードです。キーの中で「使える」コードの一覧表と言えます。

**C Major のダイアトニックコード（トライアド）:**
```
I    : C   (メジャー)    ← トニック（安定）
ii   : Dm  (マイナー)    ← サブドミナント機能
iii  : Em  (マイナー)    ← トニック代理
IV   : F   (メジャー)    ← サブドミナント（やや不安定）
V    : G   (メジャー)    ← ドミナント（緊張→解決）
vi   : Am  (マイナー)    ← トニック代理
vii° : Bdim(ディミニッシュ) ← ドミナント代理
```

**C Major のダイアトニックセブンスコード:**
```
Imaj7  : Cmaj7   ← トニック
ii7    : Dm7     ← サブドミナント
iii7   : Em7     ← トニック代理
IVmaj7 : Fmaj7   ← サブドミナント
V7     : G7      ← ドミナント
vi7    : Am7     ← トニック代理
vii∅7  : Bm7♭5   ← ドミナント代理
```

**A Minor（ナチュラル）のダイアトニックコード:**
```
i    : Am  (マイナー)    ← トニック
ii°  : Bdim(ディミニッシュ)
III  : C   (メジャー)
iv   : Dm  (マイナー)
v    : Em  (マイナー)    ※ハーモニックマイナーでは E(メジャー)
VI   : F   (メジャー)
VII  : G   (メジャー)    ※ハーモニックマイナーでは G#dim
```

### コード進行の基本パターン

#### 機能和声の3つの機能

1. **トニック（T）**: 安定、帰着点 — I, iii, vi
2. **サブドミナント（SD）**: やや不安定、動きの開始 — ii, IV
3. **ドミナント（D）**: 強い緊張、トニックへの解決を求める — V, vii°

**基本的な進行の流れ:**
```
T → SD → D → T
（安定→やや不安定→強い緊張→解決）
```

#### ダンスミュージックで頻出するコード進行

**1. I - V - vi - IV（ポップパンク進行）**
```
C - G - Am - F
```
- 現代ポップスで最も多い進行の一つ
- EDMのアンセム的な曲に多用
- 例: Avicii「Wake Me Up」、The Chainsmokers の多くの曲

**2. vi - IV - I - V（ポップ進行の変形）**
```
Am - F - C - G
```
- 上の進行の開始点をviに変えたもの
- よりエモーショナルな印象
- 例: Kygo「Firestone」

**3. i - VI - III - VII（エピック・マイナー進行）**
```
Am - F - C - G（Am キーで）
```
- トランス、プログレッシブハウスの定番
- 壮大で感動的な響き
- 例: Above & Beyond の多くの楽曲

**4. i - iv - v - i（マイナーループ）**
```
Am - Dm - Em - Am
```
- テクノ、ミニマルの定番
- ダークでhypnoticな雰囲気
- 例: ベルリンテクノの多くの楽曲

**5. I - IV（2コードシャトル）**
```
C - F（繰り返し）
```
- ハウスミュージックの定番中の定番
- シンプルだが効果的なグルーヴ
- 例: 多くのシカゴハウスクラシックス

**6. ii - V - I（ジャズ進行）**
```
Dm7 - G7 - Cmaj7
```
- ジャズハウス、ソウルフルハウスの基盤
- 洗練された響き
- 例: Masters at Work、Louie Vega の作品

**7. i - ♭VII - ♭VI - V（アンダルシア進行）**
```
Am - G - F - E
```
- フラメンコ由来の強烈な進行
- Psy-Trance、ゴアトランスに頻出
- 例: スパニッシュギターを使ったトランス

**8. I - ♭VII - IV（ミクソリディアン進行）**
```
C - B♭ - F
```
- ロック的な力強さ
- エレクトロハウス、ビッグルーム
- 例: Swedish House Mafia的なサウンド

**9. i - ♭III - ♭VII - IV（ミニマルマイナー進行）**
```
Am - C - G - D（Am キーで）
```
- プログレッシブハウス、メロディックテクノに頻出
- 希望的で浮遊感のある進行
- 例: Lane 8、Ben Bohmer の楽曲

**10. I - vi - ii - V（ターンアラウンド）**
```
C - Am - Dm - G
```
- ジャズのスタンダード進行
- ソウルフルハウスのループに最適
- Frankie Knuckles 系のクラシカルなハウス

## 5. モード（旋法）

### モードとは何か

モード（旋法）は、同じスケールの音を使いながら、開始音（トニック）を変えることで異なる雰囲気を作り出すシステムです。中世の教会音楽から発展し、ジャズやポピュラー音楽で広く活用されています。

C メジャースケール（C-D-E-F-G-A-B）の7つの音を使って、7つのモードが生まれます。

### 7つの教会旋法

#### 1. イオニアン（Ionian）= メジャースケール

```
C - D - E - F - G - A - B - C
全-全-半-全-全-全-半
```

- 最も基本的なモードで、メジャースケールそのもの
- 明るく、安定した響き
- ポップス、ハウス、トランスの基盤

#### 2. ドリアン（Dorian）

```
D - E - F - G - A - B - C - D
全-半-全-全-全-半-全
```

**特徴:** マイナースケールに似ているが、6度が長6度（明るい6度）
**響き:** マイナーだが暗すぎない、都会的でクールな雰囲気

**DJ・制作での活用:**
- ファンク、ソウルのベースライン（非常に頻出）
- ディープハウスのパッドとメロディ
- ヒップホップビートのループ
- マイルス・デイヴィスの「So What」がドリアンの代表曲
- Daft Punk「Get Lucky」のベースラインはドリアンモード

**プロのTips:** ドリアンはDJ/プロデューサーが最初に覚えるべきモード。ナチュラルマイナーと1音しか違わない（6度が半音高い）が、その1音の違いが大きなニュアンスの違いを生む。

#### 3. フリジアン（Phrygian）

```
E - F - G - A - B - C - D - E
半-全-全-全-半-全-全
```

**特徴:** 2度がフラット（♭2）。スパニッシュ、アラビックな雰囲気
**響き:** エキゾチック、ミステリアス、ダーク

**DJ・制作での活用:**
- フラメンコ系のハウスやテクノ
- ミドルイースタン系のサウンド
- Psy-Trance のベースライン
- メタル系のリフ（ダーク系EDMでのクロスオーバー）
- インダストリアルテクノのシンセリフ

#### 4. リディアン（Lydian）

```
F - G - A - B - C - D - E - F
全-全-全-半-全-全-半
```

**特徴:** 4度がシャープ（#4）。メジャーよりもさらに明るく、夢幻的
**響き:** 浮遊感、壮大、映画音楽的

**DJ・制作での活用:**
- 映画のサウンドトラック的なEDM
- アンビエント、チルアウトのパッド
- スティーヴ・ヴァイ系のギターソロ
- Joe Satriani「Flying in a Blue Dream」がリディアンの好例
- エピックトランスのメインメロディ

#### 5. ミクソリディアン（Mixolydian）

```
G - A - B - C - D - E - F - G
全-全-半-全-全-半-全
```

**特徴:** 7度がフラット（♭7）。メジャーだがブルージーな要素
**響き:** ロック的、ファンキー、パワフル

**DJ・制作での活用:**
- ロック系のEDM、エレクトロハウス
- ファンキーなベースライン
- ブルースロック的なギターリフ
- The Beatles の多くの曲がミクソリディアン
- フェスティバル系のビッグルームEDM

#### 6. エオリアン（Aeolian）= ナチュラルマイナースケール

```
A - B - C - D - E - F - G - A
全-半-全-全-半-全-全
```

- ナチュラルマイナースケールそのもの
- ダークで感情的な響き
- テクノ、ドラムンベース、ダブステップの基盤

#### 7. ロクリアン（Locrian）

```
B - C - D - E - F - G - A - B
半-全-全-半-全-全-全
```

**特徴:** 2度と5度がフラット。最も不安定なモード
**響き:** 非常にダーク、不安定、実験的

**DJ・制作での活用:**
- 実験的なエレクトロニカ
- インダストリアルテクノの不協和なリフ
- ホラー映画のサウンドデザイン
- 実際の楽曲での使用は稀（短いパッセージでの使用が多い）

### モードの比較一覧

| モード | 特徴的な音 | 明るさ | ダンスミュージックでの頻度 |
|-------|----------|-------|-------------------|
| イオニアン | 通常のメジャー | 非常に明るい | 非常に高い |
| ドリアン | ♮6 (明るいマイナー) | やや暗い | 非常に高い |
| フリジアン | ♭2 (エキゾチック) | 暗い | 中程度 |
| リディアン | #4 (夢幻的) | 非常に明るい | 中程度 |
| ミクソリディアン | ♭7 (ブルージー) | 明るい | 高い |
| エオリアン | 通常のマイナー | 暗い | 非常に高い |
| ロクリアン | ♭2, ♭5 (最暗) | 非常に暗い | 低い |

### モードの実践的な聴き分け方

すべてのモードを同じルートから弾いてみると、違いがはっきりとわかります。

**すべて C をルートにした場合:**
```
C イオニアン:     C D E F G A B C     （通常のメジャー）
C ドリアン:      C D E♭ F G A B♭ C   （明るいマイナー）
C フリジアン:     C D♭ E♭ F G A♭ B♭ C （エキゾチック）
C リディアン:     C D E F# G A B C    （夢幻的メジャー）
C ミクソリディアン: C D E F G A B♭ C   （ブルージーメジャー）
C エオリアン:     C D E♭ F G A♭ B♭ C  （通常のマイナー）
C ロクリアン:     C D♭ E♭ F G♭ A♭ B♭ C（最も暗い）
```

## 6. Camelot Wheel とハーモニックミキシング

### Camelot Wheel（カメロットホイール）とは

Camelot Wheel は、音楽のキーの互換性を視覚的に表現した円形のダイアグラムです。Mark Davis によって開発され、DJのハーモニックミキシングを劇的に簡単にしました。

**基本構造:**
- 12の数字（1〜12）が時計のように配置
- 各数字に A（マイナー）と B（メジャー）がある
- 合計24のセグメント（12キー × メジャー/マイナー）

### 完全な Camelot Wheel マッピング

```
Camelot | キー (Open Key) | 音楽キー
--------|----------------|--------
  1A    |    6m          | A♭ minor
  1B    |    6d          | B major
  2A    |    7m          | E♭ minor
  2B    |    7d          | F# major
  3A    |    8m          | B♭ minor
  3B    |    8d          | D♭ major
  4A    |    9m          | F minor
  4B    |    9d          | A♭ major
  5A    |    10m         | C minor
  5B    |    10d         | E♭ major
  6A    |    11m         | G minor
  6B    |    11d         | B♭ major
  7A    |    12m         | D minor
  7B    |    12d         | F major
  8A    |    1m          | A minor
  8B    |    1d          | C major
  9A    |    2m          | E minor
  9B    |    2d          | G major
  10A   |    3m          | B minor
  10B   |    3d          | D major
  11A   |    4m          | F# minor
  11B   |    4d          | A major
  12A   |    5m          | C# minor
  12B   |    5d          | E major
```

### ハーモニックミキシングのルール

**ルール1: 同じ番号内の移動（最も安全）**
```
8A (Am) ↔ 8B (C)
相対調への移動。マイナーとメジャーを切り替え。
```

**ルール2: 隣の番号への移動（自然な進行）**
```
8A (Am) → 9A (Em)  完全5度上のマイナーへ
8A (Am) → 7A (Dm)  完全4度上のマイナーへ（5度下）
8B (C)  → 9B (G)   完全5度上のメジャーへ
8B (C)  → 7B (F)   完全4度上のメジャーへ（5度下）
```

**ルール3: エネルギーブースト（上級テクニック）**
```
8A (Am) → 9B (G)   マイナーからメジャーへ+5度上（明るく高揚）
8B (C)  → 7A (Dm)  メジャーからマイナーへ+5度下（ダークに沈降）
```

**ルール4: 大きなジャンプ（使用注意）**
```
2つ以上離れた番号への移動は不協和のリスクが高い
ただし、EQカットやブレイクを挟めば可能
```

### 実践的なセットプランの例

**ディープハウスセット（2時間）でのキー移動例:**
```
曲1:  8A (Am)     ← 開始
曲2:  8B (C)      ← 相対メジャーへ（明るく）
曲3:  9B (G)      ← 5度上へ（エネルギー上昇）
曲4:  9A (Em)     ← 相対マイナーへ（少し落ち着く）
曲5:  10A (Bm)    ← 5度上のマイナー（テンション維持）
曲6:  10B (D)     ← 相対メジャーへ（明るさ戻す）
曲7:  11B (A)     ← 5度上（さらにエネルギー上昇）
曲8:  11A (F#m)   ← 相対マイナーへ（ピーク前のダーク展開）
曲9:  12A (C#m)   ← 5度上のマイナー（テンション最高潮）
曲10: 12B (E)     ← メジャーへ解決（ピーク！）
```

### よくある間違いと対策

**間違い1: キーだけに頼りすぎる**
キーが合っていても、エネルギーレベルやBPMが大きく異なると不自然なミックスになります。キーはあくまで要素の一つです。

**間違い2: DJ ソフトウェアのキー解析を盲信する**
キー解析の精度は完璧ではありません（特に打ち込みではないライブ演奏の曲）。耳でも確認する習慣をつけましょう。Mixed In Key は比較的精度が高いとされています。

**間違い3: 常にハーモニックミキシングをしなければならないと思う**
パーカッシブな曲同士やノイズ系テクノではキーの一致は重要度が下がります。曲の特性に応じて使い分けましょう。

**間違い4: キー変更のためにピッチを大きく変える**
BPMを大幅に変えてキーを合わせようとすると、曲の雰囲気が崩れます。Master Tempo を使うか、元々キーの近い曲を選びましょう。
