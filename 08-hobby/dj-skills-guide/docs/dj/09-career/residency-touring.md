## レジデンシーとは

### 定義

**定期的な出演契約:**

```
レジデンシー:
特定の会場で定期的に DJ プレイ

頻度:
- 毎週
- 隔週
- 毎月

期間:
数ヶ月〜数年

例:
「毎週金曜、Club XYZ でプレイ」
「毎月第1土曜、Bar ABC でプレイ」

特徴:
安定した仕事
会場との長期関係
```

---

## レジデンシーのメリット

### DJ側のメリット

**安定と成長:**

```
1. 安定収入
   毎月の収入が予測できる

   例:
   月4回 × ¥30,000 = ¥120,000/月

2. スキル向上
   定期的にプレイ
   → 急速に成長

3. ファン獲得
   毎週来る客
   → あなたのファンになる

4. 実験の場
   新しい曲、技術を試せる
   → リスク低い

5. 知名度向上
   「〇〇といえば△△」
   → ブランディング

6. 交渉力
   信頼関係
   → ギャラ交渉しやすい

7. 他ギグにつながる
   レジデンシーが実績
   → 他会場からオファー

8. ネットワーク
   ゲストDJ と交流
   → 人脈拡大
```

### 会場側のメリット

**Win-Win:**

```
1. 安定したクオリティ
   毎回同じDJ
   → 質が保証される

2. 集客
   DJ のファンが来る
   → 売上↑

3. コスト削減
   毎回探す手間なし

4. ブランディング
   「〇〇の日は△△DJ」
   → 会場の特色

お互いにメリット:
長期関係が築ける
```

## レジデンシー獲得方法

### ステップ1: ターゲット選定

**どの会場を狙う？**

```
条件:

□ あなたのジャンルに合う
□ 通える距離
□ 雰囲気が好き
□ 客層が合う
□ 定期イベントがある/ありそう

リサーチ:

□ 実際に通う（3回以上）
□ 現在レジデントDJ がいるか
□ 客の入り具合
□ プロモーターの人柄

候補:
3-5会場リストアップ
```

### ステップ2: 関係構築

**すぐには獲得できない:**

```
Month 1-2: 常連になる

□ 週1回通う
□ 音楽を楽しむ
□ 軽く挨拶
□ SNS でフォロー

Month 3-4: 交流深める

□ プロモーターと会話
□ 他の常連と仲良く
□ イベント手伝い申し出
□ フィードバック提供

Month 5-6: 信頼構築

□ オープンデッキ参加
□ 良いプレイを見せる
□ 継続的に通う
□ コミュニティの一員に

焦らない:
時間をかけて信頼を
```

### ステップ3: 提案

**タイミングが重要:**

```
良いタイミング:

□ レジデントDJ が辞めた
□ 新しい曜日/イベント開始
□ あなたのプレイが好評
□ プロモーターから相談された

アプローチ:

「もし定期的にプレイする機会があれば、
ぜひ検討していただけませんか。

毎週/隔週/毎月で、
[曜日] に [時間帯] のプレイを
させていただけたら嬉しいです。

デモミックスもありますので、
ぜひ聴いていただけないでしょうか。」

謙虚に、でも自信を持って

提案内容:

□ 頻度（週1/隔週/月1）
□ 曜日
□ 時間帯
□ ギャラ
□ 開始日

柔軟性:
相手の要望も聞く
```

### ステップ4: 契約

**条件を明確に:**

```
契約内容:

□ 頻度（毎週金曜等）
□ 演奏時間（22:00-24:00等）
□ ギャラ（¥XX,000/回）
□ 期間（3ヶ月、6ヶ月、1年等）
□ 更新条件
□ キャンセルポリシー
□ 独占条項（他会場 OK/NG）

書面で:
口約束はトラブルの元

試用期間:
最初3ヶ月は試用
→ お互い様子見
→ 問題なければ更新
```

## レジデンシー成功の秘訣

### 継続のコツ

**長く続けるために:**

```
1. 一貫性
   毎回同じクオリティ
   → 信頼される

   フロアの反応に合わせる

   → 常連が飽きない

   客と交流
   → ファンが増える

5. プロ意識
   遅刻しない
   準備万端
   → プロモーターが安心

6. 進化
   技術向上
   選曲進化
   → 成長を見せる

長期レジデンシー:
これらが鍵
```

## ツアー計画

### ステップ1: 目的設定

**なぜツアーをする？**

```
目的:

□ 知名度拡大
□ 新しい市場開拓
□ ファン獲得
□ 収入増加
□ 経験値稼ぎ
□ ネットワーク拡大

明確な目的:
計画がブレない
```

### ステップ2: ルート設計

**効率的に:**

```
考慮事項:

□ 地理的な近さ
□ 移動時間
□ 移動コスト
□ ギャラ
□ 会場の質

例: 国内ツアー（1週間）

Day 1 (金): 東京 → 名古屋（新幹線2時間）
Day 2 (土): 名古屋 → 大阪（新幹線1時間）
Day 3 (日): 大阪 → 神戸（電車30分）
Day 4 (月): 休息
Day 5 (火): 神戸 → 京都（電車1時間）
Day 6 (水): 京都 → 東京（新幹線2.5時間）

効率的:
無駄な移動を減らす
```

### ステップ3: ブッキング

**会場を確保:**

```
方法:

1. 自分で営業
   各都市のクラブにメール
   EPK、デモ送付

2. ブッキングエージェント
   複数都市を一括で
   手数料10-20%

3. ローカルプロモーターと提携
   その都市の専門家に依頼

4. 既存のつながり
   他都市のDJ仲間に紹介依頼

タイムライン:

3-6ヶ月前:
ブッキング開始

2ヶ月前:
確定

1ヶ月前:
プロモーション開始
```

## レジデンシーDJの役割と責任

### 会場の顔としての役割

**単なるプレイヤー以上の存在:**

```
レジデントDJの責任:

1. 会場のブランディング
   あなたの音楽性が会場の特色に

   例:
   「金曜はテクノの日」
   「このクラブといえばハウス」

   一貫性:
   毎週似た雰囲気を保つ
   でも同じプレイはしない

2. コミュニティ形成
   常連客との関係構築

   方法:
   □ プレイ前後に挨拶
   □ リクエストに柔軟対応
   □ SNSで交流
   □ 誕生日など覚えておく

   結果:
   「あのDJに会いに行く」
   → 強力なファンベース

3. 新規客の獲得
   既存客だけでは縮小

   戦略:
   □ SNSで告知強化
   □ 友人を招待
   □ フライヤー配布協力
   □ 他DJとのコラボ

   目標:
   毎回5-10人の新規客

4. フロアマネジメント
   客層を読んで調整

   シチュエーション別:

   客が少ない時:
   - 落ち着いた曲
   - 徐々にテンポアップ
   - 焦らず待つ

   満員の時:
   - エネルギー全開
   - ピーク作る
   - でも休憩も

   混合客層:
   - 幅広いジャンル
   - 偏らない選曲
   - 全員を楽しませる

5. トラブル対応
   予期せぬ事態への準備

   よくあるトラブル:

   機材故障:
   □ 予備USB常備
   □ 会場スタッフに即連絡
   □ 音楽止めない工夫

   酔客のクレーム:
   □ 冷静に対応
   □ スタッフに任せる
   □ プレイ継続

   リクエスト対応:
   □ 基本は聞く姿勢
   □ 持ってない→謝罪
   □ 雰囲気に合わない→丁寧に断る
```

### プロフェッショナリズム

**信頼される行動:**

```
時間管理:

□ 30分前到着が基本
□ サウンドチェック時間確保
□ 終了時間厳守（延長は要相談）

準備:

□ 曲は前日までに準備
□ 機材メンテナンス済み
□ 服装も考慮（TPOに合わせる）

コミュニケーション:

□ 会場スタッフと良好な関係
□ 問題は早めに相談
□ 感謝の言葉を忘れない

継続的改善:

□ 毎回のプレイを振り返る
□ 客の反応をメモ
□ フィードバックを求める
□ 新曲研究を怠らない

プロの証:
細部への気配り
```

## エージェント・マネージメントとの契約

### いつエージェントが必要？

**タイミングの見極め:**

```
エージェント不要な段階:

- ローカルでのみ活動
- 月1-2回のギグ
- 自分で管理できる範囲
- 知名度がまだ低い

エージェント検討段階:

✓ 週3回以上のギグ
✓ 複数都市からオファー
✓ 事務作業が負担に
✓ ギャラ交渉が難しい
✓ 海外ツアーを考えている

目安:
年間50ギグ以上で検討
```

### エージェントの種類

**役割別の違い:**

```
1. ブッキングエージェント

役割:
□ ギグの獲得
□ スケジュール管理
□ ギャラ交渉
□ 契約書作成

手数料:
ギャラの10-20%

メリット:
- ギグ増える
- 良い条件引き出す
- 時間節約

契約:
案件ごと or 年間契約

2. マネージャー

役割:
□ キャリア全体の戦略
□ ブランディング
□ メディア対応
□ 長期計画立案

手数料:
収入の15-25%

メリット:
- 包括的サポート
- 業界人脈活用
- キャリア加速

契約:
通常1-3年の独占契約

3. ツアーマネージャー

役割:
□ ツアーロジスティクス
□ 移動・宿泊手配
□ 現地対応
□ トラブル解決

報酬:
日給制（¥10,000-30,000/日）
or ツアー収入の一定%

メリット:
- プレイに集中できる
- ストレス軽減
- 効率的な移動

契約:
ツアーごと

選び方:
今の段階で何が必要か
```

### 良いエージェントの見分け方

**チェックポイント:**

```
実績:

□ 類似アーティストの実績
□ 業界での評判
□ 何年営業しているか
□ 得意なジャンル・市場

条件:

□ 手数料は適正か
□ 独占契約か
□ 契約期間
□ 解約条件

相性:

□ コミュニケーションが取りやすい
□ あなたのビジョンを理解
□ 誠実な対応
□ レスポンスが早い

ネットワーク:

□ 会場との関係
□ 他エージェントとの繋がり
□ メディアコンタクト
□ 海外ネットワーク

警告サイン:

✗ 前払い費用要求
✗ 非現実的な約束
✗ 契約を急がせる
✗ 過去のクライアント名を言わない
✗ 契約書が曖昧

慎重に選ぶ:
長い付き合いになる
```

### 契約書の重要ポイント

**必ず確認すべき項目:**

```
1. 契約期間
   - 開始日・終了日
   - 自動更新の有無
   - 更新条件

2. 手数料率
   - パーセンテージ
   - 計算方法（グロス/ネット）
   - 支払いタイミング

3. 独占性
   - 地域の限定（国内のみ等）
   - ジャンルの限定
   - 完全独占 or 部分的

4. 責任範囲
   - エージェントが何をするか
   - あなたが何をするか
   - 費用負担の分担

5. 解約条件
   - 解約通知期間（通常30-90日）
   - ペナルティの有無
   - 解約後の既存契約の扱い

6. 収入の分配
   - 物販収入
   - ストリーミング収入
   - その他収入

7. 紛争解決
   - 仲裁条項
   - 準拠法
   - 管轄裁判所

重要:
不明点は弁護士に相談
サインする前に理解を
```

## ツアーロジスティクス

### 移動計画の詳細

**効率的なルート作成:**

```
計画の基本:

1. 地理的効率

   悪い例:
   Day 1: 東京
   Day 2: 福岡
   Day 3: 大阪
   Day 4: 札幌
   → 移動時間・費用が膨大

   良い例:
   Day 1: 東京
   Day 2: 横浜
   Day 3: 名古屋
   Day 4: 大阪
   → 順に移動、効率的

2. 休息日の設定

   連続ギグの限界:
   - 2-3日連続: 問題なし
   - 4-5日連続: 疲労蓄積
   - 6日以上: 推奨しない

   休息日配置:
   Day 1-3: ギグ
   Day 4: 休息
   Day 5-7: ギグ
   Day 8: 休息

   休息日の過ごし方:
```

```
通常OK

契約で制限されることも

対策:
□ 契約書をよく読む
□ 不明点は確認
□ 誠実に相談

自由:
どこでもプレイ可能

レジデンシーを優先
スケジュール管理重要

会場同士が近すぎない
客層の奪い合いにならない
```

```
健康第一
```
