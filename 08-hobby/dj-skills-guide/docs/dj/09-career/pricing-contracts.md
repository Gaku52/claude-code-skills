# 料金設定と契約

お金の話を避けてはいけない。適正な料金設定と、自分を守る契約書の知識で、プロとして活動する完全ガイドです。DJとしてのキャリアを長期的に持続させるためには、技術面だけでなくビジネス面の知識が不可欠です。本章では、ギャラの相場把握から料金体系の構築、交渉術、契約書作成、税務処理まで、DJ活動のビジネス面を包括的に解説します。

## なぜ料金設定が重要か

### プロとしての価値認識

DJとしての料金設定は、単にお金を稼ぐためだけではなく、自分自身の専門性とプロフェッショナリズムを示す重要な指標です。適正な料金を設定することで、クライアントからの信頼を獲得し、持続可能なキャリアを構築できます。

```
適正な料金の効果:
- 自分の価値を明確に示す
  → クライアントの期待値が適切になる
- 持続可能な活動が可能になる
  → 機材投資、音楽購入、スキルアップに再投資
- プロ意識の表明
  → 準備に十分な時間をかけられる
- 業界全体の健全化
  → 他のDJも正当な報酬を得やすくなる

安すぎる料金の弊害:
- 精神的・肉体的に疲弊する
  → モチベーション低下
- クライアントから軽く見られる
  → 「安いから何でもやって」
- 業界全体の相場を下げる
  → ダンピング競争に陥る
- 再投資ができない
  → 機材更新、音楽購入ができない
- 時間の無駄
  → 他の仕事をした方がよかった

高すぎる料金の弊害:
- ギグの依頼が来ない
  → 実績を積めない
- 期待値が上がりすぎる
  → プレッシャー過多
- クライアントとの関係悪化
  → 「高いだけで大したことない」
- 競合に流れる
  → 適正価格のDJに仕事を取られる

バランスが最も重要:
経験 × スキル × 市場相場 × 需要
→ 適正価格の算出
```

### 料金設定のマインドセット

```
間違ったマインドセット:
- 「好きなことだからお金もらうのは申し訳ない」
  → 好きなことこそプロとして価値がある
- 「まだ上手くないから無料で」
  → 時間と準備に価値がある
- 「他のDJが安いから合わせないと」
  → 安売り競争は全員が損する
- 「お金の話は失礼」
  → プロフェッショナルの証

正しいマインドセット:
- 「適正な報酬はプロの証」
  → 自信を持って提示する
- 「料金は自分への投資回収」
  → 機材、音楽、練習時間のコスト
- 「価値を提供する対価」
  → エンターテインメントの対価
- 「持続可能な活動のために必要」
  → 長く続けるための基盤
```

### 料金設定と市場ポジショニング

```
料金帯によるポジショニング:

低価格帯:
→ 量で勝負（多くのギグ）
→ 初心者〜中級者向け
→ 認知度向上フェーズ
→ リスク: 疲弊、ブランド価値低下

中価格帯:
→ バランス型
→ 中級者〜上級者
→ 安定成長フェーズ
→ 最も持続可能

高価格帯:
→ 質で勝負（厳選されたギグ）
→ プロ・有名DJ
→ ブランド確立フェーズ
→ 付加価値が必要

プレミアム価格帯:
→ 唯一無二の価値
→ トップDJ
→ ブランドアイコンフェーズ
→ 代替不可能な存在
```

## 料金設定の方法

### 考慮すべき要素

**単純に演奏時間だけではない、総合的なコスト計算:**

```
1. 演奏時間（直接コスト）

基本レート:
1時間 ¥10,000-30,000（中級者の場合）

ただし:
30分 = 半額 ではない
→ セットアップ、片付け時間は同じ
→ 最低料金を設定すべき

例:
1時間でも2時間でも
最低¥15,000は変わらない

長時間ディスカウント:
2時間: 基本料金 × 2
3時間: 基本料金 × 2.5（割引）
4時間: 基本料金 × 3（割引）
5時間+: 要相談

─────────────────

2. 準備時間（間接コスト）

含まれる作業:
- リハーサル/練習: 2-4時間
- プレイリスト作成・選曲: 2-6時間
- Hot Cue/メモリーポイント設定: 1-2時間
- 楽曲分析（BPM、Key）: 1-2時間
- 新曲のチェック・購入: 1-2時間/週
- イベントリサーチ: 0.5-1時間
- クライアントとの打ち合わせ: 0.5-1時間
- 移動時間（往復）: 1-3時間

合計例:
2時間ギグの場合
→ 実質8-16時間の労働

時給換算:
¥30,000 ÷ 12時間 = ¥2,500/時間
→ バイトより安い...
→ だから適正価格が重要

─────────────────

3. 経費（固定・変動コスト）

固定費（月額）:
□ 機材の減価償却
  DDJ-FLX10: ¥180,000 ÷ 36ヶ月 = ¥5,000/月
  ヘッドフォン: ¥30,000 ÷ 24ヶ月 = ¥1,250/月
  PCラップトップ: ¥300,000 ÷ 48ヶ月 = ¥6,250/月

□ ソフトウェアサブスクリプション
  rekordbox: ¥1,100/月
  Serato DJ Pro: ¥1,100/月
  DJ Pool/レコードプール: ¥3,000-5,000/月

□ 音楽購入費
  Beatport等: ¥5,000-15,000/月
  レコード: ¥10,000-30,000/月（アナログの場合）

□ ストレージ/クラウド
  SSD、バックアップ: ¥1,000-2,000/月

□ 通信費（一部）
  SoundCloud Pro: ¥800/月
  Mixcloud: ¥500/月

固定費合計: ¥20,000-60,000/月

変動費（ギグごと）:
□ 交通費: ¥500-5,000
□ 駐車場代: ¥500-2,000
□ 食事代: ¥1,000-2,000
□ 衣装代（一部）: ¥1,000-5,000
□ USBメモリ: ¥500-1,000（消耗品）

変動費合計: ¥3,000-15,000/ギグ

─────────────────

4. スキル・経験プレミアム

考慮すべきスキル:
□ DJ歴（年数）
□ 技術レベル（スクラッチ、ビートジャグリング等）
□ ジャンルの専門性
□ 読みの能力（フロアリーディング）
□ MC能力
□ 音楽プロデュース実績
□ メディア露出実績
□ 受賞歴

プレミアム率:
基本料金 × 1.2-2.0
（スキルに応じて）

─────────────────

5. 集客力プレミアム

SNSフォロワー数:
1,000-5,000: +10-20%
5,000-10,000: +20-30%
10,000-50,000: +30-50%
50,000+: +50-100%

実際の集客実績:
50-100人動員: +20%
100-300人動員: +30-50%
300-500人動員: +50-80%
500人+動員: +80-150%

メーリングリスト:
DM等でファンに直接告知できる
→ 確実な集客につながる

─────────────────

6. 会場・イベントの格

会場のランク:
ローカルバー: 基本料金
中規模クラブ: 基本料金 × 1.2-1.5
有名クラブ: 基本料金 × 1.5-2.0
大型会場: 基本料金 × 2.0-3.0

イベントの規模:
50人以下: 基本料金
50-200人: 基本料金 × 1.2
200-500人: 基本料金 × 1.5
500-1,000人: 基本料金 × 2.0
1,000人+: 基本料金 × 2.5+

─────────────────

7. 時間帯・曜日

平日: 基本料金
金曜: 基本料金 × 1.1-1.2
土曜: 基本料金 × 1.2-1.5
祝前日: 基本料金 × 1.2-1.5
年末年始: 基本料金 × 1.5-2.0
GW/お盆: 基本料金 × 1.3-1.5

深夜手当:
0:00以降: +10-20%
3:00以降: +20-30%

─────────────────

■ 総合計算式

基本料金:
(時給 × 演奏時間)

+ 準備コスト:
(準備時間 × 準備時給)

+ 経費:
(固定費按分 + 変動費)

+ プレミアム:
(スキル + 集客力 + 会場格)

+ 時間帯加算:
(深夜・祝日等)

= 合計見積金額

例（中級者、土曜クラブ2時間）:
基本: ¥15,000 × 2時間 = ¥30,000
準備: ¥5,000 × 4時間 = ¥20,000
経費: ¥5,000（交通費等）
プレミアム: × 1.2（土曜）
= (¥30,000 + ¥20,000 + ¥5,000) × 1.2
= ¥66,000

→ 実際の提示: ¥50,000-60,000
  （市場相場も考慮して調整）
```

## 交渉術

### 交渉の準備

**交渉前のリサーチと戦略:**

```
■ 事前リサーチ

会場について:
□ 会場の規模（キャパシティ）
□ 入場料（チャージ）
□ ドリンク価格帯
□ 会場の格（高級 or カジュアル）
□ 通常の出演DJ
□ 過去のイベント

イベントについて:
□ イベントの種類（レギュラー or 単発）
□ 予想集客数
□ 出演者の顔ぶれ
□ チケット価格
□ スポンサーの有無

相手について:
□ プロモーター/オーガナイザーの評判
□ 過去のギャラ実績（推測）
□ 予算感
□ 支払いの信頼性

市場について:
□ 同ジャンルDJの相場
□ 同レベルDJのギャラ
□ 類似イベントの相場

─────────────────

■ 交渉の3つのライン

BATNA（最良の代替案）:
この交渉が決裂した場合の代替案
例: 別のギグ依頼がある

最低ライン（Walk-away Point）:
これ以下では受けない金額
例: ¥15,000

根拠:
- 経費（交通費、準備時間）
- 機会損失（他のギグを断る）
- 自分の最低時給

妥協点（Target）:
現実的な落とし所
例: ¥22,000

希望額（Aspiration）:
理想的な金額
例: ¥30,000

提示額（Opening）:
最初に伝える金額
例: ¥35,000（希望額の10-20%高め）

戦略:
提示額 > 希望額 > 妥協点 > 最低ライン
→ 交渉の余地を確保
→ 最低ラインは絶対に守る
```

### 交渉の実践

**ステップバイステップの交渉フロー:**

```
■ Step 1: 情報収集（先に聞く）

プロモーター:
「今度のイベント出てもらえない？」

あなた:
「ありがとうございます！嬉しいです。
まず、イベントの詳細を教えていただけますか？」

確認事項:
- 日時（いつ？）
- 演奏時間（何時間？）
- 会場（どこ？）
- 時間帯（何時から何時？）
- 観客数（何人規模？）
- 他の出演DJ（誰と？）
- 機材（何が使える？）
- イベントのコンセプト

ポイント:
先にギャラの話をしない
→ 情報を集めてから判断
→ 相手の予算感を推測

─────────────────

■ Step 2: 条件の確認

物理的条件:
□ 演奏時間と休憩
□ サウンドチェック時間
□ 搬入・搬出時間
□ 機材（会場提供 or 持参）
□ 控え室の有無

経済的条件:
□ 交通費の負担
□ 食事・ドリンクの提供
□ 宿泊費（遠方の場合）
□ 駐車場代

その他:
□ 録音・録画の可否
□ フライヤーへの掲載
□ ゲストリスト（何名分）
□ 物販の許可

─────────────────

■ Step 3: ギャラの提示

相手が先に提示する場合:

プロモーター:
「ギャラは¥20,000でどう？」

パターンA（希望以上）:
「承知しました。その条件で大丈夫です」
→ 即決。交渉不要。

パターンB（妥協点付近）:
「ありがとうございます。検討させてください」
→ 少し考える時間をもらう
→ 条件次第では受ける

パターンC（最低ライン付近）:
「ありがとうございます。通常この条件ですと
¥30,000でお受けしているのですが、
¥25,000ではいかがでしょうか？」
→ カウンターオファー

パターンD（最低ライン以下）:
「申し訳ありませんが、その金額では
お受けするのが難しい状況です。
¥XX,000でしたら対応可能です」
→ 明確に断り、代替案を提示

自分が先に提示する場合:

プロモーター:
「ギャラはいくら？」

あなた:
「この条件でしたら、
¥35,000でお受けしております」
→ 希望額の10-20%高めから提示
→ 交渉の余地を残す

─────────────────

■ Step 4: 交渉（すり合わせ）

相手が値切ってきた場合:

プロモーター:
「ちょっと予算が厳しくて...」

対応1: 条件変更で調整
「では、演奏時間を1時間にして
¥25,000ではいかがでしょうか？」

対応2: 付加価値を説明
「私のSNSでのプロモーションも含めて
この価格です。集客にも貢献できます」

対応3: 将来の取引を提案
「今回は¥25,000でお受けして、
次回からは通常料金でお願いできますか？」

対応4: 非金銭的価値を交渉
「ギャラは¥25,000で、
代わりにゲストリスト5名分と
イベント写真のデータをいただけますか？」

対応5: 段階的妥協
「¥30,000 → ¥28,000 → ¥25,000」
→ 段階的に下げることで譲歩感を演出

─────────────────

■ Step 5: 合意と確認

口頭合意後:

あなた:
「では、以下の条件で確定ということで
よろしいでしょうか？

- 日時: XX月XX日 22:00-24:00
- 会場: Club XYZ
- ギャラ: ¥30,000（税込）
- 交通費: 実費別途
- 機材: 会場提供（CDJ-3000 × 2）
- 支払い: 当日現金

後日、契約書をお送りしますので
ご確認ください」

→ 必ず書面（メール含む）で確認
→ 口頭だけは絶対にNG
```

### 交渉の心理テクニック

```
■ アンカリング効果

最初の提示が基準になる
→ 高めから始めることで有利に

例:
あなた: 「¥40,000です」
相手: 「¥25,000で」
落とし所: ¥30,000-35,000

もし最初から¥30,000と言ったら:
相手: 「¥20,000で」
落とし所: ¥22,000-25,000

─────────────────

■ 沈黙の力

提示後、沈黙する
→ 相手が先に譲歩することが多い

あなた: 「¥35,000です」
相手: 「...」
あなた: （沈黙を維持）
相手: 「わかりました」or「¥30,000でどう？」

─────────────────

■ 譲歩の演出

簡単に値下げしない
→ 「考えた結果...」と間を置く

あなた: 「¥35,000です」
相手: 「高いな...」
あなた: 「そうですか...
少し考えさせてください...
では、¥30,000ではいかがでしょう」

→ 5,000円の値引きに価値を感じさせる

─────────────────

■ FOMO（Fear Of Missing Out）

「他にも依頼があるので
早めにお返事いただけると助かります」

→ 焦らせすぎない程度に
→ 嘘はNG（信頼を失う）

─────────────────

■ Win-Win の強調

「お互いにとって良いイベントに
しましょう」

→ 対立ではなく協力関係
→ 長期的な関係を重視

─────────────────

■ 交渉のDO / DON'T

DO（推奨）:
✓ 自信を持って提示する
✓ 柔軟性を見せる
✓ Win-Winを追求する
✓ 礼儀正しく、プロフェッショナルに
✓ 代替案を複数用意する
✓ 相手の立場も理解する
✓ メールの場合は返信を急がない
✓ 実績・価値を具体的に示す

DON'T（避けるべき）:
✗ 最初から最低額を提示する
✗ 感情的になる
✗ 相手を批判する
✗ すぐに妥協する
✗ 曖昧な約束をする
✗ 嘘をつく
✗ 他のDJの悪口を言う
✗ SNSで不満を発信する
```

```
■ 4. 機材・技術要件

□ 会場提供機材
  例: CDJ-3000 × 2, DJM-900NXS2
      モニタースピーカー ×1
      メインスピーカーシステム
□ DJ持参機材
  例: USBメモリ × 2, ヘッドフォン
      ノートPC（バックアップ用）
□ 機材故障時の責任
  例: 会場提供機材の故障: 主催者負担
      DJ持参機材の故障: DJ負担
      不可抗力による損害: 双方協議
□ 音響エンジニアの配置
  例: 主催者が手配（音響スタッフ1名）
□ 技術的要件
  例: 最低PA出力: 2,000W
      モニターの返し: 必須

■ 5. キャンセルポリシー

□ 主催者都合のキャンセル
  30日前まで: 無料（前金がある場合は返金）
  14-29日前: ギャラの25%支払い
  7-13日前: ギャラの50%支払い
  3-6日前: ギャラの75%支払い
  2日前〜当日: ギャラの100%支払い

□ DJ都合のキャンセル
  30日前まで: 無料（前金返金）
  14-29日前: 前金の返金なし
  7-13日前: 違約金ギャラの25%
  3-6日前: 違約金ギャラの50%
  2日前〜当日: 違約金ギャラの100%

  DJ側の義務:
  代替DJの紹介に努める

□ 不可抗力（Force Majeure）
  対象: 天災、疫病、政府の命令、戦争等
  処理: 双方協議の上決定
  原則: 違約金なし、前金は返金

□ イベント内容の大幅変更
  例: 会場変更、日時変更、規模変更
  → DJは契約解除可能（違約金なし）

■ 6. 権利関係

□ 録音・録画の可否
  ライブ録音: 可 / 不可 / 要事前承認
  ライブ配信: 可 / 不可 / 要事前承認
  写真撮影: 可 / 不可 / 要事前承認
  動画撮影: 可 / 不可 / 要事前承認

  DJ側: SoundCloud/Mixcloudへの投稿 可/不可
  主催者側: プロモーション使用 可/不可
  第三者への提供: 不可
  商用利用: 要別途協議・契約

□ 著作権

□ 肖像権
  イベント写真の使用:
    DJ のSNS/ウェブサイトでの使用: 可
    主催者のプロモーション使用: 可
    商用利用（広告等）: 要事前承認
```

## トラブル回避と危機管理

### よくあるトラブルと対策

```
■ トラブル1: ギャラの未払い

発生率: 高い（特に初心者）

原因:
- 口約束のみ
- 主催者の資金不足
- 悪意のある主催者

予防策:
1. 契約書の締結（必須）
2. 前払い50%の要求
3. 信頼できるプロモーターとの取引
4. 過去の実績・評判をリサーチ
5. 知人DJからの情報収集

対処法:
1. まずは穏やかに催促
   「お支払いの件、ご確認いただけますか？」
2. 期限を設定
   「○日までにお振込みをお願いします」
3. 内容証明郵便の送付
   → 法的に記録が残る
4. 少額訴訟（60万円以下）
   → 1回の審理で判決
   → 弁護士不要で可能

─────────────────

■ トラブル2: 急なキャンセル

発生率: 中程度

原因:
- 集客不足
- 天候不良
- 主催者の事情

予防策:
1. キャンセルポリシーの明記
2. 前金の受領
3. キャンセル保険（大型イベント）
4. 代替日程の事前合意

対処法:
1. 契約書に基づきキャンセル料請求
2. 証拠の保全（メール等）
3. 冷静に対処（感情的にならない）

─────────────────

■ トラブル3: 条件変更の要求

発生率: 非常に高い

原因:
- 「もう1時間やって」
- 「音量上げて/下げて」
- 「違う曲かけて」

予防策:
1. 演奏時間を契約書に明記
2. 延長料金を事前に合意
3. 選曲の自由度を確認

対処法:
「延長は可能ですが、追加料金
¥X,000/30分が発生します」
→ 事前に合意していれば問題なし

─────────────────

■ トラブル4: 機材トラブル

発生率: 中程度

原因:
- 古い機材
- メンテナンス不足
- 使用方法の問題

予防策:
1. 事前に機材の確認
2. 責任範囲を契約書に明記
3. バックアップ機材の準備
4. 到着後すぐに機材チェック
5. 問題があれば写真/動画で記録

対処法:
1. 機材の状態を記録（写真）
2. 会場スタッフに報告
3. 契約書に基づき責任を明確化

─────────────────

■ トラブル5: 録音物の無断使用

発生率: やや高い

原因:
- 権利意識の低さ
- 契約書に記載なし
- SNSの気軽な投稿

予防策:
1. 録音・録画の可否を契約書に明記
2. 使用範囲を具体的に規定
3. 事前に確認

対処法:
1. 削除依頼
2. 著作権侵害として通報
3. 損害賠償の請求（悪質な場合）

─────────────────

■ トラブル6: 酔客・暴力

発生率: 低〜中程度

原因:
- クラブ/バーの環境
- アルコール
- 音楽への不満

予防策:
1. DJブースのセキュリティ確認
2. 会場のセキュリティ体制確認
3. 緊急連絡先の確保

対処法:
1. 会場スタッフに即座に通報
2. 自分の安全を最優先
3. 演奏を一時中断も辞さない
4. 暴力を受けた場合は警察に通報

─────────────────

■ トラブル7: 著作権問題

発生率: 低い（但し影響大）

原因:
- 主催者がJASRAC等に未申告
- 違法音源の使用
- 海賊版の配信

予防策:
1. 正規の音源のみ使用
2. 著作権料は主催者負担を確認
3. JASRAC包括契約の有無を確認

対処法:
1. 自分は正規音源を使用していることを証明
2. 主催者に著作権料の支払いを求める
3. 弁護士に相談（必要な場合）
```

```
■ 青色申告 vs 白色申告

白色申告:
- 簡易帳簿でOK
- 特別控除なし
- 初心者向け

青色申告（10万円控除）:
- 簡易簿記
- 10万円の特別控除
- 少し手間が増える

青色申告（65万円控除）:
- 複式簿記
- 65万円の特別控除
- e-Taxで申告
- 会計ソフト推奨

推奨:
年間収入50万円以上なら
青色申告（65万円控除）がお得
→ 税金が大幅に減る
```

### インボイス制度への対応

```
■ インボイス制度とは（2023年10月〜）

概要:
適格請求書（インボイス）を発行できる
事業者として登録する制度

DJに影響するケース:
- 法人（企業）からの依頼
- 消費税を含む請求をする場合
- 年間売上1,000万円超の場合

─────────────────

■ 登録すべきか

登録するメリット:
- 法人クライアントから選ばれやすい
- プロフェッショナルな印象

登録するデメリット:
- 消費税の納税義務
- 事務作業の増加
- 免税事業者のメリットを失う

判断基準:
□ 法人からの依頼が多い → 登録推奨
□ 個人間の取引がメイン → 登録不要の場合も
□ 年間売上1,000万円以下 → 経過措置を活用

─────────────────

■ 免税事業者のまま活動する場合

注意点:
- インボイスを発行できない
- 法人クライアントが消費税を控除できない
  → 値下げ要求の可能性
  → 取引を断られる可能性

対策:
- 個人クライアント中心の活動
- 消費税分の値引きを検討
- 2029年まで経過措置あり
  （段階的に控除率が下がる）
```

## 海外ギグの料金と契約

### 海外での活動

```
■ 海外ギグの料金交渉

通貨:
- USD（ドル）が基本
- EUR（ユーロ）はヨーロッパ
- 現地通貨は避ける（為替リスク）

交渉言語:
- 英語が基本
- エージェント経由が安全

含めるべき経費:
□ フライト代（ビジネスクラス or エコノミー）
□ 空港送迎
□ ホテル宿泊（前泊 + 当日泊）
□ 食事代（Per Diem）
□ ビザ取得費用
□ 海外旅行保険
□ 機材輸送費（必要な場合）

─────────────────

■ 海外契約の注意点

ビザ:
→ 就労ビザが必要な国がある
→ アーティストビザ（P-1, O-1 等）
→ ビザなしで働くと違法

税金:
→ 源泉徴収される場合がある
→ 租税条約の確認
→ 二重課税の回避

通貨:
→ 為替リスクを考慮
→ 送金手数料（SWIFT、Wise等）
→ PayPalは手数料が高い

契約書:
→ 英語の契約書が必要
→ 準拠法の確認
→ 紛争時の管轄裁判所

─────────────────

■ エージェントの活用

海外ギグはエージェント経由が安全

エージェントの役割:
- ギグの獲得
- ギャラの交渉
- 契約書の作成
- ビザの手配
- 旅程の管理
- ギャラの回収

手数料:
ギャラの10-20%（相場）

有名エージェント:
- CAA (Creative Artists Agency)
- WME (William Morris Endeavor)
- Coda Agency
- Paradigm Talent Agency
- AM Only

日本のエージェント:
- DOMMUNICATIONS
- Plus Agency
- 各レーベルのブッキング部門

エージェントなしで:
→ 海外の主催者と直接交渉も可能
→ ただしリスクは自己責任
→ 信頼できるコネクションが必要
```

## よくある質問

### Q1: 最初から高く設定すべき？

**A:** 経験とスキルに見合った適正価格を設定する

```
初心者の場合:
低めからスタート（¥0-10,000）
→ 経験を積むことが優先
→ ただし「タダが当たり前」にしない

3ヶ月後:
¥10,000-15,000に引き上げ
→ 実績ができたら適正価格へ

目安の計算:
アルバイト時給 × 2-3倍
→ 準備時間、スキルを考慮
→ 例: 時給¥1,500 × 2.5 = ¥3,750/時間
→ 2時間ギグ = ¥7,500 + 準備代

ポイント:
安すぎると:
→ 自分の価値が下がる
→ 他のDJにも迷惑
→ プロモーターが「安くて当然」と思う

適正価格を知る:
→ 同レベルのDJに聞く
→ プロモーターに聞く
→ 業界の相場をリサーチ
```

### Q2: 友人のイベントは無料？

**A:** ケースバイケースだが、基準を持つ

```
無料でOK:
- 小規模な友人パーティ（10-20人）
- 商業目的ではない（個人の誕生日等）
- お互い様の関係（次は自分が何かを頼む）
- DJ経験のため（初心者の場合）

有料にすべき:
- 入場料を取るイベント
- 主催者が利益を得る
- 自分の時間・労力が大きい（3時間+）
- 機材を持参する必要がある
- 会場が商業施設

友人価格のルール:
通常料金の50-70%
例: 通常¥30,000 → 友人¥15,000-20,000

最低限もらうべき:
□ 交通費（実費）
□ ドリンク代
□ 食事

一貫性が大事:
「AさんにはA円、Bさんには無料」
→ 不公平感が生まれる
→ 基準を決めておく
```

### Q3: 値切られたらどうする？

**A:** 最低ラインを守りつつ、柔軟に対応する

```
相手: 「¥30,000は高い、¥15,000で」

Step 1: 感謝を伝える
「ご依頼ありがとうございます」

Step 2: 価値を説明
「私の料金には、○時間の準備時間と
○年の経験が含まれています」

Step 3: 代替案を提示
案A: 時間短縮
「では、1時間に短縮して¥20,000は
いかがでしょうか？」

案B: 次回割引
「今回は¥25,000でお受けして、
次回以降は通常料金でお願いします」

案C: 条件交換
「¥20,000でお受けする代わりに、
交通費別途とゲストリスト3名分を
いただけますか？」

案D: 丁重にお断り
「申し訳ありませんが、
¥XX,000以下ではお受けするのが
難しい状況です。
またの機会によろしくお願いします」

大切なこと:
- 安売りしない（自分の価値を守る）
- 感情的にならない
- 長期的な関係を考える
- 断ることも大切なスキル
```

### Q4: 確定申告は必要？

**A:** ほとんどの場合、必要

```
副業DJの場合:
年間所得（収入-経費）> 20万円
→ 確定申告が必要

専業DJの場合:
年間所得 > 48万円（基礎控除）
→ 確定申告が必要

確定申告しないと:
→ 脱税になる可能性
→ 追徴課税（最大40%）
→ 延滞税（年14.6%）

今から準備:
□ 収支記録をつける
□ 領収書を保管する
□ 会計ソフトを導入する
□ 税理士に相談する（大きい収入の場合）
```

### Q5: 値段を公開すべき？

**A:** 基本的には非公開が推奨

```
公開するメリット:
- 問い合わせの質が上がる
  → 予算がない人からの連絡が減る
- 透明性がある
  → 信頼感
- 交渉の手間が減る

非公開のメリット:
- 柔軟に対応できる
  → ケースバイケースで変えられる
- 競合に知られない
  → 価格競争を避ける
- 値上げしやすい
  → 公開価格を変更する必要がない

推奨:
「料金はお問い合わせください」
→ 個別に見積もり
→ 最も柔軟

ウェブサイトには:
「参考価格: ¥XX,000〜」
→ 「〜」で幅を持たせる
→ 最低ラインの目安を示す
```

### Q6: 海外の仕事はどう受ける？

**A:** エージェント経由が安全だが、直接も可能

```
エージェント経由:
メリット: 安全、プロフェッショナル
デメリット: 手数料10-20%

直接:
メリット: 手数料なし、直接交渉
デメリット: リスクが高い

海外の注意点:
□ ビザの確認（就労ビザが必要な国）
□ 税金（源泉徴収、租税条約）
□ 保険（海外旅行保険）
□ 通貨（為替リスク）
□ 契約書（英語、準拠法）
□ 送金方法（Wise推奨）
```

## まとめ

### 料金設定の7原則

```
1. 相場を知る
   └─ リサーチは定期的に
   └─ 同レベルDJの情報交換

2. 自分の価値を知る
   └─ スキル、経験、集客力を客観評価
   └─ 過小評価も過大評価もしない

3. 契約書は必須
   └─ 自分とクライアント双方を守る
   └─ メール確認でも最低限OK

4. 交渉力を磨く
   └─ 準備が9割
   └─ Win-Winを追求

5. 値上げを恐れない
   └─ 成長に合わせて段階的に
   └─ 1ヶ月前に通知

6. 税務をしっかり
   └─ 確定申告は義務
   └─ 経費をきちんと記録

7. 収入を多角化
   └─ DJ出演だけに依存しない
   └─ 複数の収入源を構築
```

### 経験レベル別アクションプラン
