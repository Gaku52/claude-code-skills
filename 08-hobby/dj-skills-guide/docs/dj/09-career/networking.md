## なぜネットワーキングが重要か

**DJ 業界は「誰を知っているか」:**

```
現実:

ギグの80%:
紹介、人脈から

公募:
わずか20%

良いネットワーク:
- ギグ機会↑
- 情報が早い
- サポート得られる
- 孤独感↓

ネットワークなし:
- 機会が限られる
- 情報が遅い
- 一人で悩む

人脈 = キャリアの財産
```

## マインドセット

### Give & Take

**与えることから始める:**

```
間違ったアプローチ:
「仕事ください」
「紹介してください」
→ 自分の利益だけ × NG

正しいアプローチ:
「何かお手伝いできることありますか？」
「素晴らしいイベントでした！」
→ 相手への価値提供 ○ OK

Give First:
先に与える
→ 返ってくる

長期的視点:
すぐに結果を求めない
```

### 本物の関係

**打算的にならない:**

```
NG:
「有名DJだから近づく」
「利用できそう」
→ 見透かされる

OK:
「音楽が好き」
「尊敬している」
→ 本物のつながり

人として:
音楽を愛する仲間として
```

## オフラインネットワーキング

### イベント参加

**顔を出す:**

```
頻度:
週1-2回
ローカルイベントに行く

目的:
- 観察
- 学び
- 交流
- 顔を覚えてもらう

やること:

到着（早めに）:
□ 会場の雰囲気を掴む
□ DJブースを観察
□ 音響チェック

イベント中:
□ 音楽を楽しむ
□ DJのプレイを学ぶ
□ 他の客と自然に交流
□ プロモーターを見つける

休憩時間:
□ バー/喫煙エリアで交流
□ 軽い会話
□ 名刺交換（自然に）

イベント後:
□ DJ/プロモーターに挨拶
  「素晴らしかったです」
□ 感想を伝える
□ SNS フォロー

帰宅後:
□ Instagram でタグ付け投稿
□ ストーリーズでシェア
□ 感謝のDM

継続:
毎回行く
→ 顔を覚えてもらう
```

### 会話のコツ

**自然に、誠実に:**

```
初対面:

NG:
「DJ やってます、ギグください」
→ 押しつけがましい

OK:
「今日のセット、素晴らしかったですね。
特に [曲名/瞬間] が印象的でした」
→ 具体的な賞賛

会話の流れ:

1. 音楽の話（5分）
   - 今日のセット
   - 好きなアーティスト
   - 最近の音楽

2. 共通点を見つける（3分）
   - 同じジャンル
   - 同じ機材
   - 同じ悩み

3. 自己紹介（1分）
   「私も DJ やってるんです」
   → 押しつけがましくなく

4. 連絡先交換（1分）
   「Instagram フォローさせてください」
   → 自然に

終わり方:
「またイベント来ますね！」
→ 次につながる

注意:
- 長話しない（10分以内）
- 忙しそうなら避ける
- 笑顔で
- 聞き役に徹する
```

### 名刺の使い方

**スマートに:**

```
タイミング:

良い:
- 会話が盛り上がった後
- 「連絡先交換しましょう」の流れ
- 相手から聞かれた時

悪い:
- 会った瞬間
- 一方的に押し付ける
- 大量配布

渡し方:

「もしよければ、名刺を」
→ 両手で、丁寧に

受け取る:
相手の名刺も丁寧に
その場で確認
「[名前] さんですね」

後日:
受け取った名刺の裏に
- 日付
- 場所
- 話した内容
→ 次回の話題に
```

### DM の送り方

**スパムにならない:**

```
NG:
「フォローしました！
私の SoundCloud 聴いてください！
ギグください！」
→ 自分勝手、スパム

OK:
「こんにちは！
昨日の [イベント名] でお会いした [名前] です。
[具体的な内容] のお話、とても勉強になりました。
Instagram でもつながれたら嬉しいです。」
→ パーソナル、価値ある

テンプレート:

初回DM:
「
こんにちは、[名前] さん

[場所/イベント] で [行動]
していた [あなたの名前] です。

[具体的な内容] が印象的で、
ぜひフォローさせていただきました。

これからもよろしくお願いします！
」

返信:
- 24時間以内
- 丁寧に
- 質問には詳しく

頻度:
- 週1回まで
- 用事がある時だけ
- スパムしない
```

## コミュニティ貢献

### 価値提供

**もらうより与える:**

```
貢献の方法:

1. 知識シェア

例:
- 「DDJ-FLX4 設定のコツ」投稿
- 「初心者向けビートマッチング」動画
- 「おすすめ曲リスト」シェア

→ 専門家として認識

2. フィードバック提供

他のDJのミックス:
- 丁寧に聴く
- 建設的なコメント
  「00:15 のトランジション最高でした！
   ただ、00:30 の EQ が少し...」
- 励ます

→ 信頼関係

3. イベント手伝い

プロモーターに:
「何か手伝えることありますか？」

役割:
- フライヤー配布
- SNS 拡散
- 当日受付手伝い

→ 信頼↑、ギグ機会↑

4. 紹介・推薦

良いDJ を見つけたら:
プロモーターに紹介
「この人すごいですよ」

→ 感謝される
→ あなたも紹介される

5. 情報シェア

イベント情報:
コミュニティで共有

新技術:
「Rekordbox の新機能すごい」

→ 価値ある存在
```

## メンター探し

### メンターの重要性

**経験者から学ぶ:**

```
メンターのメリット:

1. 的確なアドバイス
   - 経験に基づく
   - あなた専用

2. 近道
   - 失敗を避けられる
   - 効率的な成長

3. ネットワーク
   - メンターの人脈
   - 紹介

4. モチベーション
   - 励まし
   - 目標モデル

メンターなし:
遠回り、孤独
```

### 見つけ方

**アプローチ:**

```
ステップ1: 候補を探す

理想のメンター:
□ 3-5年以上経験
□ あなたと同じジャンル
□ あなたが尊敬している
□ 教えるのが好き
□ アクセスしやすい

候補:
- ローカルの中堅DJ
- DJ スクール講師
- オンラインコミュニティの先輩

ステップ2: 関係構築

□ イベント参加
□ SNS フォロー
□ 質問する（少しずつ）
□ アドバイスを実践
□ フィードバック報告

数ヶ月かけて:
自然に関係を

ステップ3: 依頼

タイミング:
ある程度関係ができたら

依頼:
「もしよろしければ、
定期的にアドバイスいただけないでしょうか。
月1回、30分程度お時間いただけたら幸いです。」

お礼:
- 感謝の気持ち
- 小さなギフト
- 成長を報告

NG:
一方的に頼る
時間を奪う
```

## B2B・コラボレーション

### 他DJ と組む

**シナジー:**

```
B2B (Back to Back) とは:
2人のDJ が交互にプレイ

メリット:
1. 学び合える
2. 相手のファンも獲得
3. 楽しい
4. クリエイティブ

相手の選び方:
□ 同レベル
□ 同ジャンル（or 補完的）
□ 相性が良い
□ お互いに成長意欲

練習:
□ 週1回リハーサル
□ DDJ-FLX4 2台 or CDJ で
□ 録音して改善

イベント:
□ プロモーターに提案
  「B2B セットやりたいです」
□ 自主イベント企画

効果:
ネットワーク2倍
成長速度↑
```

## DJ業界のネットワーキング戦略

### 業界構造の理解

**エコシステムを知る:**

```
DJ業界の主要プレイヤー:

1. クラブ/会場オーナー
   - 最終決定権
   - ブッキング方針
   - 予算管理

2. プロモーター/イベンター
   - DJブッキング
   - イベント企画
   - マーケティング

3. ブッキングエージェント
   - DJ代理人
   - 交渉役
   - スケジュール管理

4. レーベルマネージャー
   - 楽曲リリース
   - アーティスト育成
   - プロモーション

5. メディア/ブロガー
   - 情報発信
   - レビュー
   - 影響力

6. 他のDJ
   - 仲間/競争相手
   - 情報源
   - コラボ相手

7. ファン/観客
   - 最重要
   - 口コミ力
   - サポーター

関係性:
全てが繋がっている
→ 一つのつながりが次を生む
```

### ターゲット別アプローチ

**相手に合わせた戦略:**

```
クラブオーナー向け:

アプローチ:
□ イベント常連になる
□ 他の客を連れてくる
□ トラブルを起こさない
□ プロフェッショナル

タイミング:
忙しくない時間帯
（開店直後、平日昼間）

話題:
- 会場の雰囲気の良さ
- 音響システム
- ビジネス視点

NG:
- 忙しい時に話しかける
- すぐにギグを要求
- 値引き交渉

プロモーター向け:

アプローチ:
□ イベント手伝い
□ SNS で拡散協力
□ 集客に貢献
□ 信頼関係構築

提供価値:
- 集客力
- 音楽性
- プロ意識
- 協力姿勢

話題:
- イベントの成功
- 観客の反応
- 次回企画

NG:
- ギャラ交渉ばかり
- 他イベント批判
- 約束を破る

ブッキングエージェント向け:

アプローチ:

□ フォローアップ

必要なもの:

- SNS実績
- ギグ履歴

タイミング:
ある程度実績ができてから
（最低50ギグ以上）

レーベル関係者向け:

アプローチ:
□ レーベルの音楽を研究

- レーベル曲のプレイ
- フィードバック

NG:
- 質の低いデモ送付
- しつこい連絡
- 他レーベル批判

メディア/ブロガー向け:

アプローチ:
□ 記事を読む
□ SNS フォロー
□ コメント
□ 情報提供

価値提供:
- ニュース性のある情報
- 独占インタビュー
- 業界インサイト
```
