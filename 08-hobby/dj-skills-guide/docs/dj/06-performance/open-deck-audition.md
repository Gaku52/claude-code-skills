```

## 1. オープンデッキとは

### 定義

```
オープンデッキ = 誰でも参加できるDJセッション

特徴:
- 事前登録（通常）
- 短時間プレイ（15-30分）
- 審査なし（ほとんど）
- 経験不問
- 無料または少額

目的: 経験を積む、実力を見せる
```

### 開催場所

**よくある場所:**
```
- 小規模バー
- クラブの早い時間帯（18:00-20:00）
- DJ学校のイベント
- レコードショップ
- コミュニティスペース

頻度:
- 毎週または隔週
- 固定曜日（例: 毎週水曜）

東京、大阪等の都市部に多い
```

---

## 2. トライアウトとは

### 定義

```
トライアウト = オーディション形式のオープンデッキ

特徴:
- 審査あり
- 勝者がレジデンシー獲得
- 競争的
- ハイレベル
- プレッシャー大

例:
「Club Asia DJ Audition」
「Contact DJ Trial」

勝つ = 定期ブッキング
```

### オープンデッキ vs トライアウト

**違い:**
```
オープンデッキ:
✓ カジュアル
✓ 審査なし
✓ 全員プレイ可能
✓ プレッシャー小
✓ 初心者向け

トライアウト:
- 競争的
- 審査あり
- 勝者のみ報酬
- プレッシャー大
- 中級者以上

まずはオープンデッキから
```

## 3. 準備するもの

### 必須

**機材:**
```
□ DDJ-FLX4（持ち込みの場合）
□ ノートPC
□ USBメモリ x 2（メイン + 予備）
□ ヘッドフォン
□ ケーブル類

または:
- 会場のCDJ使用
- USBメモリのみ持参

事前に確認必須
```

### セットリスト

**完璧な30分セット:**
```
準備:
1. 30分ぴったりのセットリスト
   - 7-10曲
   - BPM昇順
   - エネルギーカーブ

2. Plan A, B, C
   - メインセット（Plan A）
   - 代替曲（Plan B）
   - 緊急曲（Plan C）

3. 全曲を100回聴く
   - 完全に覚える
   - ミックスポイント完璧
   - Hot Cue設定済み

準備 = 自信 = 成功
```

## 4. 審査ポイント

### プロモーターが見るポイント

**Top 5:**
```
1. 技術（30点）
   - ミックス精度
   - ビートマッチング
   - EQ使い方
   - トランジション

2. 選曲（30点）
   - 曲の質
   - 流れ
   - 独自性
   - 会場に合う

3. プレゼンス（20点）
   - ステージング
   - 自信
   - エネルギー
   - 観客とのコネクション

4. 時間管理（10点）
   - 時間厳守
   - 30分ぴったり
   - オーバーしない

5. プロフェッショナリズム（10点）
   - 機材の扱い
   - マナー
   - 態度

技術 + 選曲 = 60%
最も重要
```

## 5. プレイ戦略

### 30分で印象を残す

**戦略:**
```
時間が短い:
→ 最初の1分が勝負
→ 強い曲で始める
→ エネルギーを保つ
→ 最後も強く

30分の流れ:

0-5分: 強いスタート
- 印象的な曲
- エネルギー中〜高
- 技術を見せる

5-20分: メインパート
- 流れを作る
- 選曲センスを見せる
- 安定したミックス

20-25分: ピーク
- 最高のエネルギー
- 最高の曲
- 観客を盛り上げる

25-30分: 強い終わり
- 記憶に残る曲
- エネルギー高いまま
- 「もっと聴きたい」

最初と最後 = 最重要
```

### やってはいけないこと

**NG行動:**
```
✗ ウォームアップから始める
   理由: 時間が短い、退屈

✗ 実験的すぎる曲
   理由: リスク高い、観客困惑

✗ 技術を見せつけすぎ
   理由: スクラッチ連発等、やりすぎ

✗ エネルギー低い曲
   理由: 盛り上がらない

✗ 時間オーバー
   理由: 次のDJに迷惑

✗ 緊張で下を向く
   理由: プレゼンスゼロ

安全に、でも印象的に
```

## 7. 失敗例と対処

### 失敗例1: 緊張でミス連発

**問題:**
```
緊張:
→ 手が震える
→ ビートマッチミス
→ 焦る
→ さらにミス
→ パニック
```

**対処:**
```
ミスしたら:
1. 深呼吸
2. 次の曲に集中
3. 笑顔を作る（強制的に）
4. 「大丈夫」と自分に言う

1つのミス ≠ 失敗
全体で判断される

落ち着く
```

### 失敗例2: 時間オーバー

**問題:**
```
30分のはずが35分:
→ 次のDJに迷惑
→ プロモーター激怒
→ 印象最悪
```

**対処:**
```
予防:
✓ タイマーセット（25分でアラーム）
✓ 最後の曲は短め（3分）
✓ 余裕を持つ

もしオーバーしそうなら:
- 1曲飛ばす
- 早めに終わる

時間厳守 = プロの証
```

### 失敗例3: 観客ゼロ

**問題:**
```
オープンデッキ:
→ 観客が少ない（5人）
→ モチベーション下がる
→ 手を抜く
```

**対処:**
```
✓ 観客数に関係なく全力
✓ プロモーターが見ている
✓ 練習の機会と考える
✓ 5人でも全力

観客5人 = 5人のファン獲得チャンス
```

## 11. よくある質問

### Q1: 全く経験なしでも参加できる？

**A: オープンデッキはOK、トライアウトは難しい**

```
オープンデッキ:
✓ 初心者歓迎
✓ 経験不問
✓ 練習の場

ただし:
- 最低限の技術は必要
- ビートマッチングできる
- 30分プレイできる

推奨:
- 自宅で6ヶ月練習
- 友達の前で10回
- その後オープンデッキ

準備してから参加
```

### Q2: 何回参加すればギグ獲得？

**A: 個人差大、平均5-10回**

```
目安:
- 5-10回参加
- プロモーターに顔を覚えてもらう
- 技術向上
- ネットワーク拡大

早い人: 1回で獲得
遅い人: 20回以上

継続 = 成功
```

### Q3: 年齢制限は？

**A: 会場による、18歳以上が多い**

```
一般的:
- 18歳以上（バー、クラブ）
- 20歳以上（アルコール提供）

例外:
- オールエイジズイベント
- DJ学校のイベント

事前確認必須
```

### 3ヶ月前の準備

**スケジュール:**
```
3ヶ月前（12週間前）:

Week 1-2: リサーチ
✓ オーディション情報収集
✓ 会場リサーチ
✓ 過去のイベント動画視聴
✓ 音響システム確認
✓ ジャンル確認

Week 3-4: セットリスト構築
✓ 50曲候補をリスト
✓ BPM整理
✓ キー分析
✓ エネルギーレベル分類
✓ 30曲に絞る

Week 5-6: ミックス練習
✓ 全曲のミックスポイント把握
✓ Hot Cue設定
✓ Loop設定
✓ トランジション練習
✓ 毎日2時間

Week 7-8: セット完成
✓ 30分セット完成
✓ Plan A確定
✓ 毎日1回通し練習
✓ 録音して聞き直す

Week 9-10: 代替案作成
✓ Plan B作成
✓ Plan C作成
✓ 予備曲10曲準備

Week 11-12: 最終調整

✓ 微調整

## 14. デモミックスの作成と提出

### デモミックスとは

**定義:**
```
デモミックス = あなたの実力を示す録音

目的:

- DJ歴: 最低6ヶ月
- 自宅練習: 300時間以上
- 友達の前: 10回以上
- セット完璧: 100回練習

準備 > 早さ
```

### 失敗パターン2: 準備不足

**問題:**
```
セット5回しか練習せず:
→ ミス連発
→ 時間管理失敗
→ プロモーター失望
→ 二度と呼ばれない
```

**回避策:**
```
十分な準備:
- セット: 100回練習
- 全曲暗記
- Plan A/B/C作成
- 録音して分析
- フィードバック収集

準備 = 成功の鍵
```

### 失敗パターン3: フォローアップなし

**問題:**
```
成功したのに:
→ プロモーターに挨拶なし
→ 連絡先交換しない
→ 後日フォローなし
→ ブッキングなし
```

**回避策:**
```
適切なフォロー:
- 当日: 挨拶、連絡先交換
- 3日後: お礼メール
- 月1回: 近況報告
- 新ミックス: 必ず送る

関係構築 = 次のチャンス
```

## 22. オープンデッキ後の自己評価

### パフォーマンス分析

**チェックリスト:**
```
プレイ直後に評価:

技術面（/30点）:
□ ビートマッチング: ミスなし（10点）
□ トランジション: スムーズ（10点）
□ エフェクト: 適切（5点）
□ Hot Cue/Loop: 効果的（5点）

選曲面（/30点）:
□ 曲の質: 高品質（10点）
□ 流れ: ストーリーあり（10点）
□ 独自性: 個性的（5点）
□ 会場適合: ぴったり（5点）

プレゼンス（/20点）:
□ 姿勢: 自信あり（8点）
□ コネクション: 観客と繋がった（7点）
□ 没入感: 楽しんだ（5点）

時間管理（/10点）:
□ 時間: ぴったり30分（10点）

マナー（/10点）:
□ 態度: プロフェッショナル（10点）

合計: /100点

80点以上: 素晴らしい
60-79点: 良い
40-59点: 改善必要
40点未満: 要練習
```

### 改善点の洗い出し

**分析方法:**
```
録音/録画があれば:
1. プレイを聞き直す
   - ミスした箇所
   - 良かった箇所
   - タイミング
   - 選曲の流れ

2. ビデオを見る
   - 姿勢
   - 動作
   - 表情
   - プレゼンス

3. メモを取る
   - 改善点リスト
   - 次回の目標
   - 具体的アクション

録音/録画なし:
1. 記憶を頼りに
   - すぐにメモ
   - 感じたこと
   - 観客の反応
   - プロモーターの反応

2. 友達に聞く
   - 客観的意見
   - 良かった点
   - 改善点

振り返り = 成長
```

## 25. 機材トラブル対策

### よくあるトラブルと対処

**USBメモリ関連:**
```
トラブル1: USBが認識しない
対処:
- 予備USB使用（必ず2本持参）
- 再挿入
- 別のUSBポート試す
- CDJ再起動

予防:
- 事前に会場の機材で確認
- USBは高品質なもの
- 定期的にフォーマット
- バックアップ必須

トラブル2: 曲が見つからない
対処:
- フォルダ構造確認
- 検索機能使用
- 予備プラン曲へ切替

予防:
- シンプルなフォルダ構造
- 曲名統一ルール
- セットリスト紙で持参
```

**ヘッドフォン関連:**
```
トラブル: ヘッドフォン故障
対処:
- 予備イヤフォン使用
- 会場のヘッドフォン借りる
- モニタースピーカーのみでミックス

予防:
- 高品質ヘッドフォン
- ケーブル確認
- 予備イヤフォン携帯
```

**音響トラブル:**
```
トラブル: 片方のチャンネル音が出ない
対処:
- ケーブル確認
- ゲイン確認
- スタッフに報告
- 片方で続行

予防:
- サウンドチェック徹底
- ケーブル動作確認
```

## まとめ

- **オープンデッキ**: 誰でも参加可能、15-30分プレイ、経験を積む場
- **トライアウト**: オーディション形式、審査あり、勝者がレジデンシー
- **準備**: 完璧な30分セット、Plan A/B/C、100回練習
- **審査**: 技術30点、選曲30点、プレゼンス20点、時間管理10点、マナー10点
- **戦略**: 強い始まり、メイン、ピーク、強い終わり、最初と最後が重要
- **成功の秘訣**: 完璧な準備、独自性、観客を見る、プレゼンス
- **失敗対処**: ミスしても落ち着く、時間厳守、観客少なくても全力
- **見つけ方**: Google検索、SNS、クラブに直接問い合わせ、コミュニティ参加
- **アプローチ**: 終了後挨拶、連絡先交換、後日フォロー、定期連絡
- **デモミックス**: 30-60分、高音質、SoundCloudアップロード、プロモーターに送る
- **当日**: 2時間前到着、サウンドチェック、メンタル準備、楽しむ
- **合格後**: 継続的努力、ネットワーク構築、スキル向上、次のステップへ
- **長期戦略**: 1年目基盤、2年目拡大、3年目プロ確立、5年目トップDJへ
