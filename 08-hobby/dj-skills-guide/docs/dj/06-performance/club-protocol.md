```
マナー違反:
- 遅刻
- 機材を雑に扱う

→ 二度と呼ばれない

プロの振る舞い:
- 時間厳守
- 機材を大切に
- 他DJと協力

→ リピートブッキング

プロトコル = プロの証
```

## 2. 時間厳守の重要性

### 絶対に遅刻しない

**到着時間:**
```
ルール: 自分のセット時間の60分前到着

例:
あなたのセット: 22:00-23:00
到着時間: 21:00

理由:
- セットアップ時間（15分）
- サウンドチェック（10分）
- 前のDJ観察（20分）
- 余裕（15分）

遅刻 = 最悪のマナー違反
```

### 遅刻しそうな場合

**緊急時の対応:**
```
もし遅刻しそうなら:

1. 即座に連絡（30分前に気づいた時点で）
   - プロモーターに電話
   - 理由と到着時刻を伝える

2. 謝罪
   「申し訳ありません、電車遅延で20分遅れます」

3. 到着したら再度謝罪

4. 挽回
   - 完璧なセット
   - スタッフに感謝

理由なき遅刻 = アウト
```

## 3. DJブースでのマナー

### ブースは神聖な場所

**ルール:**
```
✓ DJと関係者のみ
✓ 飲み物は蓋付きのみ
✓ 機材に触れない（自分のターン以外）
✓ 大声で話さない
✓ 写真は許可を得てから

✗ 友達を連れ込む
✗ お酒をこぼす
✗ 勝手に機材を触る
✗ DJの集中を妨げる
```

### 前のDJへの敬意

**スムーズな交代:**
```
あなたのターンが来たら:

1. ブースに入る許可を得る
   - 「入ってもいいですか？」
   - 前のDJに確認

2. 静かにセットアップ
   - 前のDJがプレイ中
   - 邪魔しない
   - 静かに準備

3. 最後の曲まで待つ
   - 前のDJが終わるまで
   - 焦らない

4. 交代時に挨拶
   「お疲れ様でした」
   「良いセットでしたね」

敬意を払う
```

### 次のDJへの配慮

**スムーズな引き継ぎ:**
```
あなたのセットが終わったら:

1. 次のDJにスペースを譲る
   - 機材を片付ける
   - ブースを空ける

2. 引き継ぎ情報
   （もし聞かれたら）
   - 「観客はTech Houseで盛り上がってました」
   - 「エネルギーは高めです」

3. 挨拶
   「頑張ってください」

4. ブースから退出
   - すぐに退出
   - 長居しない

次のDJのため
```

### 他人の機材は自分の物以上に大切に

**基本原則:**
```
クラブの機材:
- 数百万円
- 壊したら弁償
- 超高額

扱い方:
✓ 優しく
✓ 丁寧に
✓ 自分の物より大切に

✗ 乱暴に
✗ 雑に
✗ 無理に動かす
```

### やってはいけないこと

**絶対NG:**
```
✗ フェーダーを無理やり動かす
✗ ノブを限界まで回す
✗ ボタンを強く叩く
✗ ケーブルを引っ張る
✗ 機材に飲み物を置く
✗ 機材に寄りかかる
✗ ジャンプして振動を与える

壊したら:
→ 弁償（数十万円）
→ 二度と呼ばれない
→ 業界で悪評

```

```
✓ クラブのスペースに配慮
```

## 10. トラブル対応

### 機材トラブル

**対処法:**
```
機材が故障した:

1. 音響スタッフに即座に報告
   - 手を上げる
   - 「機材トラブルです」

2. 冷静に対応
   - 焦らない
   - 笑顔を保つ

3. スタッフに任せる
   - 自分で無理に直さない
   - プロに任せる

4. 予備プラン
   - 予備USB
   - 別の機材

5. 謝罪（もし自分のミスなら）

プロは冷静
```

### 観客とのトラブル

**対処法:**
```
観客が迷惑行為:

1. 無視（最初は）
   - 反応しない
   - プレイに集中

2. 悪化したら
   - セキュリティに合図
   - 自分で対応しない

3. 絶対に喧嘩しない
   - プロフェッショナル
   - 大人の対応

セキュリティの仕事
DJは音楽に集中
```

## 11. DJ交代時の引き継ぎプロトコル

### 交代の基本フロー

**タイムライン:**
```
[前のDJセット終了15分前]
あなた: ブースに入る許可を得る
      静かにセットアップ開始

[終了5分前]
あなた: 最終チェック完了
      次の曲を選択済み
      前のDJの最後の曲を観察

[終了1分前]
あなた: ミキサーのフェーダーをチェック
      ヘッドホン準備完了
      最初の曲をCue

[交代時刻]
前のDJ: 最後の曲のアウトロ
あなた: 「お疲れ様でした」
      素早く接続切り替え

[交代後30秒]
あなた: 最初の曲を開始
前のDJ: ブース退出
      機材を片付け
```

### 引き継ぎ情報の伝達

**何を伝えるか:**
```
前のDJから聞くべきこと:
✓ 「フロアの雰囲気はどうですか？」
✓ 「BPMはどのくらいで回してましたか？」
✓ 「盛り上がっている曲のジャンルは？」
✓ 「機材の調子はどうですか？」

例:
前のDJ: 「126-128BPMのTech Houseで
         盛り上がってました。
         フロアのエネルギーは高めです。
         左のCDJが少し調子悪いかもしれません」

あなた: 「ありがとうございます。参考にします」
```

### スムーズな音楽的接続

**ジャンル・BPMの引き継ぎ:**
```
理想的な接続:

前のDJ終了時:
- ジャンル: Tech House
- BPM: 126
- エネルギー: High

あなたの開始:
- ジャンル: Tech House → Progressive House（段階的変化）
- BPM: 126-128（±2BPM以内）
- エネルギー: High維持 or 少し下げる

避けるべき急激な変化:
✗ Tech House 126BPM → Techno 140BPM（急激すぎる）
✗ House → Hip Hop（ジャンル飛躍）
✗ High Energy → Ambient（エネルギー落差大）

段階的な変化がプロ
```

### 機材引き継ぎチェックリスト

**確認項目:**
```
□ ミキサー設定
  - EQすべて12時位置
  - フェーダーすべて下
  - クロスフェーダー中央
  - エフェクトOFF

□ CDJ/ターンテーブル
  - Pitch設定リセット（0%）
  - プラッター停止
  - 不要なUSB抜く

□ ケーブル接続
  - 緩んでいないか
  - ヘッドホンジャック
  - 電源ケーブル

□ 音響設定
  - Master音量変更していないか
  - モニター音量適切か
  - 外部エフェクト状態

□ 環境
  - ブース内整理整頓
  - 飲み物片付け
  - 私物撤去

引き継ぎ完璧 = プロ
```

## 12. バーカウンター・VIPエリアとの連携

### バーカウンターとの関係

**なぜ重要か:**
```
バーカウンター = クラブの収益源

関係性:
- DJが盛り上げる
- 観客が踊る
- 喉が渇く
- ドリンク売上増加
→ クラブが儲かる
→ DJのブッキング継続

良いDJ = ドリンク売上を上げるDJ
```

### バーテンダーとのコミュニケーション

**やるべきこと:**
```
✓ 到着時に挨拶
  「こんばんは、今夜DJを担当します」

✓ ドリンクを頼む時は丁寧に
  「お忙しいところすみません」
  「水をいただけますか？」

✓ チップ（あれば）
  日本では必須ではないが、
  500円チップは好印象

✓ 終了時にお礼
  「ありがとうございました」

✓ SNSでバーをタグ付け
  「素晴らしいカクテルでした @バー名」

良好な関係 = 優遇される
```

### VIPエリア対応

**VIPゲストへの配慮:**
```
VIPエリアとは:
- 高額料金を払ったゲスト
- クラブの上顧客
- 特別待遇

DJの対応:
✓ VIPエリアも意識した音楽
  （フロアだけでなく全体を見る）

✓ リクエストには柔軟に
  VIPからのリクエスト
  → プロモーターに確認
  → 可能なら対応

✓ 目線を配る
  フロアだけでなく
  VIPエリアにも視線

✓ プロモーターの指示に従う
  「VIPが○○をリクエストしています」
  → 可能な限り対応

✗ VIPを無視
✗ リクエスト拒否（頭ごなし）
✗ フロアだけ見る

VIP満足 = クラブ繁盛 = あなたのブッキング
```

### ボトルサービス時の対応

**ボトルが入った時:**
```
ボトルサービス = 高額シャンパン等の注文

DJの対応:

1. 視認
   VIPエリアでボトルが運ばれる
   → スタッフが光る演出

2. 音楽で盛り上げる
   - エネルギー高めの曲
   - 盛り上がる瞬間作る
   - 「祝福」の雰囲気

3. 目線を送る
   VIPテーブルに軽く視線
   → 認知されたと感じさせる

4. プロモーターと連携
   複数ボトルが入る
   → クラブ大成功の夜
   → あなたの株も上がる

ボトルサービス = クラブの大収益
DJも協力して盛り上げる
```

## 13. トラブル時の対応プロトコル

### 音響トラブル詳細対応

**フィードバック（ハウリング）:**
```
症状:
「キーン」という高音のノイズ

原因:
- マイクとスピーカーの位置
- EQ設定ミス
- 音量過大

即座の対応:

1. 音量を下げる（Master）
   急いで音量ダウン

2. 音響スタッフに合図
   手を上げて「フィードバック」と伝える

3. 自分でできること
   - 問題の周波数帯をEQでカット
   - マイク使用中止（もし使っていたら）

4. スタッフに任せる
   根本的な解決はプロに

5. 観客に謝罪（軽く）
   笑顔で手を上げて「ごめんなさい」ジェスチャー

フィードバック = 即対応
```

### 機材故障時の代替手段

**各機材のバックアッププラン:**
```
CDJ故障:
→ もう片方のCDJで1台プレイ
→ ループ機能活用
→ 長めの曲選択
→ スタッフに修理依頼

ミキサー故障:
→ 音響スタッフ即呼ぶ
→ 予備ミキサーに交換
→ 最悪は音源直結

ヘッドホン故障:
→ 予備ヘッドホン使用
→ なければモニタースピーカーで確認
→ ノーヘッドホンDJ（上級テク）

PC/コントローラー故障（持ち込み時）:
→ クラブ常設機材に切り替え
→ USBメモリから再生
→ スマホから緊急プレイ（Bluetooth DJ）

最悪のシナリオ:
全機材ダウン
→ 音響スタッフと協力
→ BGM流す
→ 観客にアナウンス
→ 復旧まで待機

事前準備:
✓ 予備ヘッドホン
✓ USBメモリ複数
✓ ケーブル予備
✓ スマホに音楽ダウンロード済み

備えあれば憂いなし
```

```
継続的な関係構築
```

## 18. 最終アドバイス：プロのマインドセット

### プロフェッショナルとは

**プロの定義:**
```
技術だけでは不十分:
- DJスキル = 50%
- 人間性 = 30%
- プロトコル = 20%

全てが揃って初めてプロ

プロフェッショナル = 信頼される人
```

### 長期的キャリア構築

**持続可能なDJキャリア:**
```
短期思考 vs 長期思考:

短期:
- ギャラだけ重視
- 1回きりのブッキング
- 評判気にしない
→ すぐに行き詰まる

長期:
- 関係構築重視
- リピートブッキング
- 評判を大切に
→ 持続可能なキャリア

5年後、10年後も活躍するために
今日のプロトコルを守る
```

### 最も重要なこと

**絶対に忘れてはいけないこと:**
```
1. 時間厳守
   → 全ての基本

2. 感謝の気持ち
   → 全てのスタッフ、DJ、観客に

3. 謙虚さ
   → どんなに上手くなっても

4. プロフェッショナリズム
   → 常に冷静、常に丁寧

5. 音楽への愛
   → これが全ての原点

これらを忘れなければ
必ず成功する
```

### ケーススタディ2: 機材トラブル発生

**シナリオ:**
```
状況:
- セット中盤（30分経過）
- 左のCDJが突然停止
- フロアは盛り上がっている
- 次の曲が再生できない
```

**間違った対応:**
```
✗ パニックになる
✗ 大声で「壊れた！」と叫ぶ
✗ 勝手に機材をいじり回す
✗ 音響スタッフを怒鳴る
✗ セットを投げ出す

→ 最悪の結果
```

**正しい対応:**
```
即座（5秒以内）:
1. 冷静を保つ
   - 笑顔を保つ
   - 深呼吸

2. 右のCDJで継続
   - 右の曲をループ
   - エネルギー維持

3. 音響スタッフに合図
   - 手を上げる
   - 「左のCDJ停止しました」

音響スタッフ対応中（1分）:
4. 右のCDJで1台DJプレイ
   - ループ活用
   - 長めの曲選択
   - フロアの雰囲気維持

5. 観客に軽く謝罪ジェスチャー
   - 笑顔で手を上げて
   - 「ごめんなさい」

修理完了後:
6. 音響スタッフにお礼
   - 笑顔で「ありがとうございます」

7. セット再開
   - スムーズに両CDJ使用
   - エネルギー回復

セット終了後:
8. 改めて音響スタッフにお礼
   「さっきはありがとうございました。
    助かりました」

9. プロモーターに報告
   「トラブルありましたが、
    音響スタッフのおかげで
    無事に終了できました」

結果:
→ 観客はトラブルを気にしない
→ プロモーター「冷静な対応、素晴らしい」
→ 音響スタッフ「また一緒に仕事したい」
→ 評価が上がる

プロの対応 = トラブル時こそ真価
```

### ケーススタディ3: VIPからの無理なリクエスト

**シナリオ:**
```
状況:
- セット中盤
- Tech House 127BPMでプレイ中
- フロアは完璧に盛り上がっている
- VIPテーブルから
  「今すぐJ-POP○○かけろ」とリクエスト
```

**間違った対応:**
```
✗ 「無理です」と頭ごなしに拒否
✗ VIPを無視
✗ セットの流れを完全に壊してJ-POPをかける

→ どちらもNG
```

**正しい対応:**
```
即座:
1. 笑顔で対応
   「リクエストありがとうございます」

2. プロモーターに目配せ
   - プロモーターを呼ぶ

プロモーター到着:
3. 状況説明
   「VIPから○○のリクエストがあったんですが、
    今のセットには合わないんです。
    どうしましょうか？」

4. プロモーターの判断を待つ

パターンA（プロモーター「対応して」）:
5. セットの最後に配置
   「わかりました。ラスト付近で
    BPMとジャンルを徐々に変えて
    自然に繋ぎます」

6. 計画的にBPMダウン
   127 → 124 → 120 → J-POP対応BPM

7. ジャンル徐々に変化
   Tech House → House → Pop House → J-POP

8. VIPに向けて軽く視線
   - 「かけますよ」のサイン

9. J-POPを自然に繋ぐ

10. セット終了後、VIPテーブルへ
    「リクエストありがとうございました」

パターンB（プロモーター「断っていい」）:
5. プロモーターからVIPに説明
   - DJ側は関与しない

6. あなたはセット継続
   - Tech Houseで盛り上げ続行

結果:
→ プロモーター「適切な対応」
→ VIP満足（対応した場合）
→ フロアの流れも維持
→ プロフェッショナル評価

ポイント:
- VIPリクエストは必ずプロモーター経由
- 自己判断で対応しない
- 音楽的整合性を説明
- 可能なら対応、無理なら丁寧に断る
```

### ケーススタディ4: 他DJの時間オーバー

```

   - イライラしない
   - 表情に出さない

   「前のDJさん、まだ続けますか？」

   「お疲れ様でした」

   - 文句は言わない
   - スムーズに交代

23:17セット開始:
7. 予定より17分短いが全力でプレイ
   - 手を抜かない
   - プロフェッショナル

24:00セット終了:

   - 自分は絶対に延長しない

終了後:
9. プロモーターに報告

    問題ありませんでした」
   - 冷静に事実だけ伝える

翌日:
11. SNSでは一切触れない
    - ポジティブな投稿のみ

結果:
→ プロモーター「大人の対応、素晴らしい」
→ 次回からあなたの時間枠を優先
→ 信頼度大幅アップ
→ 前のDJとも良好な関係維持

ポイント:
- 問題はプロモーターに任せる

- 他DJを批判しない
- 短い時間でも全力
- 時間厳守は自分だけでも守る

他人の行動は変えられないが
自分の評判は自分で作る
```

## まとめ

- **時間厳守**: 60分前到着、遅刻は最悪のマナー違反、連絡必須
- **DJブース**: 神聖な場所、前後のDJに敬意、静かにセットアップ
- **機材**: 優しく丁寧に、壊したら弁償、飲み物注意
- **スタッフ**: 全員に敬意、挨拶とお礼、音響スタッフは重要
- **他DJ**: 仲間、ネットワーキング、褒める、連絡先交換
- **プロモーター**: 最重要人物、感謝の気持ち、イベント後にお礼
- **音響スタッフ**: 音を作る人、丁寧なコミュニケーション、お礼必須
- **NG行動**: 遅刻、泥酔、機材破損、時間オーバー、悪口厳禁
- **トラブル**: 冷静に対応、スタッフに任せる、プロフェッショナル
- **ギャラ**: 事前にメールで交渉、当日はNG、丁寧に
- **DJ交代**: スムーズな引き継ぎ、情報共有、音楽的接続を意識
- **バー・VIP**: 連携して盛り上げる、良好な関係、クラブ全体を見る
- **海外**: 文化の違い尊重、契約慎重に、言語準備、タブー注意
- **運営視点**: ビジネス理解、再ブッキング狙う、Win-Win関係
- **チェックリスト**: 前日、当日、セット中、終了後、翌日のフォロー
- **マインドセット**: プロ意識、長期思考、謙虚さ、感謝、音楽愛
