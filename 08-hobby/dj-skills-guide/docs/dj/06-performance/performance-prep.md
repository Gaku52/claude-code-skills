# パフォーマンス準備

本番で最高のパフォーマンスを発揮するための完璧な準備方法を学びます。

## この章で学ぶこと

- 完璧な準備の重要性
- 持ち物チェックリスト
- セットリスト最終確認
- メンタル準備
- 本番前ルーティン
- 食事と睡眠管理
- 服装選び
- 当日のスケジュール
- 緊急時の対応

## なぜ準備が重要か

**準備不足 vs 完璧な準備:**
```
準備不足:
- 忘れ物
- セットリスト未完成
- 睡眠不足
- 緊張しすぎ
→ 失敗

完璧な準備:
- 全て揃っている
- セットリスト完璧
- 体調万全
- 自信
→ 成功

準備 = 成功の80%
```

### 必須アイテム（絶対忘れない）

**機材関連:**
```
□ DDJ-FLX4

これを忘れたら終わり

```

**ケーブル類（予備）:**
```
□ 予備USBケーブル
□ 予備RCAケーブル（DDJ-FLX4用）
□ 変換アダプター（6.3mm ↔ 3.5mm）
□ 電源タップ（任意）

ケーブルは消耗品
予備必須
```

**デジタル準備:**
```
□ Rekordboxに全曲ロード済み
□ Hot Cue全て設定済み
□ プレイリスト完成
□ 全曲解析済み
□ PCのバッテリー70%以上
□ Rekordboxライセンス有効
```

### 推奨アイテム

**パフォーマンス向上:**
```
□ 水（500ml）- 水分補給
□ タオル（汗拭き用）
□ ガムまたはミント（口臭対策）
□ スマートフォン（充電済み、Shazam用）
□ モバイルバッテリー
□ 名刺（ネットワーキング用）
□ ペン（セットリスト調整用）

**緊急時対応:**
```
□ 胃腸薬（緊張で腹痛の場合）
□ 痛み止め（頭痛の場合）
□ 目薬（長時間プレイ）
□ リップクリーム（乾燥対策）
□ マスク（感染症対策）

```

## 9. 緊急時の対応

### 忘れ物をした場合

**DDJ-FLX4を忘れた:**
```
対処法:
1. 家に戻る（時間があれば）
2. プロモーターに連絡
3. 借りられるか確認
4. CDJでプレイ（可能なら）

予防:
✓ 前日夜にバッグに入れる
✓ 出発前に必ずチェック
```

**USBメモリを忘れた:**
```
対処法:
1. 予備USBを使う
2. Dropbox/Google Driveから再ダウンロード（事前にアップロード必須）
3. ストリーミング（Beatport Link等）

予防:
✓ 予備USBメモリ必須
✓ クラウドバックアップ
```

### 体調不良の場合

**当日体調が悪い:**
```
軽い体調不良:
- 痛み止め、胃腸薬
- 水分補給
- なんとか乗り切る

重い体調不良:
- プロモーターに連絡
- 代わりのDJ手配
- キャンセル

無理は禁物
```

## 11. 会場リサーチと事前準備

### 1週間前: 会場の下調べ

**会場情報の収集:**
```
調べること:
□ 会場の住所と最寄り駅
□ 電車の時刻表（往復）
□ 徒歩ルート（駅→会場）
□ 会場の写真（Google Maps等）
□ 会場のキャパシティ
□ 音響システム（可能なら）
□ 過去のイベント映像（YouTubeやInstagram）

なぜ重要か:
- 当日迷わない
- 時間配分が正確
- 会場の雰囲気を予測
- メンタル的に安心
```

**会場特性の理解:**
```
小規模クラブ（100名以下）:
- 観客との距離が近い
- アイコンタクト重要
- アットホームな雰囲気
- エネルギーは控えめスタート

中規模クラブ（100-300名）:
- バランス型
- エネルギー構築が重要
- ピーク時が勝負
- 標準的なセットリスト

大規模クラブ（300名以上）:
- 圧倒的なエネルギー
- 大きなビルドアップ
- 派手な演出
- ハイエナジーな選曲
```

### 3日前: プロモーターとの連絡

**確認すべき事項:**
```
□ プレイ時間（正確な開始時刻と長さ）
□ ジャンル/スタイル（期待されている音楽性）
□ ターゲット層（年齢、性別、音楽嗜好）
□ 前後のDJ（誰がプレイするか）
□ 機材（DDJ-FLX4持ち込みOKか）
□ サウンドチェック時間
□ 駐車場/搬入口（機材持ち込みの場合）
□ ドリンクチケット/出演料
□ 緊急連絡先

## 13. メンタルトレーニング・テクニック

### 呼吸法

**4-7-8呼吸法（緊張緩和）:**
```
やり方:
1. 口から完全に息を吐く（フーッ）
2. 鼻から4秒かけて息を吸う（1, 2, 3, 4）
3. 7秒間息を止める（1, 2, 3... 7）
4. 口から8秒かけて息を吐く（フーーーー）

繰り返し: 4回

効果:
- 心拍数が下がる
- リラックス

→ 冷静にACアダプターを接続し直す
→ Rekordboxは自動保存しているので再起動でOK
```

## 18. セットリスト最終確認 - 高度なテクニック

### キー（調性）による曲順最適化

**カメロットホイールを活用した曲順:**
```
カメロットホイールとは:
- 12の長調と12の短調を時計型に配置
- 隣接するキーの曲は自然にミックスできる
- Rekordboxでは自動でキーが検出される

ミキシング・ルール:
1. 同じキー同士: 完璧な相性（例: 8A → 8A）
2. 隣接キー: 自然な流れ（例: 8A → 7A, 8A → 9A）
3. 平行調: エネルギー変化（例: 8A → 8B）
4. 対角キー: 劇的な変化（例: 8A → 2A）← 上級者向け

セットリスト例（60分House Set）:
曲1:  8A (Cm)   - 120 BPM - ウォームアップ
曲2:  8A (Cm)   - 121 BPM - 安定
曲3:  9A (Dm)   - 122 BPM - 自然な上昇
曲4:  9B (F)    - 124 BPM - 明るく変化
曲5:  10B (Bb)  - 125 BPM - エネルギー増加
曲6:  10A (Em)  - 126 BPM - ダークに
曲7:  11A (Bm)  - 127 BPM - テンション上昇
曲8:  11B (D)   - 128 BPM - 明るいピーク
曲9:  12B (Eb)  - 129 BPM - クライマックス
曲10: 12A (Abm) - 130 BPM - 最高潮
曲11: 11A (Bm)  - 128 BPM - クールダウン開始
曲12: 10A (Em)  - 126 BPM - 余韻
曲13: 9A (Dm)   - 124 BPM - フィナーレ

キーの流れがスムーズ
→ ミックスが自然に聴こえる
→ 観客に違和感を与えない
```

### エネルギーマネジメント詳細

**エネルギーレベルの7段階管理:**
```
レベル1（10-20%）: アンビエント
- BPM: 100-115
- 特徴: 静かで穏やか
- 用途: 開場時BGM、深夜のクールダウン

レベル2（20-35%）: ディープ
- BPM: 115-122
- 特徴: グルーヴィー、落ち着き
- 用途: ウォームアップ序盤

レベル3（35-50%）: グルーヴ
- BPM: 122-126
- 特徴: 体が動き始める
- 用途: ウォームアップ後半

レベル4（50-65%）: ドライビング
- BPM: 126-128
- 特徴: 踊りたくなる
- 用途: ビルドアップ前半

レベル5（65-80%）: ハイエナジー
- BPM: 128-130
- 特徴: フロアが動く
- 用途: ビルドアップ後半

レベル6（80-90%）: ピーク
- BPM: 130-133
- 特徴: 最大限の盛り上がり
- 用途: クライマックス

レベル7（90-100%）: アンセム
- BPM: 130-135+
- 特徴: 全員が手を上げる
- 用途: 最高潮の瞬間（1-2曲のみ）

エネルギーの上げ下げを意識的に行う
一直線に上げるのではなく、波を作る
```

## 19. 会場下見とサウンドチェック手順

### 会場の音響特性を把握する

**音響チェックで確認すべきポイント:**
```
会場の音響特性:

1. 残響時間（リバーブ）
   - コンクリート壁: 残響が長い → エフェクトのリバーブは控えめに
   - カーテンや布張り: 残響が短い → エフェクト自由に使える
   - 確認方法: 手を叩いて残響を聴く

2. 低音の回り方
   - 小さい部屋: 低音がこもりやすい → BASSを若干下げる
   - 天井が高い: 低音が拡散 → BASSを少し上げる
   - 確認方法: キックの強い曲でテスト

3. スピーカーの配置
   - メインスピーカーの位置と向き
   - サブウーファーの位置
   - モニタースピーカーの有無
   - デッドスポット（音が届きにくい場所）

4. フロアの広さと形状
   - 正方形: 均一な音響
   - 長方形: 奥と手前で音量差
   - L字型: 死角がある
   - 柱や障害物の有無

これらを把握することで
EQの調整方針が決まる
```

**サウンドチェックの実施手順（到着後15分間）:**
```
手順1: テストトラックの準備（2分）
  - 自分がよく知っている曲を使う
  - キック、ベース、ボーカル、ハイハットが
    バランスよく含まれている曲が理想
  - 推奨: セットリストの5曲目あたり
    （中間的なエネルギーの曲）

手順2: 小音量でのテスト（3分）
  - マスターボリュームを20%程度で再生
  - 左右のスピーカーから音が出ているか
  - ノイズやハム音がないか
  - 接続に問題がないか

手順3: 通常音量でのテスト（5分）
  - マスターボリュームを60-70%に上げる
  - LOW/MID/HIGHのバランス確認
  - モニタースピーカーの音量調整
  - ヘッドフォンの音量調整
  - フロアの各位置で音を確認（可能なら）

手順4: ピーク音量の確認（3分）
  - マスターボリュームを80-90%に
  - 歪みが出ないか確認
  - ピークメーターが赤にならないか
  - 本番中の最大音量を把握

手順5: 最終調整（2分）
  - 必要に応じてEQを微調整
  - モニターの角度調整
  - ヘッドフォンのケーブル取り回し確認
  - 全てを元の音量に戻す

サウンドチェック完了
→ 安心して本番に臨める
```

### DJ交代時のスムーズな引き継ぎ

**前のDJからの引き継ぎ手順:**
```
前のDJのセット終了5分前:

1. ブース横でスタンバイ
   - 機材を手元に準備
   - USBケーブルをすぐ挿せる状態

2. 前のDJに挨拶
   - 「お疲れ様です、次よろしくお願いします」
   - 最後の曲のBPMとキーを確認（可能なら）

3. 前のDJの最終曲中に接続
   - DDJ-FLX4をテーブルに設置
   - USBケーブル接続
   - Rekordbox起動
   - ヘッドフォン接続

4. 引き継ぎ方法の確認
   パターンA: クロスフェードで引き継ぎ
   → ミキサーの空きチャンネルに接続
   → 前のDJが徐々にフェードアウト
   → 自分がフェードイン

   パターンB: 完全停止後に開始
   → 前のDJが曲を終わらせる
   → 数秒の沈黙
   → 自分の1曲目を開始

   パターンC: 前のDJの最後の曲にミックスイン
   → BPMを合わせてスムーズに移行
   → 最も難しいが、最もプロフェッショナル

5. 引き継ぎ完了
   - 前のDJにお礼を言う
   - 自分のセット開始

会場のルールに従う
プロモーターに事前確認しておく
```

## 20. バックアップ計画と緊急時対応 - 完全版

### トラブルシナリオ別対処マニュアル

**シナリオ1: PCがフリーズした場合:**
```
発生確率: 中程度（特にWindows）
影響: 音が止まる（致命的）

即時対応（30秒以内）:
1. 慌てない（深呼吸1回）
2. 会場PAにマイクで「少々お待ちください」は不要
   → 黙って対処する方がプロフェッショナル
3. 予備USBメモリを会場のCDJに挿す（あれば）
4. スマートフォンのDJアプリでつなぐ（最終手段）

復旧手順:
1. Rekordboxを強制終了（Alt+F4 or Cmd+Q）
2. 30秒待つ
3. Rekordbox再起動
4. 最後にプレイしていた曲の次の曲から再開
5. セットリストは記憶 or 紙のバックアップで確認

予防策:
□ Rekordboxを最新版に更新
□ PC再起動してからイベントに向かう
□ 不要なアプリを全て閉じる
□ メモリを十分に確保（8GB以上推奨）
□ SSD搭載PCを使用（HDD非推奨）
```

**シナリオ2: DDJ-FLX4が認識されない場合:**
```
発生確率: 低（USBケーブルの問題が多い）
影響: コントローラーが使えない

即時対応:
1. USBケーブルを抜き差し
2. 別のUSBポートに接続
3. 予備USBケーブルに交換
4. Rekordboxを再起動

それでもダメな場合:
1. PCのキーボードモードでプレイ
   - Rekordboxのキーボードショートカット
   - Space: Play/Pause
   - 矢印キー: 曲送り/戻し
2. 会場のCDJを使用（USBメモリから）
3. スマートフォンでプレイ

予防策:
□ USBケーブルは2本持参
□ USBドライバーを最新に
□ 定期的にUSB端子を清掃
□ ケーブルを折り曲げない
```

**シナリオ3: 音が片方のスピーカーからしか出ない場合:**
```
発生確率: 中程度
影響: 音響バランスが崩れる

即時対応:
1. RCAケーブルの接続確認
2. 赤と白が正しい位置か確認
3. ケーブルを抜き差し
4. 予備RCAケーブルに交換

それでもダメな場合:
1. 会場PAスタッフに報告
2. ミキサー側の問題の可能性
3. スピーカーケーブルの確認を依頼

予防策:
□ RCAケーブルは2セット持参
□ ケーブルの断線チェック（前日）
□ 接続部分に緩みがないか確認
```

## 21. メンタルルーティンと本番中の心理管理

### パフォーマンス中の集中力維持テクニック

**集中力が切れるサインと対処法:**
```
サイン1: 次の曲が思い浮かばない
対処法:
→ セットリストを見る（紙 or Rekordbox）
→ Plan B/Cのプレイリストを開く
→ 今プレイ中の曲と同じBPM・キーで検索
→ 焦らず、現在の曲をLoop延長して時間を稼ぐ

サイン2: 観客の反応が気になりすぎる
対処法:
→ 一旦目を閉じて音楽に集中（5秒）
→ ヘッドフォンで次の曲に意識を向ける
→ 「コントロールできることに集中」と自分に言う
→ 観客は楽しんでいる（自信を持つ）

サイン3: ミスの後に動揺している
対処法:
→ 「ミスは過去。今に集中」と心で言う
→ 深呼吸1回（ボックス呼吸の簡易版）
→ 次のミックスに全集中
→ 観客はミスに気づいていないことがほとんど

サイン4: 体が疲れてきた
対処法:
→ 水を一口飲む
→ 姿勢を正す
→ 軽く体を動かす（リズムに乗る）
→ 残り時間を確認（ゴールが見える安心感）
```

### マイクロリカバリー（本番中の短い回復）

**30秒リカバリーテクニック:**
```
曲が流れている間の30秒で行う:

1. 呼吸リセット（10秒）
   - 鼻から4秒吸う
   - 口から6秒吐く
   - 1回だけで十分

2. 身体リセット（10秒）
   - 肩を3回回す
   - 手首を振る
   - 首を左右にゆっくり

3. メンタルリセット（10秒）
   - 「良い流れだ」と心で言う
   - 次にやることを1つだけ決める
   - 笑顔を作る（口角を上げるだけ）

この30秒で:
→ 疲労が軽減
→ 集中力が回復
→ ポジティブな気持ちに

60分セットなら4-5回実行
曲のブレイクやアウトロ中が最適
```

## まとめ

- **持ち物**: DDJ-FLX4、PC、ケーブル類、予備USB、ヘッドフォン、水
- **セットリスト**: 1週間前に作成、3日前に全曲確認、前日にPlan A/B/C準備
- **メンタル**: ポジティブな自己暗示、ビジュアライゼーション、成功イメージ
- **ルーティン**: 朝のルーティン確立、会場1時間前到着、セットアップ15分
- **食事**: 前日は早めに軽め、当日朝は軽め、本番3時間前に軽食、直前は水のみ
- **睡眠**: 7-8時間確保、23:00までに就寝、質の高い睡眠
- **服装**: ジャンルに合わせる、動きやすさ重視、黒が基本
- **スケジュール**: 余裕を持った時間配分、早めの到着、最終確認
- **緊急対応**: 予備機材、クラウドバックアップ、体調管理
- **会場リサーチ**: 1週間前から下調べ、プロモーターと連絡、天気と交通確認
- **セットリスト戦略**: ストーリーテリング、BPMマッピング、柔軟性確保
- **メンタルトレーニング**: 呼吸法、セルフトーク、プレッシャー管理
- **機材チェック**: 前日夜の最終確認、会場での15分セットアップ、予備体制
- **振り返り**: 必ず録音、当日夜に30分振り返り、翌日60分詳細分析
