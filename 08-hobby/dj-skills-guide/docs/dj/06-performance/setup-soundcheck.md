## 3. DDJ-FLX4セットアップ手順

### Step 1: 機材の配置

**ブースでの配置:**
```
1. DJブースに入る
   - 前のDJが終わるのを待つ
   - スペース確認

2. DDJ-FLX4を設置
   場所:
   - ブースの中央
   - 安定した場所
   - 手が届きやすい高さ

3. ノートPCを配置
   - DDJ-FLX4の後ろ
   - 画面が見える位置
   - 安定した場所

注意:
✗ 飲み物をコントローラーの近くに置かない
✗ 不安定な場所
✗ ケーブルが邪魔な位置
```

### Step 2: ケーブル接続

**接続順序（重要）:**
```
正しい順序:

1. ノートPC電源接続（バッテリー保護）
2. DDJ-FLX4とPCをUSBケーブル接続
   - DDJ-FLX4背面のUSB端子
   - PCのUSB端子
   - しっかり差し込む

3. ヘッドフォン接続
   - DDJ-FLX4前面のヘッドフォン端子
   - 6.3mm標準ジャックまたは3.5mm

4. DDJ-FLX4の電源を入れる
   - 背面の電源スイッチON
   - LEDが点灯

5. クラブのミキサーまたはスピーカーに接続
   （DDJ-FLX4の場合、クラブのシステムに接続する場合）
   - Master Out（背面）からミキサーへ
   - RCAケーブル（赤/白）

USB給電の場合:
- AC電源不要
- PCから給電
- ケーブル1本で完結
```

### Step 3: Rekordbox起動と設定

**Rekordbox 7.2.7:**
```
1. Rekordboxを起動
   - アプリケーション起動
   - 数秒待つ

2. DDJ-FLX4が認識されているか確認
   - 左上に「DDJ-FLX4」表示
   - もし表示されない:
     → Preferences > Audio
     → Audio Device: DDJ-FLX4選択

3. オーディオ設定確認
   Preferences > Audio:

   ✓ Audio Device: DDJ-FLX4
   ✓ Sample Rate: 44100 Hz（推奨）
   ✓ Buffer Size: 256 samples（推奨）

   Buffer Size:
   - 小さい（128）: レスポンス速い、CPU負担大
   - 大きい（512）: レスポンス遅い、CPU負担小
   - 256が最適

4. キーボードショートカット無効化（重要）
   Preferences > Controller:
   ✓ Keyboard Shortcuts: OFF

   理由: 誤操作防止
```

## 4. サウンドチェック

### サウンドチェックとは

```
サウンドチェック = 音量レベルの最適化

目的:
- Master音量の確認
- 各チャンネルのゲイン調整
- モニターバランス確認
- 音質チェック

時間: 5-10分
絶対に省略しない
```

### Step 1: Master音量確認

**基本手順:**
```
1. テスト曲を1曲ロード
   - Deck Aに
   - よく知っている曲
   - 音圧が標準的な曲

2. Deck Aを再生
   - Playボタン押す
   - 音が出ない（まだMasterが0）

3. Channel Fader（縦フェーダー）を上げる
   - Deck Aのフェーダーを上（MAX）

4. Trimノブ（ゲイン）を調整
   - まず12時（中央）
   - レベルメーター確認

5. レベルメーター確認
   DDJ-FLX4のレベルメーター:
   - 緑: 良い（-18dB ~ -6dB）
   - 黄色: 注意（-6dB ~ -3dB）
   - 赤: 大きすぎ（0dB以上）

   目標: 緑〜黄色の境界（-6dB付近）

6. Master Level（Master音量）を上げる
   - 徐々に上げる
   - クラブスタッフに確認
   「この音量で大丈夫ですか？」

7. 適切な音量に調整
   - 大きすぎず
   - 小さすぎず
   - スタッフのOKをもらう
```

### Step 2: 両デッキの音量を揃える

**ゲインマッチング:**
```
1. Deck Aでテスト曲再生中

2. Deck Bに別の曲をロード

3. Deck Bのヘッドフォンキューボタン押す
   - ヘッドフォンでDeck Bを聴く
   - まだスピーカーからは出ない

4. Deck BのTrim（ゲイン）を調整
   - レベルメーターを見る
   - Deck Aと同じレベルに
   - 緑〜黄色の境界

5. 両デッキを比較
   - Deck A再生
   - Deck Bに切り替え
   - 音量が同じか確認

6. 全ての曲で繰り返す
   （本番中も常にゲイン調整）

ゲインが揃っている = プロ
```

## 5. モニターバランス調整

### モニターとは

```
モニター = ヘッドフォンから聞こえる音

2つの音源:
1. Master（スピーカーから出ている音）
2. Cue（次にプレイする曲）

ミックス比を調整可能
```

### DDJ-FLX4でのモニターバランス

**Mixing（ミキシング）ノブ:**
```
DDJ-FLX4のヘッドフォンセクション:

Mixingノブ（中央上部）:
- 左（Master）: スピーカーの音のみ
- 中央: Master + Cue 50/50
- 右（Cue）: 次の曲のみ

おすすめ設定:

初心者:
→ 右（Cue）寄り
理由: 次の曲をしっかり聞く

中級者:
→ 中央
理由: 両方バランスよく

上級者:
→ 状況に応じて調整

自分に合った設定を見つける
```

### ヘッドフォン音量調整

**Headphone Levelノブ:**
```
DDJ-FLX4のヘッドフォン音量:

Level（音量）ノブ:
- 左: 小さい
- 右: 大きい

適切な音量:
✓ 曲のビートがはっきり聞こえる
✓ 会場の音も少し聞こえる
✓ 耳が痛くならない

大きすぎ:
✗ 耳を痛める
✗ 会場の音が聞こえない
✗ 観客の反応がわからない

小さすぎ:
✗ ビートマッチできない
✗ 音質確認できない

中間〜やや大きめが最適
```

## 6. クラブでの標準的なセットアップ

### CDJセットアップとの違い

**DDJ-FLX4の利点:**
```
CDJセットアップ（標準的なクラブ）:
- CDJ x 2台
- DJミキサー x 1台
- USBメモリ必須
- 複雑な接続

DDJ-FLX4セットアップ:
- DDJ-FLX4のみ
- ノートPC
- USBケーブル1本
- シンプル

セットアップ時間:
CDJ: 15-20分
DDJ-FLX4: 5-10分

DDJ-FLX4の方が簡単
```

### クラブのサウンドシステムへの接続

**2つの方法:**
```
方法1: DDJ-FLX4から直接出力（推奨）

接続:
DDJ-FLX4背面 Master Out（RCA）
→ クラブのミキサーまたはアンプ

利点:
✓ シンプル
✓ トラブル少ない
✓ 初心者向け

方法2: クラブのミキサー経由

接続:
DDJ-FLX4 → クラブのミキサー → スピーカー

利点:
✓ クラブのEQ使える
✓ 他DJと共有しやすい

初心者は方法1推奨
```

## 7. トラブル回避

### よくあるトラブルと対処

**トラブル1: 音が出ない**
```
原因:
- ケーブル接続ミス
- Master Level が0
- Channel Fader が0
- オーディオ設定ミス

対処:
1. 全てのケーブル確認
2. Master Level を上げる
3. Channel Fader を上げる
4. Rekordbox Audio設定確認
5. DDJ-FLX4再起動
```

**トラブル2: 音が歪む**
```
原因:
- Trim（ゲイン）が大きすぎ
- Master Level が大きすぎ
- クリッピング

対処:
1. レベルメーター確認（赤点灯？）
2. Trimを下げる
3. Master Level を下げる
4. EQを使いすぎていないか確認
```

**トラブル3: モニターが聞こえない**
```
原因:
- ヘッドフォン接続ミス
- Headphone Level が0
- Cueボタン押していない

対処:
1. ヘッドフォンしっかり差し込む
2. Headphone Level を上げる
3. Cueボタンが点灯しているか確認
```

**トラブル4: 片方のチャンネルから音が出ない**
```
原因:
- RCAケーブルの片方が抜けている
- ケーブル不良

対処:
1. 両方のRCAケーブル確認（赤/白）
2. 予備ケーブルに交換
3. 別の出力端子を使う
```

## 11. 会場別セットアップガイド

### 小規模バー・カフェ（収容30-50人）

**特徴:**
```
環境:
- 小さなDJスペース
- 壁際またはコーナー配置
- 限られた電源
- PA（音響）システムがシンプル

音響特性:
- 低音が籠もりやすい
- 残響が多い
- 近接音場（お客さんが近い）
```

**セットアップポイント:**
```
機材配置:
✓ コンパクトに収める
✓ 最小限のケーブル
✓ DDJ-FLX4 + ノートPCのみ
✗ 大きなスピーカー不要

接続:
DDJ-FLX4 Master Out
→ バーのミキサーまたは
→ 小型PAシステム（JBL EONなど）

音量設定:
- Master Level: 50-70%程度
- Trim: 控えめ（会話できる音量）
- 低音（Bass）: -2〜-4dB削る
  理由: 小空間では低音が膨らむ

推奨時間配分:
- セットアップ: 10分
- サウンドチェック: 5分
- 計15分
```

### 中規模クラブ（収容100-300人）

**特徴:**
```
環境:
- 専用DJブース
- プロ仕様ミキサー常設
- CDJ設置済み（多くの場合）
- 専用モニタースピーカー

音響特性:
- 低音が強い
- 音響設計されている
- サブウーファー完備
```

**セットアップポイント:**
```
機材配置:
✓ 既存のCDJ/ミキサーと併用可能
✓ DDJ-FLX4を横に配置
✓ 十分なスペース

接続パターン1（DDJ-FLX4のみ使用）:
DDJ-FLX4 Master Out
→ クラブのミキサー Channel 1/2
→ メインPA

接続パターン2（クラブミキサー経由）:
DDJ-FLX4を外部入力として
クラブミキサーのCH経由
→ クラブのEQ/エフェクト使用可能

音量設定:
- Master Level: 70-80%
- Trim: やや強め（クラブ標準）
- レベルメーター: 黄色まで許容
- サブウーファー考慮: 低音調整重要

ブースモニター:
✓ 専用ブースモニター使用
✓ 音量バランス重要
✓ ヘッドフォン + モニター併用

推奨時間配分:
- セットアップ: 15分
- サウンドチェック: 10分
- 計25分
```

### 大規模フェス・野外イベント（収容500人以上）

**特徴:**
```
環境:
- 大型ステージ
- プロ音響チーム常駐
- 複雑なPAシステム
- FOH（Front of House）ミキサー
- 多数のモニター

音響特性:
- 広い空間
- 音の遅延
- 野外の場合は音が拡散
- 強力な低音
```

**セットアップポイント:**
```
機材配置:
✓ ステージマネージャーと連携
✓ 専用DJブース/ステージエリア
✓ 複数のモニター配置
✓ バックアップシステム必須

接続:
DDJ-FLX4 Master Out
→ ステージボックス（XLR変換）
→ FOHミキサー
→ メインPAシステム

重要: RCA to XLR変換器必須
（大型イベントはXLR接続が標準）

音量設定:
- Master Level: FOHエンジニアに一任
- Trim: 標準レベル維持
- レベルメーター: 緑〜黄色キープ
- 絶対にクリッピングさせない

サウンドチェック:
1. FOHエンジニアと打ち合わせ
2. テスト曲を3-4曲用意
   - 静かなイントロ
   - 通常のトラック
   - ヘビーなベースライン
3. エンジニアが調整
4. ブースモニター確認
5. IEM（イヤーモニター）確認（使用する場合）

バックアップ体制:
✓ 予備DDJ-FLX4またはUSBメモリ
✓ 予備ノートPC
✓ 全曲USBメモリにも保存
✓ クラウドバックアップ（Wi-Fi利用可能な場合）

推奨時間配分:
- サウンドチェック専用時間: 30分-1時間
- リハーサル: 15-30分（場合による）
- 本番前最終確認: 10分
```

## 12. DJブースの配線パターンとトラブルシューティング

### 基本配線パターン

**パターン1: ホームセットアップ（最小構成）**
```
ノートPC
  |（USB）
DDJ-FLX4
  |（RCA）
パワードスピーカー（JBL 305P MkII等）

利点:
✓ シンプル
✓ トラブル少ない
✓ 初心者向け
```

**パターン2: クラブセットアップ（標準）**
```
ノートPC
  |（USB）
DDJ-FLX4
  |（RCA Master Out）
クラブミキサー（DJM-900NXS2等）
  |（XLR）
メインPAシステム
  |
スピーカー

ブースモニター ← 別系統
```

**パターン3: 外部機材統合セットアップ（上級）**
```
ノートPC
  |（USB）
DDJ-FLX4
  |（RCA）
外部ミキサー
  ↑
  |（追加入力）
  ├─ エフェクター（Pioneer RMX-1000等）
  ├─ ドラムマシン（Roland TR-8S等）
  └─ サンプラー（Akai MPC等）
  |（Master Out）
PAシステム
```

### 配線トラブルシューティング

**問題1: 音が全く出ない**
```
チェックリスト:
□ 電源が入っているか
  - DDJ-FLX4のLED点灯確認
  - スピーカー/ミキサーの電源確認

□ ケーブル接続確認
  - RCA（赤/白）両方とも差さっているか
  - 入力/出力を間違えていないか
  - ケーブルがしっかり奥まで差さっているか

□ ソフトウェア設定
  - Rekordbox Audio設定確認
  - DDJ-FLX4が選択されているか

□ 音量設定
  - Master Levelが0になっていないか
  - Channel Faderが上がっているか
  - Trimが最小になっていないか

対処:
1. 全てのケーブルを一度抜く
2. 順番に接続し直す
3. 各段階で音を確認
```

**問題2: 片チャンネルのみ音が出ない**
```
原因:
- RCAケーブルの片方（L/R）が抜けている
- ケーブル内部断線
- 接触不良

対処:
1. RCAの赤/白両方を確認
2. ケーブルを交換
3. 別の出力端子を試す
4. スピーカー側の入力も確認
```

**問題3: ノイズ・ハム音が入る**
```
原因:
- グラウンドループ
- ケーブルの品質
- 電源の問題
- 近くの電気機器からの干渉

対処:
1. グラウンドリフトアダプター使用
2. 高品質なケーブルに交換
3. 電源を別系統にする
4. ケーブルを電源コードから離す
5. DDJ-FLX4のグラウンド端子確認
```

**問題4: 音が途切れる・クラックルノイズ**
```
原因:
- Buffer Sizeが小さすぎ
- CPUパワー不足
- USBケーブル不良
- ドライバーの問題

対処:
1. Buffer Size上げる（256 → 512）
2. Rekordbox以外のアプリ終了
3. USBケーブル交換
4. USBポート変更（USB 2.0 → 3.0等）
5. ドライバー再インストール
```

**問題5: レイテンシー（遅延）が大きい**
```
原因:
- Buffer Sizeが大きすぎ
- USB接続の問題

対処:
1. Buffer Size下げる（512 → 256 → 128）
2. 直接USBポートに接続（ハブ経由NG）
3. 高速USBポート使用
4. Sample Rate確認（44100Hz推奨）
```

### ケーブル管理のベストプラクティス

**配線の基本ルール:**
```
1. 電源ケーブルとオーディオケーブルを分ける
   - ノイズ防止
   - 最低30cm離す

2. ケーブルの長さ
   - 必要最小限
   - 長すぎるケーブルは巻かない（ノイズ源）

3. ケーブルの固定
   - ガムテープで床に固定
   - ケーブルタイで束ねる
   - 足で引っ掛けない配置

4. ケーブルの識別
   - 色分けテープで識別
   - ラベリング（L/R、Input/Output）

5. 予備ケーブルの配置
   - すぐ取り出せる場所
   - ケースに整理
```

## 17. サウンドチェック詳細手順とチェックリスト

### プロフェッショナルなサウンドチェックの流れ

**5分間サウンドチェック（標準）:**
```
タイムライン:

0:00-1:00 準備
- テスト曲を3曲選定
  → 低音圧の曲（静かなイントロ）
  → 標準音圧の曲（通常のトラック）
  → 高音圧の曲（ヘビーなビート）
- Deck Aに最初のテスト曲ロード

1:00-2:00 Deck A調整
- 再生開始
- Channel Fader MAX
- Trim調整（緑〜黄色）
- Master Level徐々に上げる
- スタッフに音量確認

2:00-3:00 Deck B調整
- 2曲目をDeck Bにロード
- Cueボタンでモニター
- Trimを調整
- Deck Aと音量を揃える

3:00-4:00 EQ/周波数バランス確認
- 低音（Bass）: 籠もっていないか
- 中音（Mid）: ボーカルが明瞭か
- 高音（High）: シンバルが刺さらないか
- 会場の音響特性を把握

4:00-5:00 最終確認
- 3曲目（高音圧）でクリッピング確認
- モニターバランス最終調整
- ヘッドフォン音量確認
- 「準備完了」をスタッフに報告

効率的かつ完璧
```

### 音響特性別の調整

**小規模会場（バー・カフェ）:**
```
音響特性:
- 低音が籠もる
- 近接音場
- 残響多い

調整:
Bass EQ: -2〜-4dB削る
Mid EQ: +1〜+2dB上げる（明瞭に）
High EQ: 標準〜やや控えめ
Master Level: 50-70%

チェックポイント:
✓ 会話できる音量か
✓ 低音が壁に反射していないか
✓ お客さんの近くで音量確認
```

**中規模会場（クラブ）:**
```
音響特性:
- 低音が強い（サブウーファー）
- 音響設計済み
- ブースモニター完備

調整:
Bass EQ: 標準〜やや控えめ
Mid EQ: 標準
High EQ: 標準〜やや上げる（明瞭に）
Master Level: 70-85%

チェックポイント:
✓ ダンスフロアで音量確認
✓ サブウーファーの低音バランス
✓ ブースモニターとメインPAのバランス
```

**大規模会場（フェス・野外）:**
```
音響特性:
- 音が拡散
- 音の遅延
- FOHエンジニア管理

調整:
Bass EQ: FOHエンジニアに一任
Mid EQ: 標準
High EQ: 標準
Master Level: 80-90%（エンジニア指示）

チェックポイント:
✓ FOHエンジニアとの連携
✓ ブースモニター音量十分か
✓ IEM使用の場合は遮音性確認
```

### 周波数帯域別チェック

**低音域（Bass: 20Hz-250Hz）:**
```
確認事項:
- キックドラムがクリアに聞こえるか
- ベースラインが明瞭か
- 低音が籠もっていないか
- サブウーファーが暴れていないか

調整目安:
小規模: -2〜-4dB削る
中規模: 標準〜-2dB
大規模: エンジニアに一任

テスト曲推奨:
- House: 低音が強いトラック
- Techno: ヘビーなキック
```

**中音域（Mid: 250Hz-4kHz）:**
```
確認事項:
- ボーカルが明瞭か
- スネアが聞こえるか
- 楽器が埋もれていないか

調整目安:
小規模: +1〜+2dB上げる
中規模: 標準
大規模: 標準

テスト曲推奨:
- ボーカル入りトラック
- メロディが明確な曲
```

**高音域（High: 4kHz-20kHz）:**
```
確認事項:
- シンバルが刺さらないか
- ハイハットがクリアか
- 耳が痛くならないか

調整目安:
全会場: 標準〜やや控えめ

テスト曲推奨:
- シンバル多用のトラック
- 高音が目立つ曲
```

## 19. よくあるミスとその対策

### セットアップ時のよくあるミス

**ミス1: ケーブルの接続順序を間違える**
```
間違い:
- DDJ-FLX4の電源を先に入れる
- USBケーブルを後から接続
→ デバイス認識されない

正しい順序:
1. PC電源接続
2. USB接続
3. DDJ-FLX4電源ON
4. Rekordbox起動

ポイント:
USB接続してから電源ON
```

**ミス2: Buffer Sizeを適切に設定しない**
```
問題:
- 小さすぎる（64, 128）
  → 音が途切れる
- 大きすぎる（1024）
  → レイテンシーが大きい

推奨:
256 samples（標準）
512 samples（CPUパワー不足の場合）

確認方法:
Preferences > Audio > Buffer Size
```

**ミス3: ゲイン調整をしない**
```
問題:
- 全曲同じTrim設定
- 音量がバラバラ
→ プロっぽくない

正しい方法:
□ 毎曲ゲイン調整
□ レベルメーター確認
□ 緑〜黄色をキープ

習慣化する
```

**ミス4: モニターバランスを調整しない**
```
問題:
- Mixingノブを初期設定のまま
- ヘッドフォン音量が小さい/大きい
→ ミックスしにくい

正しい設定:
初心者: Cue寄り（次の曲重視）
中級者: 中央（バランス）
上級者: 状況に応じて

自分に合った設定を見つける
```

**ミス5: 予備機材を持ってこない**
```
問題:
- USBケーブルが1本だけ
- USBメモリなし
- 予備ヘッドフォンなし
→ トラブル時に対応できない

必須の予備:
✓ USBケーブル x2
✓ USBメモリ x2（同じ内容）
✓ 予備ヘッドフォン
✓ RCAケーブル（任意）

安心 = パフォーマンス向上
```

### サウンドチェック時のよくあるミス

**ミス6: テスト曲を適当に選ぶ**
```
問題:
- セットリストの曲でテスト
- 音圧がバラバラ
→ 正確なチェックできない

推奨テスト曲:
1. 標準音圧の曲（よく知っている曲）
2. 低音強めの曲（Bass確認）
3. 高音圧の曲（クリッピング確認）

毎回同じ曲でテスト = 基準ができる
```

**ミス7: 音量を上げすぎる**
```
問題:
- Master Level 100%
- Trim MAXで赤点灯
→ 音が歪む、クリッピング

正しい設定:
Master Level: 70-85%
Trim: 緑〜黄色の境界
レベルメーター: 赤は避ける

余裕を持たせる = 音質良い
```

**ミス8: 会場で音量確認しない**
```
問題:
- ブース内だけで確認
- ダンスフロアに行かない
→ 実際の音量がわからない

正しい方法:
1. ブースで基本調整
2. ダンスフロアに移動
3. 実際の音量・バランス確認
4. 必要なら再調整

観客の位置で聞く = 正確
```

## 20. セットアップの上級テクニック

### テクニック1: ゲインステージングの完璧化

**プロのゲイン管理:**
```
目標:
全ての音源が同じ音量
クリッピングゼロ
ダイナミックレンジ最大化

実践手順:

1. 基準曲を設定
   - 音圧が標準的な曲
   - よく知っている曲
   - これを基準に全て揃える

2. 基準曲のゲイン設定
   Trim: 12時〜1時（中央やや上）
   レベルメーター: 緑〜黄色の境界
   → これが「0dB」基準

3. 他の全曲を基準に揃える
   音圧低い曲: Trim上げる
   音圧高い曲: Trim下げる
   → 全てレベルメーター同じ位置

4. Rekordboxのゲイン分析活用
   Preferences > Analysis
   ✓ Gain Analysis ON
   → 自動でゲイン調整値表示

プロは全曲揃える
```

### テクニック2: 周波数別モニタリング

**EQを使った確認:**
```
方法:

1. 低音域チェック
   EQ設定:
   Bass: 標準
   Mid: -∞（完全カット）
   High: -∞（完全カット）
   → 低音だけ聞こえる
   確認: キック・ベースが明瞭か

2. 中音域チェック
   EQ設定:
   Bass: -∞
   Mid: 標準
   High: -∞
   → 中音だけ聞こえる
   確認: ボーカル・スネアが明瞭か

3. 高音域チェック
   EQ設定:
   Bass: -∞
   Mid: -∞
   High: 標準
   → 高音だけ聞こえる
   確認: シンバル・ハイハットが刺さらないか

4. 全帯域バランス確認
   EQ設定: 全て標準
   全体のバランスが良いか確認

周波数別 = 正確な調整
```

### テクニック3: ABテスト比較

**2曲を瞬時に比較:**
```
方法:

1. Deck Aにテスト曲1
2. Deck Bにテスト曲2
3. 同じ部分（サビ等）を頭出し
4. クロスフェーダーで瞬時切り替え
5. 音量・音質を比較

確認ポイント:
✓ 音量が揃っているか
✓ 低音バランスが同じか
✓ 全体の明瞭さが同じか

違いがあれば即調整

プロの耳を鍛える
```

### テクニック4: ルームアコースティック分析

**会場の音響特性を把握:**
```
分析手順:

1. 会場内を歩き回る
   - ダンスフロア中央
   - ダンスフロア端
   - 壁際
   - スピーカー近く
   → 各場所で音の聞こえ方確認

2. 低音の籠もりチェック
   - 壁際やコーナー
   → 低音が膨らむ場所
   - 中央
   → 低音が適切な場所

3. 音の遅延確認（大規模会場）
   - ステージから遠い場所
   → 遅延がある場合あり
   - FOHエンジニアに確認

4. デッドスポット確認
   - 音が小さい/聞こえにくい場所
   → スピーカー配置の問題
   - エンジニアに報告

会場を知る = 最適な調整
```

## まとめ

- **到着**: 60分前が理想、最低30分前
- **接続**: PC電源 → USB → ヘッドフォン → DDJ-FLX4電源ON
- **Rekordbox**: Audio設定確認（44100Hz, 256samples）
- **サウンドチェック**: Master音量確認、ゲイン調整、両デッキ揃える
- **モニター**: Mixingノブで調整、Cue寄り〜中央推奨
- **トラブル**: 音が出ない → 接続確認、レベルメーター確認
- **チェックリスト**: 機材設置 → Rekordbox設定 → サウンドチェック → 最終確認
- **プロのTips**: 前のDJ観察、余裕持った到着、ケーブル固定、予備機材
- **時間配分**: セットアップ15分、サウンドチェック10分、余裕5分
- **会場別**: 小規模バー（15分）、中規模クラブ（25分）、大規模フェス（1時間）
- **配線**: シンプルに、電源とオーディオケーブル分離、固定
- **モニター**: ヘッドフォン + ブースモニター併用、IEMは上級者向け
- **外部機材**: エフェクター・ドラムマシン・サンプラーで表現拡張
- **バックアップ**: USBメモリ2本以上、予備PC、クラウド同期
- **コミュニケーション**: エンジニアと良好な関係、礼儀・確認・報告
- **サウンドチェック詳細**: 5分間で完璧に、周波数帯域別確認
- **実践シミュレーション**: 初クラブ・野外フェスの具体的手順
- **よくあるミス**: 接続順序・ゲイン・モニター・予備機材
- **上級テクニック**: ゲインステージング完璧化、周波数別モニタリング、ABテスト、会場分析
