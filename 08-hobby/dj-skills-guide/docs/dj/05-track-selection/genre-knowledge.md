# ジャンル知識

主要DJ音楽ジャンルの歴史、特徴、代表的アーティストを完全網羅します。

## この章で学ぶこと

- House音楽（Deep、Tech、Progressive）
- Techno音楽（Minimal、Industrial、Acid）
- Hip Hop（Boom Bap、Trap）
- その他の主要ジャンル
- サブジャンルの違いと特徴
- 代表的アーティスト100名
- 必聴曲リスト
- ジャンルの選び方

## なぜジャンル知識が重要か

**選曲の基礎:**
```
ジャンルを知らない = 選曲できない

例:
Deep HouseとTech Houseの違いがわからない
→ セットがバラバラ
→ フロアが混乱

ジャンル知識 = DJの教養
```

## 1. House

### 起源と歴史

```
起源: 1980年代、シカゴ
創始者: Frankie Knuckles（"Godfather of House"）

特徴:
- BPM: 118-130
- 4つ打ちキック
- ディスコの影響
- ソウルフルなボーカル
- ピアノ、ストリングス

Warehouse Club（シカゴ）が名前の由来
```

### サブジャンル

#### Deep House

**特徴:**
```
BPM: 118-125（遅め）
雰囲気: 深い、メロウ、ソウルフル

サウンド:
- 深いベースライン
- ジャズの影響
- 温かいパッド
- ボーカル（時々）

用途: ウォームアップ、ラウンジ
```

**代表的アーティスト:**
```
- Kerri Chandler
- Larry Heard
- Moodymann
- Osunlade
- Black Coffee
```

**必聴曲:**
```
- Kerri Chandler - "Rain"
- Larry Heard - "Can You Feel It"
- Moodymann - "I Can't Kick This Feeling"
```

#### Tech House

**特徴:**
```
BPM: 125-130（速め）
雰囲気: エネルギッシュ、ダンサブル

サウンド:
- ミニマルなTechnoの影響
- グルーヴィーなベース
- パーカッシブ
- ボーカルサンプル

用途: ピークタイム
```

**代表的アーティスト:**
```
- Carl Cox
- Jamie Jones
- Green Velvet
- Fisher
- Chris Lake
```

**必聴曲:**
```
- Green Velvet - "La La Land"
- Fisher - "Losing It"
- Chris Lake - "Turn Off The Lights"
```

#### Progressive House

**特徴:**
```
BPM: 125-130
雰囲気: ビルドアップ、エピック

サウンド:
- 長いビルドアップ
- メロディアス
- レイヤー重ね
- エモーショナル

用途: フェスティバル、大型クラブ
```

**代表的アーティスト:**
```
- Sasha
- John Digweed
- Eric Prydz
- Deadmau5
- Anjunadeep
```

## 2. Techno

### 起源と歴史

```
起源: 1980年代、デトロイト
創始者: The Belleville Three
- Juan Atkins
- Derrick May
- Kevin Saunderson

特徴:
- BPM: 125-145
- 工業的
- 未来的
- ミニマル
- 反復的

ヨーロッパで爆発的人気（特にベルリン）
```

### サブジャンル

#### Minimal Techno

**特徴:**
```
BPM: 125-135
雰囲気: ミニマル、催眠的

サウンド:
- 最小限の要素
- 反復的なループ
- 微細な変化
- ディープなグルーヴ

用途: 長時間セット、アフターアワーズ
```

**代表的アーティスト:**
```
- Richie Hawtin
- Robert Hood
- Ricardo Villalobos
- Dubfire
```

#### Industrial Techno

**特徴:**
```
BPM: 130-145（速い）
雰囲気: ダーク、アグレッシブ

サウンド:
- ハードなキック
- ノイズ
- ディストーション
- 工業的な音

用途: ピークタイム、アンダーグラウンド
```

**代表的アーティスト:**
```
- Amelie Lens
- Charlotte de Witte
- I Hate Models
- Kobosil
```

#### Acid Techno

**特徴:**
```
BPM: 130-145
雰囲気: サイケデリック

サウンド:
- TB-303ベースライン
- スキーキーな音
- アシッド
- グルーヴィー

用途: 実験的なセット
```

**代表的アーティスト:**
```
- Josh Wink
- Phuture
- Hardfloor
```

**代表的アーティスト:**
```
- DJ Premier

- 9th Wonder

```

## 5. ジャンルの聴き分け

### House vs Techno

**違い:**
```
House:
- ソウルフル
- ボーカル多い
- メロディアス
- 温かい

Techno:
- 工業的
- ボーカル少ない
- ミニマル
- クール

両方: 4つ打ち、でも雰囲気が違う
```

### Deep House vs Tech House

**違い:**
```
Deep House:
- 遅い（118-125 BPM）
- ジャジー
- メロウ

Tech House:
- 速い（125-130 BPM）
- パーカッシブ
- エネルギッシュ

ミックス: スムーズに移行可能
```

## 6. ジャンル別のミックス戦略

### House DJ

**セット構成:**
```
60分セット:
0-15分: Deep House（ウォームアップ）
15-40分: Tech House（ピーク）
40-50分: Progressive House（メインピーク）
50-60分: Deep House（クールダウン）

BPM: 118 → 125 → 128 → 125
```

### Techno DJ

**セット構成:**
```
60分セット:
0-15分: Minimal Techno（125 BPM）
15-40分: Techno（130 BPM）
40-50分: Industrial Techno（138 BPM、ピーク）
50-60分: Techno（130 BPM）

長時間セット推奨（2-3時間）
```

### Hip Hop DJ

**セット構成:**
```
60分セット:
- Boom Bap（クラシック）
- 現代Hip Hop
- Trap
- スクラッチで装飾
- ジャグリング

BPM: 85-100（変化が少ない）
```

## 7. ジャンル知識の学習方法

### Week 1-2: House完全理解

```
毎日2時間:
- House プレイリスト50曲作成
- 毎日聴き込む
- Deep、Tech、Progressiveの違いを体感
- アーティスト20名を覚える

目標: House を聴いてサブジャンルが判別できる
```

### Week 3-4: Techno完全理解

```
毎日2時間:
- Techno プレイリスト50曲作成
- Minimal、Industrial、Acidを聴き分ける
- Berghain（ベルリン）のセットを研究
- アーティスト20名を覚える

目標: Techno の雰囲気を理解
```

### Week 5-6: その他ジャンル

```
Hip Hop、Drum & Bass、Dubstep:
各ジャンル20曲
アーティスト10名ずつ

目標: 主要ジャンルを網羅
```

## 9. ジャンルの系譜と歴史的展開

### 電子音楽の黎明期（1950年代〜1970年代）

電子音楽のルーツを理解することは、現代のDJジャンルを深く把握するための基盤となる。

```
■ 電子音楽タイムライン（前史）

1950年代: ミュージック・コンクレート / 電子音楽スタジオの誕生
├── Pierre Schaeffer（パリ）: テープ操作による音楽
├── Karlheinz Stockhausen（ケルン）: 電子音の作曲
└── Columbia-Princeton Electronic Music Center

1960年代: シンセサイザーの誕生
├── Robert Moog → Moog Synthesizer（1964）
├── Don Buchla → Buchla Series（西海岸）
├── Wendy Carlos - "Switched-On Bach"（1968）
└── Silver Apples、White Noise → 初期エレクトロニカ

1970年代: クラフトワークとディスコ
├── Kraftwerk - "Autobahn"（1974）
│   └── 電子音楽をポップに昇華した最重要バンド
├── Giorgio Moroder - "I Feel Love"（Donna Summer, 1977）
│   └── シンセサイザー主体のディスコ → House の直接的祖先
├── Yellow Magic Orchestra（1978）
│   └── テクノポップの先駆者、後のTechnoに影響
└── Tangerine Dream、Jean-Michel Jarre
    └── アンビエント/シーケンサー音楽
```

### ディスコからHouseへの系譜（1977年〜1986年）

```
■ House誕生の流れ

1977-1979: ディスコ全盛期
├── Studio 54（ニューヨーク）
├── Paradise Garage（Larry Levan）
│   └── "Garage House" の語源
├── The Loft（David Mancuso）
│   └── サウンドシステム文化の確立
└── Disco Demolition Night（1979）
    └── ディスコへのバックラッシュ

1980-1983: ポストディスコとエレクトロ
├── ディスコが地下に潜る
├── エレクトロファンク（Afrika Bambaataa - "Planet Rock", 1982）
├── Italo Disco（Giorgio Moroder の影響、ヨーロッパ）
├── Hi-NRG（ゲイクラブシーン）
└── シカゴのクラブシーンが独自進化

1984-1986: House Music の誕生
├── Frankie Knuckles @ The Warehouse（シカゴ）
│   ├── ディスコレコードのリエディット
│   ├── ドラムマシン（Roland TR-808/909）の導入
│   └── "House Music" という名称の確立
├── Jesse Saunders - "On and On"（1984）
│   └── 最初のHouseレコードとされる
├── Chip E. - "Time to Jack"（1985）
├── Farley "Jackmaster" Funk - "Love Can't Turn Around"（1986）
└── Marshall Jefferson - "Move Your Body"（1986）
    └── "The House Music Anthem" と呼ばれる
```

### デトロイト・テクノの誕生（1981年〜1988年）

```
■ Techno誕生の流れ

1981-1984: ベルヴィル・スリーの出会い
├── Juan Atkins、Derrick May、Kevin Saunderson
│   └── ミシガン州ベルヴィルの高校同級生
├── 影響源:
│   ├── Kraftwerk（ヨーロッパの電子音楽）
│   ├── Parliament/Funkadelic（ファンク）
│   ├── Prince（ミネアポリスサウンド）
│   └── "The Electrifying Mojo"（ラジオDJ）
└── Cybotron（Juan Atkins + Rick Davis）
    └── "Clear"（1983）= プロトテクノ

1985-1988: テクノの確立
├── Juan Atkins（Model 500）- "No UFOs"（1985）
│   └── 最初の「テクノ」トラックの一つ
├── Derrick May - "Strings of Life"（1987）
│   └── テクノの聖典、ピアノリフの革命
├── Kevin Saunderson（Inner City）- "Big Fun"（1988）
│   └── テクノを商業的成功に導く
├── "Techno! The New Dance Sound of Detroit"（1988）
│   └── コンピレーション、ジャンル名を世界に広める
└── ヨーロッパへの波及
    ├── イギリスのレイヴカルチャー
    ├── ベルリンの壁崩壊（1989）→ テクノの聖地化
    └── Tresor、Berghain の前身
```

### UK レイヴカルチャーとジャンルの分岐（1988年〜1995年）

```
■ Second Summer of Love（1988）以降の展開

1988-1990: UKレイヴ爆発
├── Acid House ブーム（TB-303 サウンド）
├── The Hacienda（マンチェスター）
├── 巨大屋外レイヴパーティー
├── Criminal Justice Act（1994）→ レイヴ規制法
└── クラブカルチャーの制度化

ジャンル分岐図:
                    ┌─ Jungle（1991〜）→ Drum & Bass（1995〜）
                    │
UK Rave ───────────┼─ Breakbeat Hardcore → Happy Hardcore
                    │
                    ├─ Progressive House（1992〜）
                    │
                    ├─ Trance（1991〜、ドイツ起源）
                    │
                    └─ UK Garage（1994〜）→ 2-Step → Dubstep（2001〜）
```

### (4) Acid House

```
■ Acid House 完全ガイド

起源: シカゴ、1987年
BPM: 120-130
核心: Roland TB-303 ベースシンセサイザーの偶発的使用

歴史的背景:
- DJ Pierre（Phuture）が TB-303 を「間違った」使い方で演奏
- 偶然生まれた「スキーキー」なサウンドがジャンルを定義
- "Acid Tracks"（Phuture, 1987）が起源曲

サウンド特性:
┌──────────────────────────────────────────────┐
│ ・TB-303 のレゾナンスを上げたスクウェルチ音   │
│ ・フィルタースウィープ（低→高→低の周期変化）│
│ ・催眠的な反復パターン                        │
│ ・シンプルなドラムマシンビート（TR-808/909）  │
│ ・ミニマルな構成（ベースライン中心）          │
└──────────────────────────────────────────────┘

代表的アーティスト:
- Phuture（DJ Pierre, Spanky, Herb J）
- Adonis - "No Way Back"
- Bam Bam - "Where's Your Child"
- Josh Wink - "Higher State of Consciousness"

DJ での活用:
- テクノセットへのスパイスとして1-2曲投入
- ビルドアップ中のフィルター操作と相性が良い
- アシッドラインが催眠効果を生み、ロングミックスに最適
```

### (6) Garage House / US Garage

```
■ Garage House 完全ガイド

起源: ニューヨーク、1986-1992年
BPM: 118-128
核心: Paradise Garageの精神を継承、ソウルフルでボーカル重視

歴史的背景:
- Larry Levan @ Paradise Garage（1977-1987）が精神的支柱
- NYのゲイコミュニティが育てたサウンド
- ディスコの直接的後継者

特徴:
┌──────────────────────────────────────────────┐
│ ・パワフルなボーカルパフォーマンス            │
│ ・ゴスペル / R&B の影響                       │
│ ・豊かなコード進行                            │
│ ・オーケストラヒット                          │
│ ・ピアノスタブ                                │
│ ・感情的で高揚感のある展開                    │
└──────────────────────────────────────────────┘

代表的アーティスト:
- Larry Levan
- Tony Humphries
- Masters At Work（Louie Vega & Kenny "Dope" Gonzalez）
- Kerri Chandler
- Blaze
- Todd Edwards（Future Garageの先駆者）

必聴曲:
- Masters At Work - "To Be In Love"
- Blaze - "Lovelee Dae"
- Todd Edwards - "Saved My Life"
- Kerri Chandler - "Bar A Thym"
```

### (7) Soulful House

```
■ Soulful House 完全ガイド

起源: ニューヨーク / ニュージャージー、1990年代
BPM: 118-125
核心: ソウル/ゴスペル/R&B とHouseの融合

Garage House との違い:
- Garageはクラブ指向、Soulfulはよりリスニング寄り
- よりポリッシュなプロダクション
- 生楽器の使用が多い

特徴:
┌──────────────────────────────────────────────┐
│ ・フルボーカルトラック（歌もの中心）          │
│ ・生ピアノ、ギター、ストリングス              │
│ ・ゴスペルクワイア                            │
```

```
   代表: Skrillex（後期）、Four Tet、Floating Points
   レーベル: Hessle Audio、Hemlock

   起源: 2019年〜リバイバル
   BPM: 125-140
   融合元: Techno + Breakbeat + Electro
   特徴: ブレイクビートをテクノの文脈で使用
   代表: VTSS、Sherelle、Special Request
   レーベル: Hooversound

4. Afro Tech
   起源: 2018年〜
   BPM: 124-132
   融合元: Afro House + Tech House + Tribal
   特徴: アフリカンパーカッション + テクノ的ミニマリズム
   代表: Kevin de Vries、KEINEMUSIK

   代表: Nicola Cruz、Bedouin、Acid Pauli
   レーベル: Sol Selectas、Lump Records

6. Hardwave / Wave
   起源: 2017年〜
   BPM: 140-160
   融合元: Trap + Trance + Dubstep
   特徴: エセリアルなシンセ、ダーク、ドリーミー
```
