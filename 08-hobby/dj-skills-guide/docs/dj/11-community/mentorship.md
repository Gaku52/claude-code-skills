# メンターの見つけ方

一人で悩む時間を半分に。経験豊富なDJから学び、成長を加速させる完全ガイドです。

## この章で学ぶこと

- メンターとは何か
- 良いメンターの条件
- メンターの探し方
- アプローチ方法
- メンター関係の維持
- オンラインメンタリング

---

## メンターとは

**成長を導いてくれる存在:**

```
メンターの定義:

経験豊富なDJ:
あなたより先を行っている

導いてくれる:
技術、キャリア、マインド

無償が基本:
お金ではなくGive & Take

長期的関係:
一度きりではなく継続的

メンターがいると:

成長速度:
2-3倍速い

失敗回避:
先人の経験から学ぶ

モチベーション:
ロールモデルがいる

人脈:
メンターの人脈にアクセス

キャリア:
的確なアドバイス

統計:
成功したDJの80%以上が
メンターを持っていた
```

## 良いメンターの条件

### 必須条件

**この3つは絶対:**

```
1. 経験がある

あなたより:
最低2-3年先を行っている

実績:
定期的にギグをしている
レジデンシーがある
リリースがある（制作もする場合）

ただし:
トップDJである必要はない

理想的な距離:
あなた: 初心者
メンター: 中級〜上級

近すぎず、遠すぎず

2. 教えることが好き

特徴:
質問に丁寧に答える
後進育成に興味がある
時間を割いてくれる

サイン:
ワークショップ開催
SNSで tips 投稿
コミュニティ活動

3. 人格的に尊敬できる

重要:
技術だけでなく人間性

良いメンター:
謙虚
誠実
ポジティブ
時間を守る

悪いメンター:
高圧的
見下す
約束を破る
```

### プラスアルファ

**あるとさらに良い:**

```
同じ機材:
DDJ-FLX4 使用
→ 具体的に教えられる

同じジャンル:
Techno ↔ Techno
→ 選曲・技術が共通

同じエリア:
同じ街
→ 会いやすい

似た環境:
本業ありのDJ
→ 両立のアドバイス

同じ目標:
クラブレジデンシー目指す
→ 道筋が共通

ただし:
全て揃う必要なし
1-2個あればOK
```

## メンター関係の維持

### 定期セッション

**継続的な学び:**

```
頻度:

推奨:
月1回、30-60分

柔軟に:
メンターの都合に合わせる

形式:

対面:
カフェ、スタジオ

オンライン:
Zoom、Discord

テーマ:

事前に決める:
「今回は選曲について」
「トランジション技術」

準備:
質問リストを用意

セッションの流れ:

1. 近況報告（5分）
   「先月は〇〇を練習しました」

2. フィードバック（10分）
   録音したミックスを聴いてもらう

3. 質問（20分）
   準備した質問

4. アドバイス（20分）
   メンターからの提案

5. 次回までの課題（5分）
   「次回までに〇〇を練習」

記録:

ノートを取る:
アドバイスを記録

実践:
教わったことを試す

報告:
次回「やってみました」
```

```

**A:** コミュニティ活動を増やす

```
原因:

コミュニティ参加不足:
まず仲間を増やす

アプローチ不足:
10人に聞いて1人

焦りすぎ:
信頼構築に時間がかかる

対策:

イベント参加頻度UP:
月1回 → 週1回

オンライン活動:
Discord等で活発に

複数候補:
1人ではなく3-5人

ゆっくり:
自然な関係から

代替:

ピアメンタリング:
同レベルのDJ同士

オンラインリソース:
YouTube、ブログ

有償レッスン:
短期集中で学ぶ
```

メンターの仕事ではない

考えさせる:
メンターの仕事

## メンターとの効果的なコミュニケーション

### 質問の準備

**事前準備が成功の鍵:**

```
質問リスト作成:

カテゴリ分け:
技術的質問

マインドセット

具体性:

悪い質問:
「上達するには？」

「ビートマッチングは安定してきましたが、

背景情報:

現状:
「現在はこういう状態」

試したこと:
「〇〇を試しました」

結果:
「△△になりました」

質問テンプレート:

1. 状況説明
2. 試したこと
3. 問題点

時間配分:

## メンタリング上級テクニック

### 効果的な質問術

**深い学びを引き出す:**

```

「どう思いますか？」
→ 深い回答

「このトランジションで

フロアの視点で考えて。」

5W1H活用:

What:
「何が問題？」

Why:
「なぜそう思う？」

How:
「どうすれば改善？」

When:
「いつ練習する？」

Where:
「どこで使う？」

Who:
「誰に聞いた？」

ソクラテス式:

自分で考えさせる:
答えを与えない

段階的に:
小さな質問を重ねる

例:

メンティー: 「選曲が難しい」
メンター: 「何が難しい？」
メンティー: 「次の曲が思いつかない」

フィードフォワード:

過去ではなく未来:
「次はどうする？」

「明日から何ができる？」

ポジティブ:
「どう成長したい？」
```
