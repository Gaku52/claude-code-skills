## ピッチコントロールとは

### 基本的な定義

ピッチコントロールとは、再生する楽曲のBPM（Beats Per Minute）と音程（ピッチ/キー）を調整する技術です。アナログレコードの時代から存在するDJの基本技術であり、デジタル時代になってさらに高度な制御が可能になりました。

```
ピッチコントロール = BPMと音程（キー）の調整

物理的な原理:
- レコード/CDの回転速度を変える = BPMと音程が同時に変化
- デジタル処理で分離 = BPMと音程を独立に制御可能

Tempo Fader:
- BPM を変更する物理的なスライダー
- 通常モード: キーも連動して変わる
- Master Tempo: キーを固定したままBPMだけ変更

Master Tempo（Key Lock）:
- BPMを変更しても元のキーを維持する機能
- DSP（デジタル信号処理）によるリアルタイム変換
- 音質への影響はアルゴリズムの品質に依存
```

### 歴史的背景

ピッチコントロールの歴史を理解することで、現在のテクノロジーの意義がより深く分かります。

**アナログ時代（1970年代〜）**

Technics SL-1200シリーズが1972年に登場し、DJプレイに革命をもたらしました。このターンテーブルには±8%のピッチコントロールが搭載されており、2枚のレコードのBPMを合わせるビートマッチングが可能になりました。当時はMaster Tempo機能は存在せず、BPMを変えれば必然的にキーも変化しました。

```
Technics SL-1200の進化:
- SL-1200 (1972): ±8% ピッチ範囲
- SL-1200MK2 (1979): DJスタンダードモデル
- SL-1200MK3 (1989): ピッチリセットボタン追加
- SL-1200MK5 (2002): ±8%/±16% 切替
- SL-1200MK7 (2019): ±8%/±16%/±50%

ピッチ精度:
- アナログ時代: 約0.1%刻み（感覚による）
- デジタル時代: 0.01%〜0.001%刻み（数値表示）
```

**CDJ時代（1990年代〜）**

Pioneer CDJ-500（1994年）の登場により、CDでのDJプレイが本格化しました。デジタル技術の進歩により、Master Tempo機能が実装され、BPMを変えてもキーを維持できるようになりました。

```
CDJ時代の革新:
- CDJ-500 (1994): CD DJプレイの幕開け
- CDJ-1000 (2001): Master Tempo搭載、業界標準に
- CDJ-2000 (2009): USB対応、高精度ピッチ
- CDJ-3000 (2020): 高品質Master Tempo搭載

Master Tempo品質の進化:
- 初期: 音質劣化が顕著（金属的な響き）
- 中期: 改善されるが、大きなBPM変更では劣化
- 現在: ±6%程度なら聴感上ほぼ無劣化
```

**デジタル/コントローラー時代（2000年代〜）**

PCDJソフトウェア（Traktor, Serato, rekordbox）の登場により、ピッチコントロールは完全にデジタル処理となり、さらに高度な機能が実現しました。キーの自動検出、ハーモニックミキシングの提案、リアルタイムキーシフトなどが可能になっています。

```
デジタル時代の機能拡張:
- 自動BPM検出: ±0.01 BPM精度
- 自動キー検出: Camelot/Open Key表記
- Key Shift: 半音単位でキーを上下
- BPM Sync: 自動ビートマッチング
- Key Sync: 自動ハーモニックマッチング
- Elastic Beatgrid: 変動BPMへの対応
```

### ピッチコントロールの物理的原理

アナログとデジタルのピッチコントロールの仕組みを理解することは、音質への影響を正しく把握する上で重要です。

**アナログ方式（回転速度変更）**

```
レコードの場合:
- 33 1/3 rpm → 回転速度そのものを変更
- 速度 +8% → 周波数（音程）も +8% 上昇
- 速度 -8% → 周波数（音程）も -8% 低下

周波数と半音の関係:
- 1半音 = 約5.946%の周波数変化
- +8% ≈ 約1.3半音の上昇
- -8% ≈ 約1.4半音の低下

具体例:
- 440Hz（A4）の音 +8% → 475.2Hz（A4とA#4の間）
- 440Hz（A4）の音 -8% → 404.8Hz（G#4とA4の間）
```

**デジタル方式（タイムストレッチ/ピッチシフト）**

```
Master Tempo の仕組み:
1. 音声信号をフレーム単位で分割
2. 各フレームの長さを変更（タイムストレッチ）
3. ピッチ（周波数）は元の値を維持
4. フレーム間を補間して滑らかに接続

主要なアルゴリズム:
- WSOLA (Waveform Similarity Overlap-Add)
  → 波形の類似部分を重ね合わせ
  → 比較的低負荷、良好な音質

- Phase Vocoder
  → FFTベースの周波数領域処理
  → 高品質だが計算負荷大

- Elastique（zplane社）
  → rekordbox / Traktor で使用
  → 業界最高品質のタイムストレッチ

- Rubberband
  → オープンソース
  → 多くのDJソフトで採用
```

## 1. Tempo Fader

### Tempo Faderの構造と設計

Tempo Fader（ピッチフェーダー/ピッチスライダーとも呼ばれる）は、DJコントローラーやCDJに搭載された縦方向のスライダーです。物理的な設計と操作感は、精密なBPM調整を可能にする上で非常に重要です。

```
一般的なTempo Faderの仕様:
- 方向: 縦方向（上下スライド）
- 長さ: 約60mm〜100mm（機種による）
- 抵抗: 適度な摩擦（意図しない移動を防止）
- 中央位置: クリック感あり（±0%を触感で確認）
- 分解能: デジタルでは1024〜4096段階

各機種のFader長:
- DDJ-FLX4: 約60mm（エントリーモデル）
- DDJ-1000: 約100mm（ミドルクラス）
- CDJ-3000: 約100mm（プロフェッショナル）
- SL-1200MK7: 約60mm（ターンテーブル）
```

**Fader長と精度の関係**

フェーダーが長いほど、同じBPM変更量に対してフェーダーの物理的な移動距離が大きくなるため、より精密な調整が可能です。

```
精度の比較（±8%範囲の場合）:
60mm Fader:
- 全範囲16% = 60mm
- 1% = 3.75mm
- 0.1% = 0.375mm（非常に繊細な操作が必要）

100mm Fader:
- 全範囲16% = 100mm
- 1% = 6.25mm
- 0.1% = 0.625mm（やや余裕がある）

プロの感覚:
- 0.1%の違い = 聴き分け可能（訓練されたDJ）
- 0.05%の違い = フロアでは気づかないレベル
- 0.5%の違い = 数秒でズレが明確に
```

### DDJ-FLX4のTempo Fader

DDJ-FLX4は入門者向けのコントローラーですが、しっかりしたTempo Faderを搭載しています。

```
DDJ-FLX4 Tempo Fader仕様:
位置: 各デッキの中央部分
方向:
  上方向: マイナス（遅く / BPM低下）
  中央: ±0%（原曲BPM / デフォルト）
  下方向: プラス（速く / BPM上昇）

注意: 一部のDJ機器では上下の方向が逆のものもある
      必ず自分の機材の方向を確認すること

範囲設定: ±6% / ±10% / ±16% / WIDE
デフォルト: ±8%相当

範囲切替方法（rekordbox）:
1. rekordboxの設定画面を開く
2. コントローラータブを選択
3. Tempo Slider Range を変更
4. または SHIFT + TEMPO FADER で切替

例（±8%範囲の場合）:
128 BPM × 1.08 = 138.24 BPM（最大値）
128 BPM × 1.00 = 128.00 BPM（中央値）
128 BPM × 0.92 = 117.76 BPM（最小値）

範囲の使い分け:
±6%  : 精密なビートマッチングに最適
±10% : 一般的なDJプレイ（推奨）
±16% : ジャンル混合プレイ
WIDE  : エクストリームなBPM変更
```

### 基本操作テクニック

ビートマッチングでTempo Faderを使う際の基本的な手順と、精密な操作のコツを解説します。

```
ビートマッチング時の基本手順:

Step 1: 目標BPMの確認
- 再生中のデッキ（デッキA）のBPMを確認
- 次にミックスする曲（デッキB）のBPMを確認
- 差分を計算（例: 128 BPM → 126 BPM = -2 BPM）

Step 2: 大まかなTempo Fader調整
- BPM表示を見ながらFaderを大まかに移動
- 目標BPMの±0.5 BPM以内に近づける
- この時点では視覚（BPM表示）に頼ってOK

Step 3: ヘッドフォンでの聴覚的微調整
- CUEボタンでデッキBをヘッドフォンモニター
- デッキAとBを同時に聴く（Split Cue）
- ビートのズレ方向を判断:
  → デッキBが速い = Faderを上（マイナス方向）に微調整
  → デッキBが遅い = Faderを下（プラス方向）に微調整

Step 4: 精密調整（0.1%単位）
- 指先の微細な動きでFaderを操作
- ビートが完全に重なるポイントを探す
- 数小節間ズレないことを確認

Step 5: ロック
- BPMが合った状態でFaderに触らない
- ビートが安定していることを数秒確認
- トランジションの準備に移行
```

**Tempo Faderの持ち方と操作のコツ**

```
持ち方のバリエーション:

1. ピンチグリップ（推奨）
   - 親指と人差し指でFaderノブをつまむ
   - 最も精密な操作が可能
   - プロDJの大半がこのスタイル

2. スライドグリップ
   - 人差し指の腹でFaderノブを押す
   - 大まかな調整に向く
   - 速い操作が可能

3. フィンガーティップ
   - 指先でFaderノブに軽く触れる
   - 超微細な調整に最適
   - 0.05%以下の調整に使う

操作のコツ:
- 急激な動きは避ける（聴衆にBPM変化がバレる）
- 微調整は「押す」のではなく「触る」感覚
- Faderノブが汗で滑る場合は指を拭く
- 中央位置のクリック感を基準点として覚える
- ライブ中は視覚（BPM表示）に頼りすぎない
```

### Tempo Faderの範囲設定と戦略

ピッチ範囲の設定は、プレイスタイルやジャンルによって最適な値が異なります。

```
範囲設定の選び方:

±6% が最適な場面:
- House / Techno のような同ジャンル内でのプレイ
- BPM差が小さい曲同士のミックス（±5 BPM以内）
- 精密なビートマッチングを重視する場合
- 長いトランジションを行う場合
→ メリット: 高精度、意図しない大きなBPM変更を防止
→ デメリット: BPM差が大きい曲に対応できない

±10% が最適な場面:
- 一般的なクラブDJプレイ
- 多ジャンルの曲をミックスする場合
- EDM / Pop / Hip-Hop の混合セット
→ メリット: 汎用性が高い、ほとんどの場面で対応可能
→ デメリット: 精度はやや低下

±16% が最適な場面:
- テクニカルDJ（スクラッチ、ターンテーブリズム）
- エクストリームなBPMトランジション
- ジャンルの大幅な切り替え
→ メリット: 非常に幅広いBPM範囲をカバー
→ デメリット: 精密調整が難しい

WIDE（±100%）が最適な場面:
- 実験的な音楽制作
- サウンドデザイン
- 通常のDJプレイでは使用しない
→ メリット: 完全な自由度
→ デメリット: DJプレイでの精度は極めて低い
```

### BPM範囲のリファレンス（ジャンル別）

各ジャンルの一般的なBPM範囲を把握しておくことで、Tempo Faderの範囲設定を適切に選べます。

```
ジャンル別BPM範囲:

Hip-Hop / R&B:      70-100 BPM
Reggaeton:          90-100 BPM
Deep House:         118-125 BPM
House:              120-130 BPM
Tech House:         124-130 BPM
Progressive House:  126-132 BPM
Techno:             125-145 BPM
Trance:             130-150 BPM
Drum & Bass:        160-180 BPM
Dubstep:            138-142 BPM（ハーフタイム: 69-71 BPM）
Trap:               130-170 BPM（ハーフタイム: 65-85 BPM）
EDM / Big Room:     126-132 BPM
Future Bass:        130-170 BPM
Hardstyle:          150-160 BPM
Hardcore:           160-200+ BPM
Ambient / Downtempo: 60-100 BPM

ジャンル間の互換性:
- House → Tech House: ±6%で十分
- House → Techno: ±10%推奨
- Hip-Hop → House: ±16%またはダブルタイム変換
- D&B → House: BPMを半分として扱う（170 BPM ≈ 85 BPM）
```

### Tempo Faderのキャリブレーション

長期間使用するとTempo Faderの精度が低下する場合があります。定期的なキャリブレーションが重要です。

```
キャリブレーション手順（DDJ-FLX4）:
1. rekordboxの環境設定を開く
2. コントローラー → DDJ-FLX4を選択
3. キャリブレーションオプションを確認
4. 画面の指示に従ってFaderを上下に動かす
5. 中央位置（±0%）が正しく認識されることを確認

確認方法:
- Faderを中央に合わせる
- BPM表示が原曲BPMと一致するか確認
- ±0.1 BPM以内なら正常
- それ以上ズレている場合はキャリブレーション実施

注意事項:
- ホコリや液体がFaderに入ると精度低下
- 定期的にエアダスターで清掃
- 接点復活剤は機種によって使用可否が異なる
- 修理が必要な場合はメーカーに相談
```

## 2. Master Tempo（Key Lock）

### 機能の詳細

Master Tempo（別名: Key Lock、キーロック）は、BPMを変更してもオリジナルのキー（音程）を維持する機能です。デジタル信号処理（DSP）によってリアルタイムに音声を変換します。

```
Master Tempo ON:
BPM変更 → キー変わらない（DSP処理で維持）
例: 128 BPM, Key Am → 135 BPM, Key Am（キー維持）

Master Tempo OFF:
BPM変更 → キーも変わる（物理的に自然な変化）
例: 128 BPM, Key Am → 135 BPM, Key A#m〜Bm程度に上昇

各ソフトウェアでの名称:
- rekordbox: Master Tempo
- Serato DJ: Key Lock
- Traktor: Key Lock
- Virtual DJ: Master Tempo
- djay: Key Lock
→ 機能は同一、名称が異なるだけ
```

### DSP処理の仕組みと音質

Master Tempoの内部処理を理解することで、音質への影響を正しく判断できます。

```
タイムストレッチの処理フロー:

1. 分析フェーズ
   - 入力音声を短いフレーム（10-50ms）に分割
   - 各フレームのピッチ周期を検出
   - トランジェント（アタック部分）を特定

2. 処理フェーズ
   BPMを上げる場合（例: +5%）:
   - フレームの持続時間を短縮（5%削除）
   - ピッチは元の周波数を維持
   - フレーム間の接続をクロスフェードで滑らかに

   BPMを下げる場合（例: -5%）:
   - フレームの持続時間を延長（5%追加）
   - 追加部分は既存の波形を繰り返し利用
   - ピッチは元の周波数を維持

3. 出力フェーズ
   - 処理済みフレームを再結合
   - 不自然なアーティファクトを低減
   - リアルタイムで出力

音質劣化のパターン:
- 小さなBPM変更（±3%以内）: ほぼ無劣化
- 中程度のBPM変更（±3-6%）: わずかな劣化（プロが注意深く聴けば分かる）
- 大きなBPM変更（±6-10%）: 明確な劣化（フランジング、金属感）
- 極端なBPM変更（±10%以上）: 著しい劣化（使用非推奨）
```

**音質劣化の具体的な症状**

```
Master Tempo使用時に発生しうる音質劣化:

1. フランジング（flanging）
   - 原因: フレーム接続部分の位相ずれ
   - 症状: 音が「揺れる」ような効果
   - 発生条件: BPM変更量が大きい場合

2. メタリックサウンド
   - 原因: 高周波数帯域のアーティファクト
   - 症状: 金属的な響きが付加される
   - 発生条件: 特にボーカル素材で顕著

3. トランジェントの劣化
   - 原因: アタック部分のタイミングずれ
   - 症状: キックやスネアのアタックが鈍る
   - 発生条件: パーカッシブな素材で顕著

4. プリエコー/ポストエコー
   - 原因: フレーム処理によるゴーストサウンド
   - 症状: アタックの前後に微細な音が発生
   - 発生条件: 静寂部分の直前/直後

5. 周波数マスキング
   - 原因: 処理による周波数特性の変化
   - 症状: 特定帯域が強調/減衰
   - 発生条件: 複雑な音色（オーケストラ等）
```

### DDJ-FLX4での操作

```
DDJ-FLX4 Master Tempo操作:

ボタン位置:
- 各デッキに1つずつ搭載
- Tempo Faderの近くに配置

操作方法:
ON:  ボタンを1回押す → ボタンが点灯
OFF: もう一度押す → ボタンが消灯

LED表示:
- 点灯: Master Tempo ON（キー固定）
- 消灯: Master Tempo OFF（通常モード）

rekordbox側の設定:
1. 環境設定 → オーディオ
2. Master Tempoの品質設定:
   - Normal: CPU負荷小、音質標準
   - High: CPU負荷中、音質良好
   - Best: CPU負荷大、音質最高
   推奨: High（バランスが良い）

ショートカット:
- DDJ-FLX4のMaster Tempoボタン
- rekordbox画面上のMaster Tempoアイコンをクリック
- キーボードショートカット（カスタマイズ可能）
```

### 使い分けの詳細ガイド

**Master Tempo OFF（通常モード）が適している場面**

```
推奨シーン:
1. BPM差が小さい場合（±3%以内）
   - キー変化が微小で聴感上問題ない
   - 音質劣化がゼロ
   - CPU負荷が低い

2. アナログ的な音の変化を楽しみたい場合
   - レコードDJの自然な音質変化
   - ビニール（レコード）の温かみ
   - オールドスクールなDJスタイル

3. パーカッシブな音楽のミックス
   - キックやスネアのアタックが鮮明
   - トランジェントが劣化しない
   - ドラム中心の曲に最適

4. 低スペックPCでの使用
   - CPU負荷がほとんどない

## 3. キーチェンジの影響

### BPM変更とキーの関係（詳細）

音楽のキー（調性）は、再生速度の変更によって物理的に変化します。この関係を定量的に理解しておくことが重要です。

```
BPM変更とキー変化の数学的関係:

基本公式:
キー変化（セント） = 1200 × log2(変更後BPM / 元BPM)
※ 100セント = 1半音

計算例:
128 BPM → 130 BPM (+1.56%):
= 1200 × log2(130/128)
= 1200 × log2(1.0156)
= 1200 × 0.02237
= 約 26.8セント（約1/4半音）
→ ほぼ気にならないレベル

128 BPM → 135 BPM (+5.47%):
= 1200 × log2(135/128)
= 1200 × log2(1.0547)
= 1200 × 0.07687
= 約 92.2セント（ほぼ1半音）
→ 顕著なキー変化

128 BPM → 138 BPM (+7.81%):
= 1200 × log2(138/128)
= 1200 × log2(1.0781)
= 1200 × 0.1085
= 約 130.2セント（約1.3半音）
→ 明確なキー変化
```

**BPM変更とキー変化の早見表**

```
BPM変更%  セント  半音     聴感上の影響
────────  ──────  ────     ──────────────
±0.5%     ±9     微小     気づかない
±1.0%     ±17    微小     ほぼ気づかない
±1.5%     ±26    1/4半音  注意深く聴けば分かる
±2.0%     ±34    1/3半音  意識すれば分かる
±2.5%     ±43    約半半音 やや違和感
±3.0%     ±51    半半音   違和感あり（境界線）
±3.5%     ±60    半半音+  明確な違和感
±4.0%     ±68    2/3半音  かなりの変化
±5.0%     ±85    約1半音  大きな変化

### 聞こえ方の違い（詳細分析）

**キー変化（Master Tempo OFF）の聴感的影響**

```
BPMを上げた場合（Master Tempo OFF）:
- ピッチが上がる → 高く、明るく聞こえる
- テンポが速い → エネルギッシュに感じる
- ボーカルが高くなる → チップマンク効果（極端な場合）
- キックが軽くなる → 低域の存在感が減少
- 全体的に「浮いた」印象

BPMを下げた場合（Master Tempo OFF）:
- ピッチが下がる → 低く、暗く聞こえる
- テンポが遅い → ヘビーに感じる
- ボーカルが低くなる → 太く、重厚な印象
- キックが重くなる → 低域が強調
- 全体的に「沈んだ」印象

ジャンル別の許容範囲（Master Tempo OFF）:
- Techno / House: ±3-4%（インスト中心で許容範囲広い）
- Pop / Vocal: ±2%（ボーカルのキー変化が目立つ）
- Hip-Hop: ±2-3%（ラップは比較的許容）
- Trance: ±3%（メロディが目立つため注意）
- D&B: ±3-4%（高BPMで変化が相対的に小さい）
```

**キー固定（Master Tempo ON）の聴感的影響**

```
Master Tempo ON使用時の聴感:

BPMを上げた場合:
- キーは元のまま → ハーモニーが維持される
- テンポだけが速くなる → 自然に聞こえる
- 音質: わずかなタイムストレッチアーティファクト
  → 高域のシャリシャリ感
  → トランジェントのわずかな鈍り
  → ±3%以内ならほぼ気づかない

BPMを下げた場合:
- キーは元のまま → ハーモニーが維持される
- テンポだけが遅くなる → 自然に聞こえる
- 音質: タイムストレッチのアーティファクト
  → 音が若干「引き伸ばされた」感じ
  → リバーブ感が増したように聞こえることがある
  → ±3%以内ならほぼ気づかない

音質劣化を最小限にするコツ:
1. BPM変更を最小限にする
2. 高品質のMaster Tempoアルゴリズムを使用
3. ±6%を超える変更は避ける
4. ローパスフィルターで高域のアーティファクトを隠す
5. ミックスのトランジション中は他のエフェクトで音質変化を目立たなくする
```

## 4. 実践的な使い方

### ビートマッチング時のピッチコントロール

ビートマッチングにおけるピッチコントロールの実践的な手順を、初心者向けに詳しく解説します。

```
完全なビートマッチング手順:

準備段階:
1. デッキAで曲を再生中
2. デッキBに次の曲をロード
3. デッキBのBPMと現在のBPMの差を確認

例: デッキA = 128 BPM, デッキB = 125 BPM（差: 3 BPM / 2.4%）

Step 1: Master Tempoの判断
- BPM差が3%以内 → Master Tempo OFF推奨
- BPM差が3%以上 → Master Tempo ON推奨
- 今回は2.4%なのでOFFで良い

Step 2: Tempo Fader調整
a) BPM表示を見ながらFaderを下方向（プラス）に移動
b) デッキBのBPMが128.0と表示されるまで調整
c) この時点で視覚的に合っているはず

Step 3: CUEポイントの設定
a) デッキBの1拍目にCUEポイントを設定
b) ヘッドフォンでデッキBをモニター
c) デッキAの1拍目に合わせてCUEを叩く

Step 4: ヘッドフォンモニターでの聴覚確認
a) Split CueモードON
b) 左耳: デッキA / 右耳: デッキB（またはその逆）
c) ビートのズレを聴く:
   - 「ドッドッ」と2つに聞こえる = ズレている
   - 「ドン」と1つに聞こえる = 合っている

Step 5: 微調整
a) デッキBが速い場合:
   → Tempo Faderを上（マイナス）に微調整
   → 同時にジョグホイールで一時的に減速

b) デッキBが遅い場合:
   → Tempo Faderを下（プラス）に微調整
   → 同時にジョグホイールで一時的に加速

c) 0.1%単位で調整を繰り返す
d) 8〜16小節間ズレないことを確認

Step 6: トランジション開始
a) BPMが安定したことを確認
b) フェーダーを操作してトランジション開始
c) ミックス中もビートのズレを監視
d) 必要に応じてジョグホイールで微修正
```

### キー確認とハーモニックミキシングの実践

```
rekordbox でのキー表示と確認:

キー表示の設定:
1. rekordboxの表示設定
2. キー表記を選択:
   - Camelot（推奨）: 8A, 8B 形式
   - Classic: Am, C 形式
   - Open Key: 1m, 1d 形式
3. カラー表示ON（キーごとに色分け）

ミックス時のキー確認手順:

パターン1: 完全一致（ベストケース）
曲A: 8A（Am）→ 曲B: 8A（Am）
→ 完璧なハーモニー
→ Master Tempo不要（同BPMの場合）
→ メロディアスな長いトランジションが可能

パターン2: 隣接キー（安全なミックス）
曲A: 8A（Am）→ 曲B: 7A（Dm）  ← 4度上
曲A: 8A（Am）→ 曲B: 9A（Em）  ← 5度上
曲A: 8A（Am）→ 曲B: 8B（C）   ← 平行調
→ 自然で心地よいトランジション
→ メロディが重なっても調和する

パターン3: エナジーブースト（+1半音/+7ステップ）
曲A: 8A（Am）→ 曲B: 3A（Bbm）

→ ビルドアップ → ドロップで使うと効果的
→ 短いトランジションで切り替える

パターン4: キーが離れている（要注意）
曲A: 8A（Am）→ 曲B: 1A（Abm）
→ メロディが重なると不協和音
→ 対策:
  a) 短いトランジション（カットミックス）
  b) メロディをEQでカット
  c) フィルターでメロディ帯域を抑制
  d) エフェクト（リバーブ、ディレイ）で境界をぼかす
```

## 5. ハーモニックミキシング基礎

### Camelot Wheelの完全ガイド

Camelot Wheelは、Mixed In Key社が開発したDJ向けのキー互換性システムです。時計の文字盤のように12の位置に配置され、隣接するキーが調和するように設計されています。

```
Camelot Wheel 完全マップ:

        12B(E)
    11B(A)  1B(B)
  10B(D)      2B(F#)
 9B(G)          3B(Db)
  8B(C)       4B(Ab)
    7B(F)   5B(Eb)
        6B(Bb)

        12A(C#m)
    11A(F#m) 1A(Abm)
  10A(Bm)      2A(Ebm)
 9A(Em)          3A(Bbm)
  8A(Am)       4A(Fm)
    7A(Dm)   5A(Cm)
        6A(Gm)

互換ルール:
1. 同じ番号: 完全互換
   例: 8A + 8A = 完璧

2. ±1番号（同じ列）: 良い互換性
   例: 8A + 7A = Dm（良い）
   例: 8A + 9A = Em（良い）

3. 同番号のA↔B: 良い互換性（平行調）
   例: 8A + 8B = Am + C（良い）

4. ±2番号: やや注意（短いトランジションなら可）
   例: 8A + 6A = 注意

5. それ以外: 避ける（メロディ重複時に不協和）
```

### ハーモニックミキシングの高度なテクニック

```
テクニック1: エナジーブースト
- 現在のキーから+7（Camelot番号）にジャンプ
- 例: 8A → 3A（半音上昇と同等の効果）
- ドロップやクライマックスで効果的
- 短いカットミックスで切り替える

テクニック2: メランコリックドロップ
- メジャーからマイナーへの切替
- 例: 8B(C) → 8A(Am) = 明るい → 暗い
- 感情的な効果が大きい
- ブレイクダウンで使うと効果的

テクニック3: ムードリフト
- マイナーからメジャーへの切替
- 例: 8A(Am) → 8B(C) = 暗い → 明るい
- ビルドアップ後のドロップで使うと爽快感
- 聴衆の気分を高揚させる

テクニック4: サブドミナントミックス
- -1（Camelot番号）への移動
- 例: 8A(Am) → 7A(Dm) = V → I の解決感
- 自然で落ち着いた進行
- チルアウトセクションに適切

テクニック5: ドミナントミックス
- +1（Camelot番号）への移動
- 例: 8A(Am) → 9A(Em) = I → V の浮遊感
- 期待感を生む進行
- ビルドアップに適切
```

### Master Tempo活用によるハーモニックミキシング

```
シナリオ: BPM差が大きい2曲をハーモニックにミックス

曲A: 125 BPM, 8A (Am)
曲B: 132 BPM, 8A (Am)
BPM差: 7 BPM (5.6%)

方法1: Master Tempo OFF
- 曲Bを125 BPMに下げる（-5.6%）
- キー変化: 約-97セント ≈ ほぼ1半音下がる
- 結果: 曲Bのキーが 8A(Am) → 7A(Dm)寄りに
- ハーモニーがずれる
- メロディが重なると違和感あり

方法2: Master Tempo ON（推奨）
- 曲Bを125 BPMに下げる（-5.6%）
- キー: 8A(Am) のまま維持
- 結果: 完璧なハーモニックミックス
- 音質: わずかな劣化あり（-5.6%なので許容範囲）
- DSP処理によるアーティファクトは最小限

方法3: Master Tempo ON + キーシフト
- 曲Bを125 BPMに下げる
- さらにキーを+1半音シフト（8A → 9A に変更）
- エナジーブースト効果を追加
- 高度なテクニックだが効果的

判断基準:
BPM差 ≤ 3%: Master Tempo OFF推奨
BPM差 3-6%: Master Tempo ON推奨
BPM差 6-10%: Master Tempo ON必須
BPM差 > 10%: ミックスの方法自体を再検討
```
