アナログDJの原点、ターンテーブルの使い方を学びます。ターンテーブルはDJ文化の誕生とともに歩んできた最も歴史ある機材であり、デジタル全盛の現代においても多くのDJに愛用されています。本章では、ターンテーブルの基礎知識から実践的なセットアップ方法、カートリッジの選定、DVS（Digital Vinyl System）の活用、スクラッチテクニックまで、包括的に解説します。

## なぜターンテーブルが重要か

### DJの原点としての価値

ターンテーブルは1970年代にDJ文化が誕生して以来、すべてのDJスタイルの基盤となってきた機材です。Hip Hop、House、Techno、Drum and Bass、Jungle、Garage、Dubstep ― これらすべてのジャンルは、ターンテーブルの上で生まれ、進化してきました。

**歴史的な重要性:**
- 1973年、DJ Kool Hercが2台のターンテーブルを使い、同じレコードのブレイクビーツ部分をループさせる手法を開発。これがHip Hopの起源となった
- Grandmaster FlashがQuick Mix Theory（素早いミックス手法）を確立し、ターンテーブルを「楽器」として使う概念を生み出した
- Grand Wizard Theodoreがスクラッチを発明（1977年頃）。レコードを前後に動かして音を出すという革命的な発見
- Larry LevanがParadise Garageで、Frankie KnucklesがThe Warehouseで、ターンテーブルを使った長時間ミックスを展開。これがHouseミュージックの原点となった
- デトロイトのJuan Atkins、Derrick May、Kevin SaundersonがTechnoを創造する際も、ターンテーブルが中心的な役割を果たした

**最も純粋なDJスタイル:**
- CDJやコントローラーと異なり、視覚的な波形表示に頼らない
- 耳だけでビートを合わせる技術が要求される
- レコードの溝を直接触り、物理的に音をコントロールする感覚
- DJとしての基礎体力（耳の訓練）を養う最良の手段

**スクラッチの必須機材:**
- スクラッチはターンテーブルでしか表現できないアートフォーム
- DMC World DJ Championships、IDA（International DJ Association）などの世界大会はターンテーブルが前提
- Turntablism（ターンテーブリズム）という独立した音楽ジャンルが存在する
- Q-Bert、DJ Shadow、Cut Chemist、A-Trakなど、ターンテーブルを楽器として扱うアーティストが多数

### 現代でも重要な理由

**バイナルの温かみのある音質:**
- アナログレコードは連続的な波形を記録しているため、デジタルにはない「温かみ」がある
- 特に低音域の豊かさはアナログならでは。クラブのサウンドシステムで鳴らしたときの空気感が違う
- レコードプレス時の微妙なコンプレッション効果が、独特の「太い」音を生み出す
- マスタリングエンジニアがバイナルカッティング用に特別な処理を施すことが多く、デジタル配信版とは異なる音像を楽しめる

**DVSでデジタルとアナログの融合:**
- タイムコードバイナルを使えば、ターンテーブルの操作感でデジタル音源を再生できる
- Rekordbox、Serato DJ Pro、Traktor Proなど主要DJソフトウェアがDVSに対応
- レコードの重みと操作感を維持しながら、何万曲もの楽曲ライブラリにアクセス可能
- Hot Cue、ループ、エフェクトなどデジタルならではの機能も併用できる

**クラブでの存在感:**
- ターンテーブルが2台並んでいるDJブースは、視覚的にも「本格的」な印象を与える
- レコードを選び、針を落とす所作自体がパフォーマンスとなる
- 特にHouse/Technoシーンでは、CDJとターンテーブルの併用が多い
- BerghainやFabricなど世界的に有名なクラブでもターンテーブルが常設

**コレクターとしての楽しみ:**
- レコードショップ巡りの文化（ディギング/Diggin'）
- 限定プレス、レアグルーヴの発掘
- ジャケットアートの美しさ（12インチの大きなアートワーク）
- 音楽への投資としての価値（レアレコードは数十万円の価値がつくことも）
- 物理メディアならではの所有感と満足感

## 1. ターンテーブルとは

### 定義と基本原理

```
ターンテーブル = レコードを回転させ、溝に刻まれた音声信号を針で読み取り、
                電気信号に変換して出力する機材

DJ用ターンテーブル = 上記に加え、ピッチコントロール（回転速度調整）機能を備え、
                    ビートマッチングやスクラッチが可能な設計
```

**音が出る仕組み（詳細）:**
1. レコードの溝（グルーヴ）には音声波形が物理的に刻まれている
2. 溝は螺旋状に外周から内周へ向かっており、左右の壁にそれぞれLチャンネル・Rチャンネルの情報が記録されている（ステレオの場合）
3. カートリッジの先端にある針（スタイラス）が溝をトレースする
4. 溝の振動がカートリッジ内部のマグネットまたはコイルに伝わり、電磁誘導によって微弱な電気信号が生成される
5. この信号がPhono出力としてミキサーに送られ、Phonoプリアンプで増幅される
6. ミキサーを通じてスピーカーから音が出る

**DJ用とオーディオ用の違い:**

| 項目 | DJ用ターンテーブル | オーディオ用ターンテーブル |
|------|---------------------|--------------------------|
| モーター方式 | ダイレクトドライブ | ベルトドライブが多い |
| トルク | 高トルク（1.5kgf/cm以上） | 低トルク |
| ピッチコントロール | 必須（±8%以上） | なし、またはあっても微調整程度 |
| 耐久性 | 高い（酷使に耐える設計） | 繊細 |
| スタート速度 | 0.7秒以内で33 1/3rpmに到達 | 数秒かかることも |
| 価格帯 | 5万〜25万円 | 数千円〜数百万円 |
| 逆回転 | 対応（スクラッチ必須） | 非対応が多い |

### 構成要素の概要

ターンテーブルは以下の主要な構成要素で成り立っています。それぞれの部品が密接に連携して動作しており、どの一つが欠けても正常な再生はできません。

**プラッター（回転盤）:**
- レコードを載せて回転する円盤
- DJ用は重量級（1.2kg〜2.0kg）が望ましい。重いほど回転の慣性が大きく、安定する
- ダイキャストアルミニウム製が主流
- プラッターの下にモーターが直結している（ダイレクトドライブ方式）

**トーンアーム:**
- カートリッジ（針）をレコード盤面に接触させるためのアーム
- S字型とストレート型がある。DJ用はS字型が標準だが、スクラッチ重視ならストレート型も人気
- カウンターウェイト、アンチスケーティング機構、ハイトアジャスト機能を備える
- アームの共振特性がカートリッジとの相性に影響する

**カートリッジ（針）:**
- レコードの溝を物理的に読み取るトランスデューサー
- 交換式のスタイラス（針先）と本体から構成
- DJ用は高出力、高耐久性、トラッキング性能（スキップしにくさ）が重要
- ヘッドシェルに取り付けてトーンアームにセットする

**ピッチフェーダー:**
- プラッターの回転速度を微調整するスライダー
- ±8%が標準。±16%や±50%に切り替えられるモデルもある
- ビートマッチングの核となる操作子
- フェーダーの精度と滑らかさがDJパフォーマンスの質に直結する

**モーター:**
- プラッターを回転させる駆動装置
- DJ用はダイレクトドライブ方式が必須。ベルトドライブはスクラッチに不向き
- 高トルク設計により、手で触れてもすぐに元の回転速度に復帰する
- ブラシレスDCモーターやACモーターなど、方式によって特性が異なる

**ターゲットライト:**
- レコードの溝を照らすためのLEDライト
- 暗いDJブースでの視認性を確保
- 針の位置確認、レコードのラベル読み取りに使用
- ポップアップ式で、使わないときは収納できるモデルが多い

**スタート/ストップボタン:**
- プラッターの回転開始・停止を制御
- 押した瞬間の応答性が重要。DJ用は即座にフル回転に達する設計
- ストップ時のブレーキ速度を調整できるモデルもある（スピンバック演出に使う）

**33/45切替スイッチ:**
- 回転速度を33 1/3 RPM（LPレコード）と45 RPM（シングルレコード）で切り替える
- 一部モデルでは78 RPMにも対応（SP盤用）
- MK7以降のTechnicsでは電子制御で切替

## 2. Technics SL-1200シリーズ ― DJ用ターンテーブルの世界標準

### 歴史と文化的意義

Technics SL-1200シリーズは、松下電器産業（現パナソニック）が1972年に発売したダイレクトドライブ・ターンテーブルです。元々はオーディオファン向けの高品質ターンテーブルとして設計されましたが、その高いトルク、堅牢な構造、正確なピッチコントロールがDJの要求に完璧にマッチし、やがて世界中のDJブースの標準機材となりました。

**詳細な年表:**

```
1970年: Technics SP-10（世界初のダイレクトドライブ・ターンテーブル）発売
        → この技術がSL-1200シリーズの基盤となる

1972年: SL-1200（初代）発売
        → 定価 49,800円。オーディオ用として設計
        → ダイレクトドライブ方式を民生用に初めて採用

1979年: SL-1200MK2 発売（DJ文化を変えた革命的モデル）
        → ピッチフェーダーの範囲を±8%に拡大
        → ピッチリセットボタン追加
        → ストロボスコープ改良
        → ポップアップ式ターゲットライト搭載
        → ニューヨークのDJたちが飛びつき、瞬く間にクラブ標準となる
        → 定価 59,800円

1989年: SL-1200MK3 発売
        → MK2の小改良版。ピッチフェーダーの精度向上
        → 日本市場中心の展開

1997年: SL-1200MK3D 発売
        → ピッチリセットインジケーター追加
        → ブレーキ速度調整機能
        → 外装デザイン変更

2002年: SL-1200MK5 発売
        → トーンアームの改良（高さ調整機構強化）
        → ピッチフェーダーの精度さらに向上
        → MK5G（ゴールドモデル）も展開

2008年: SL-1200MK6 発売
        → MK5の後継。生産終了を見据えた最終改良版
        → 限定生産モデルとしてのプレミアム感

2010年: 生産終了発表
        → 全世界のDJコミュニティに衝撃が走る
        → 累計生産台数350万台以上
        → 中古市場価格が急騰

2016年: SL-1200G / SL-1200GAE 発売（復活）
        → ハイエンドオーディオ用として復活
        → コアレスダイレクトドライブモーター採用
        → 価格帯30万円〜（オーディオ用のためDJ使用には高価すぎる）

2019年: SL-1200MK7 発売（DJ向けの真の復活）
        → 約18万円（税込）
        → デジタルピッチコントロール（±8/16/50%切替）
        → 逆再生対応
        → トルク調整機能
        → ブラシレスDCモーター採用
        → 現代のDJニーズに完全対応

2022年: SL-1200M7L 発売（限定カラーモデル）
        → 約25万円
        → 7色のカラーバリエーション
        → ストレートトーンアーム標準装備
        → DVS完全対応設計
        → DJ Jazzy Jeff監修
```

### モデル比較（詳細）

#### SL-1200MK2（中古市場での伝説）

世界で最も多くのDJブースに設置されたターンテーブル。生産終了から15年以上経った現在でも、中古市場で安定した需要があります。

```
基本スペック:
- 駆動方式: ダイレクトドライブ
- モーター: ブラシ付きDCモーター
- 回転数: 33 1/3 / 45 RPM
- ピッチ範囲: ±8%
- ワウ・フラッター: 0.01% WRMS
- S/N比: 78dB（DIN-B）
- トルク: 1.5kgf/cm（起動時）
- 起動時間: 0.7秒（33 1/3 RPM到達まで）
- プラッター重量: 約1.6kg（ダイキャストアルミ）
- 本体重量: 約12kg
- 外形寸法: 453mm × 353mm × 162mm

中古市場情報:
- 相場: 50,000〜100,000円（状態により大きく変動）
- 極上品: 100,000〜150,000円
- ジャンク品: 20,000〜40,000円（修理前提）
- 入手先: ハードオフ、メルカリ、ヤフオク、DJ専門中古ショップ
```

**MK2が「伝説」と言われる理由:**
1. 圧倒的な耐久性 ― 30年以上前の個体が現役で使われている例が珍しくない
2. パーツの互換性 ― アフターマーケットのパーツが豊富（ピッチフェーダー、RCAケーブル、ヒンジなど）
3. シンプルな構造 ― デジタル部品がほぼなく、故障箇所が限られる
4. 「枯れた技術」の安心感 ― 挙動が完全に把握されており、意外な故障がない
5. 世界中のクラブに設置されているため、どこでプレイしても同じ操作感

**MK2購入時の注意点（プロのTips）:**
- ピッチフェーダーのガリ（ノイズ）は最も多いトラブル。フェーダーを動かしたときにピッチが安定しないならNG
- プラッターの回転むら確認 ― ストロボスコープの点が流れないか確認
- トーンアームのガタつき ― 左右に揺らして遊びが大きすぎないか
- RCAケーブルの断線 ― 片チャンネルだけ音が出ないケースが多い（交換可能だが手間）
- フット（脚）の状態 ― インシュレーターが劣化していないか。交換部品は入手可能

#### SL-1200MK7（現行DJ向けモデル）

2019年にDJ向けとして復活した最新モデル。MK2の操作感を継承しつつ、現代のDJニーズに対応した機能を追加しています。

```
基本スペック:
- 駆動方式: ダイレクトドライブ
- モーター: ブラシレスDCモーター（コアレス・ステーター）
- 回転数: 33 1/3 / 45 / 78 RPM
- ピッチ範囲: ±8% / ±16% / ±50% 切替
- ワウ・フラッター: 0.025% WRMS
- S/N比: 67dB（DIN-B）
- トルク: 高/中/低 の3段階調整可能
- 起動時間: 0.7秒（33 1/3 RPM到達まで）
- ブレーキ速度: 調整可能
- 逆回転: 対応
- プラッター重量: 約1.8kg（2層構造ダイキャストアルミ）
- 本体重量: 約11.5kg
- 外形寸法: 453mm × 353mm × 173mm
- 価格: 約180,000円（税込）

MK2からの主な改良点:
- デジタルピッチコントロール（ドリフトが発生しない）
- ±16%/±50%のワイドピッチ対応
- 逆再生機能（スイッチ一つで逆回転）
- トルク調整（スクラッチ用に高トルク、ミックス用に低トルクなど切替可能）
- ブレーキ速度調整（スピンバック演出のカスタマイズ）
- 78 RPM対応（SP盤の再生可能）
- デタッチャブルRCAケーブル（ケーブル交換が容易）
```

**プロの評価:**
- MK2の操作感にかなり近いが「完全に同じ」ではないという意見も
- デジタルピッチはドリフト（徐々にずれる現象）がなく正確
- トルク調整機能は非常に便利。ジャンルや用途に応じて変更できる
- 逆再生機能はバトルDJやターンテーブリストに好評
- 価格が18万円と高めだが、新品保証付きの安心感がある

#### SL-1200M7L（限定カラーモデル）

DJ Jazzy Jeffが監修した特別モデル。ストレートトーンアーム標準装備で、DVS・スクラッチに特化した設計。

```
基本スペック:
- MK7をベースにした特別仕様
- ストレートトーンアーム標準装備
  → S字型に比べてトラッキングエラーが少ない
  → スクラッチ時の安定性が高い
  → DVS使用時の追従性が向上
- カラーバリエーション: 全7色
  （ブラック、ホワイト、レッド、ブルー、グリーン、イエロー、ベージュ）
- 価格: 約250,000円（税込）
- 限定生産のためプレミアム感あり
```

### 他メーカーのDJ用ターンテーブル

Technics以外にも優れたDJ用ターンテーブルは存在します。予算や用途に応じて選択肢を検討しましょう。

#### Reloop RP-8000 MK2

デジタル機能を統合した革新的なターンテーブル。MIDI機能内蔵で、DJソフトウェアとのシームレスな連携が可能です。

```
基本スペック:
- 価格: 約80,000円
- 駆動方式: クォーツロック・ダイレクトドライブ
- トルク: 調整可能（強/弱）
- ピッチ範囲: ±8% / ±16% / ±50%
- MIDI対応: 8つのパフォーマンスパッド搭載
  → Hot Cue、ループ、サンプラーなどをパッドから直接操作
- Serato DJ Pro / rekordbox対応
- USBハブ内蔵（2ポート）
- トーンアーム: S字型
- Platter Lock機能（DVS使用時にプラッターを固定）

メリット:
✓ MIDIパッドでDJソフトウェアを直接操作可能
✓ コストパフォーマンスが非常に高い
✓ DVS運用時の利便性が高い
✓ Serato/rekordboxとのネイティブ対応

デメリット:
✗ Technicsと比べるとビルドクオリティがやや劣る
✗ トーンアームの精度がSL-1200に及ばない
✗ 中古市場でのリセールバリューが低い
```

#### Pioneer DJ PLX-1000

Pioneer DJが「クラブスタンダード」を目指して開発したハイトルクターンテーブル。SL-1200MK2のフィーリングを現代に再現することをコンセプトとしています。

```
基本スペック:
- 価格: 約120,000円
- 駆動方式: ダイレクトドライブ
- モーター: 3相ブラシレスDCモーター
- トルク: 4.5kgf/cm以上（非常に高い）
- ピッチ範囲: ±8% / ±16% / ±50%
- 起動時間: 0.3秒以内（超高速）
- S/N比: 60dB以上
- プラッター重量: 約1.8kg（亜鉛ダイキャスト）
- デタッチャブルRCAケーブル
- トーンアーム: S字型・スタティックバランス
- 本体重量: 約13.1kg
- テンポリセット機能

メリット:
✓ 非常に高いトルク（SL-1200MK2の約3倍）
✓ 起動時間が極めて短い
✓ Pioneer DJエコシステムとの親和性
✓ クラブでの実績が増えている
✓ 頑丈な筐体設計

デメリット:
✗ SL-1200と操作感が微妙に異なる（慣れが必要）
✗ ピッチフェーダーの質感がSL-1200に劣るとの声も

## 3. ターンテーブルの各部名称と機能

### 基本構造（図解）

```
            ┌──────────────────────────────────────────┐
            │                                          │
            │   [ターゲットライト]  ┌─────────────────┐ │

            │  │               │      ↓              │
            │  │  プラッター     │   ┌──┐            │

              ↑                          ↑
           電源スイッチ              RCA出力 + GND

### 各部の詳細解説

#### プラッター（Platter）

プラッターはターンテーブルの心臓部とも言える回転盤です。レコードを載せて安定した回転を提供する役割を担います。

**材質と重量:**
- ダイキャストアルミニウム製が主流（SL-1200シリーズ）
- 重量は1.2kg〜2.0kgが一般的。重いほど慣性モーメントが大きく、回転が安定する
- PLX-1000は亜鉛ダイキャスト製で約1.8kg。より高い安定性を実現
- 一部のモデルではゴム製のデッドニングシートがプラッター裏面に貼られており、共振を抑制

**スリップマット:**
- プラッターとレコードの間に敷く滑りやすいマット
- フェルト製が標準。レコードとの摩擦を適度に減らし、スクラッチ時にレコードだけを動かせるようにする
- プラッターのゴムマット（ターンテーブルマット）の上にスリップマットを載せる
- スクラッチDJは2枚重ね（ダブルスリップマット）にすることも。より滑りやすくなる
- コルク製、レザー製、シリコン製など素材の選択肢も多い
- Dr. Suzuki（日本製）のスリップマットは世界的に定評がある

**スリップマットの選び方（プロのTips）:**
```
用途別おすすめ:
- スクラッチ: フェルト薄手 or Dr. Suzuki Mix Edition（滑りやすさ重視）
- ミックス: フェルト標準厚 or コルク（適度なグリップ）
- リスニング: ゴムマット or レザー（振動吸収重視）

よくある間違い:
✗ ゴムマットの上に直接レコードを載せる → スクラッチ不可能
✗ スリップマットを裏返して使う → 滑りすぎて制御不能
✗ 汚れたスリップマットを使い続ける → レコードに傷がつく
```

#### トーンアーム（Tonearm）

トーンアームはカートリッジをレコード盤面に正確に接触させるための精密部品です。DJ用ターンテーブルでは、スクラッチやバックスピンなどの激しい操作にも耐える堅牢さが求められます。

**S字型トーンアーム:**
- SL-1200シリーズの標準トーンアーム
- S字カーブにより、針先がレコードの溝に対して適切な角度（オーバーハング）を維持
- トラッキングエラー（針先とレコード溝の角度ずれ）を最小限に抑える設計
- 一般的なDJプレイには最適な選択

**ストレートトーンアーム:**
- SL-1200M7Lやバトルスタイルのターンテーブルに採用
- スクラッチ時のスキップが少ない（針圧が均一にかかりやすい）
- DVS使用時のトラッキング性能が高い
- トラッキングエラーはS字型よりやや大きいが、DJ用途では問題にならない

**トーンアームの調整項目:**

1. **カウンターウェイト（Counter Weight）:**
   - トーンアームの後端にある重り
   - 回転させることで針圧を調整
   - まずアームが水平になるバランスポイントを見つけ、そこから針圧分だけ追加する

2. **アンチスケーティング（Anti-Skating）:**
   - レコードの回転により針が内側に引っ張られる力を相殺する機構
   - 通常は針圧と同じ値に設定する（例: 針圧3gならアンチスケートも3）
   - スクラッチ時はアンチスケートを0にするDJも多い（レコードを前後に動かすため、一方向への補正は不要）

3. **ハイトアジャスト（Height Adjust）:**
   - トーンアームの高さ調整
   - カートリッジ/ヘッドシェルの高さに応じて調整
   - 針先がレコードに対して垂直に接触するようにする
   - 適切に調整しないと片チャンネルの音が弱くなったり、針飛びの原因になる

#### カートリッジ取付部（ヘッドシェル）

ヘッドシェルはカートリッジをトーンアームに固定するためのアダプターです。

```
ヘッドシェルの構造:
┌──────────────────┐
│  接続端子（4ピン）  │ ← トーンアーム側
│  ┌────────────┐  │
│  │カートリッジ   │  │ ← ネジ2本で固定
│  │  本体      │  │
│  └──┤        ├──┘
│     │スタイラス│      ← 交換可能な針先
│     └────────┘
└──────────────────┘

配線（色分け）:
- 白: L+ （左チャンネル・ホット）
- 青: L- （左チャンネル・グラウンド）
- 赤: R+ （右チャンネル・ホット）
- 緑: R- （右チャンネル・グラウンド）

※ 配線を間違えると位相が逆転したり、片チャンネルしか音が出ない
```

**ヘッドシェルの種類:**
- 標準型（SL-1200付属品など）: 汎用性が高く、ほとんどのカートリッジに対応
- 軽量型（DJ Competition用）: 素材をカーボンファイバーやチタンにし、共振を抑制
- Ortofon Concordeタイプ: ヘッドシェルとカートリッジが一体化。交換が簡単で、クラブでの運用に最適

#### ピッチフェーダー（Pitch Fader / Tempo Slider）

ピッチフェーダーはプラッターの回転速度を微調整するスライダーで、ビートマッチングの核となる操作子です。

**基本的な仕組み:**
- フェーダーを上に動かす → 回転速度が遅くなる → ピッチが下がる（BPMが下がる）
- フェーダーを下に動かす → 回転速度が速くなる → ピッチが上がる（BPMが上がる）
- 中央位置（0%）でオリジナルの回転速度

**注意: SL-1200シリーズのピッチフェーダー方向:**
```
SL-1200のピッチフェーダーは「逆向き」と感じるDJが多い:
- 奥に押す（上方向） → 速くなる → BPM上がる
- 手前に引く（下方向） → 遅くなる → BPM下がる

これはCDJのピッチフェーダーと逆の方向。
Technicsに慣れているDJがCDJを使うと最初混乱する。
（一部のCDJではピッチフェーダーの方向を設定で反転可能）
```

**ピッチ範囲の使い分け:**
```
±8%: 標準的なビートマッチングに最適
     → 同ジャンル内のミックスには十分
     → 精度が高く、微調整がしやすい

±16%: ジャンルを跨ぐミックスに有用
      → 例: 120BPMのHouseと140BPMのDrum and Bassのブリッジング
      → ±8%では届かないBPM差をカバー

±50%: 特殊用途
      → ハーフスピード再生やダブルスピード再生
      → 実験的なサウンドデザイン
      → バトルDJのトリック
```

#### ストロボスコープ（Stroboscope）

プラッター側面に刻まれたドット（点）パターンと、専用のストロボライトを組み合わせて回転速度を視覚的に確認する機構です。

```
仕組み:
- ストロボライトが交流電源の周波数（50Hz/60Hz）で点滅
- プラッターが正確な速度で回転しているとき、ドットが静止して見える
- ドットが時計回りに流れる → 速すぎる
- ドットが反時計回りに流れる → 遅すぎる
- ピッチフェーダーが0%の位置でドットが止まっていることを確認する

注意:
- 50Hz/60Hz切替が必要な地域もある（日本は東日本50Hz、西日本60Hz）
- LED照明下ではストロボが正しく機能しないことがある
- MK7ではデジタル制御のため、ストロボスコープの重要性はやや低下
```

## 4. セットアップ方法 ― 基本から応用まで

### 設置場所の選定

ターンテーブルのセットアップで最も重要なのは「設置場所」です。不適切な設置は音質劣化、針飛び、機材の故障につながります。

**理想的な設置条件:**
```
1. 水平であること
   → 水準器（スマホアプリでも可）で確認
   → わずかな傾きでもトーンアームの動作に影響する
   → 特に針圧やアンチスケーティングの精度に直結

2. 振動が少ないこと
   → スピーカーの振動が伝わらない場所
   → クラブではDJブースとスピーカーが分離されている設計が理想
   → 自宅では、スピーカーとターンテーブルを同じ台に置かない

3. 安定した台の上
   → DJ用ラックまたは頑丈なテーブル
   → IKEAのKALLAX（カラックス）は重量に注意（ターンテーブル2台+ミキサーで30kg以上）
   → 専用DJファニチャー（Glorious、Zomo等）が理想

4. 適切な高さ
   → 立ってDJする場合: 台の高さ約90cm（肘が軽く曲がる高さ）
   → 座ってDJする場合: 台の高さ約65-70cm
   → レコードの操作がしやすい高さを優先

5. 振動対策（アイソレーション）
   → ターンテーブルの脚（インシュレーター）が正しく機能していることを確認
   → 追加の振動対策として、ソルボセインシートやゴムパッドを台との間に敷く
   → 極端な場合、壁掛けシェルフ（棚）に設置するDJもいる（床からの振動を完全遮断）
```

**自宅DJ環境の振動対策（プロのTips）:**
```
問題: 大音量で鳴らすとスピーカーの振動がターンテーブルに伝わり、
     低音のフィードバック（ハウリング）や針飛びが発生する

対策レベル1: スピーカーとターンテーブルを別の台に設置
対策レベル2: ターンテーブルの下にソルボセインシート or ゴムマットを敷く
対策レベル3: ターンテーブルを石板（御影石など）の上に設置し、
            石板と台の間にインシュレーターを挟む
対策レベル4: 壁掛けシェルフにターンテーブルを設置（究極の振動対策）
```

### 基本セットアップ手順（詳細）

以下の手順に従って、ターンテーブルを正しくセットアップしましょう。初めての方は、一つ一つ確認しながら進めてください。

**Step 1: 開梱と設置**
```
1. ターンテーブルを箱から取り出す（必ず両手で持つ）
2. 付属品を確認:
   - ターンテーブル本体
   - ゴムマット（ターンテーブルマット）
   - スリップマット（付属しないモデルもある）
   - ヘッドシェル（カートリッジ付属の場合も）
   - カウンターウェイト
   - RCAケーブル（デタッチャブルタイプの場合）
   - アース線
   - 電源ケーブル
   - EPアダプター（45回転レコード用）
   - 取扱説明書
3. 設置場所に水平に置く
4. 脚（インシュレーター）が4つとも接地していることを確認
5. 水準器で水平を確認し、必要に応じて脚を調整
```

**Step 2: ゴムマットとスリップマットの装着**
```
1. プラッターにゴムマット（ターンテーブルマット）を載せる
   → 中央の軸（スピンドル）にゴムマットの穴を通す
2. ゴムマットの上にスリップマットを載せる
   → スクラッチする場合はスリップマット必須
   → リスニングのみならゴムマットだけでもOK
3. スリップマットがプラッターの中心にあることを確認
```

**Step 3: カートリッジの取り付け**
```
1. ヘッドシェルにカートリッジを取り付ける
   → 2本のネジで固定（付属のワッシャーを忘れずに）
   → この時点ではネジを仮締めにしておく（後でオーバーハング調整するため）
2. リード線（4本の細い線）を接続
   → 白 = L+、青 = L-、赤 = R+、緑 = R-
   → 小さなピンなので、ピンセットか細いラジオペンチを使うと楽
3. カートリッジの位置を調整（オーバーハング）
   → カートリッジに付属のゲージ（アライメントプロテクター）を使う
   → 正しい位置にカートリッジを合わせたらネジを本締め
4. ヘッドシェルをトーンアームに取り付ける
   → ロックナットを手で回して固定
```

**Step 4: トーンアームバランス調整（ゼロバランス）**
```
1. トーンアームロックを解除
2. トーンアームリフターを下げる
3. カウンターウェイトをトーンアームの後端にねじ込む
4. トーンアームを持ち上げ、アームレストから外す
5. カウンターウェイトをゆっくり回して、トーンアームが水平に浮くポイントを見つける
   → これが「ゼロバランス」の位置
   → この状態では針圧0g
6. カウンターウェイトを動かさずに、目盛りリング（ダイヤル）だけを回して0に合わせる
```

**Step 5: 針圧設定**
```
1. ゼロバランスが取れた状態から、カウンターウェイトを時計回りに回す
2. カートリッジの推奨針圧に設定:
   - Ortofon Concorde MKII DJ: 3.0g
   - Ortofon Concorde MKII Mix: 3.0g
   - Ortofon Concorde MKII Club: 3.0g
   - Shure M44G: 1.25g（推奨）〜 3.0g（DJ用最大）
   - Audio-Technica AT-XP7: 2.0g（推奨）
3. 目盛りリングの数字が設定したい針圧値を指すまで回す

よくある間違い:
✗ ゼロバランスを取らずにいきなり針圧を設定する → 正確な針圧にならない
✗ 針圧計を使わない → 目盛りには誤差がある。正確を期すなら針圧計を使う
✗ 針圧を重くしすぎる → レコードと針の寿命が短くなる
✗ 針圧を軽くしすぎる → スクラッチ時に針が飛ぶ
```

**Step 6: アンチスケーティング設定**
```
1. アンチスケーティングダイヤルを針圧と同じ値に設定
   → 例: 針圧3.0gならアンチスケーティングも3.0
2. レコードを再生して確認:
   → 溝のない部分で針が内側にも外側にも動かないのが理想
   → 内側に流れるなら数値を上げる
   → 外側に流れるなら数値を下げる

DJ用の特別な設定:
- スクラッチ主体の場合: アンチスケーティングを0に設定するDJが多い
  → レコードを前後に動かすため、一方向への補正は邪魔になる
- ミックス主体の場合: 針圧と同値が基本
- DVS使用時: カートリッジの推奨値に従う
```

**Step 7: ケーブル接続**
```
1. RCAケーブルをミキサーのPhono入力に接続
   → 赤 = R（右チャンネル）
   → 白 = L（左チャンネル）
   → 必ずPhono入力を使う（Line入力ではPhonoイコライザーがかからない）
2. アース線（GND）をミキサーのGND端子に接続
   → アース線を接続しないと「ブーン」というハム音が発生する
   → 接続してもハム音が消えない場合は接触不良の可能性
3. 電源ケーブルを接続
   → アース付きコンセントの使用を推奨

よくある間違い:
✗ Phono入力ではなくLine入力に接続 → 音が極端に小さく、低音過剰になる
✗ Line入力をPhono入力に接続 → 音が歪んで割れる
✗ アース線を接続しない → 盛大なハム音が発生
✗ RCAの左右を逆に接続 → ステレオが左右逆になる
```

**Step 8: 動作確認**
```
1. ターンテーブルの電源を入れる
2. 33 1/3 RPMに設定
3. ストロボスコープのドットが静止していることを確認
4. ピッチフェーダーを0%に設定（Resetボタンで確認）
5. レコードを載せ、針を落とす
6. 音が正常に出ることを確認（左右両チャンネル）
7. ピッチフェーダーを動かし、速度が変化することを確認
8. トーンアームリフターの動作確認（上げ下げがスムーズか）
```

### DVS用セットアップの追加手順

DVS（Digital Vinyl System）を使用する場合は、上記の基本セットアップに加えて以下の設定が必要です。

```
追加で必要な接続:
1. ターンテーブル → ミキサー（Phono入力）：通常通り
2. ミキサー → PC/Mac（USB接続）
   → DVS対応ミキサーの場合はUSB1本でOK
   → DVS非対応ミキサーの場合は別途オーディオインターフェースが必要
3. PC/MacでDJソフトウェア起動
4. タイムコードバイナルをターンテーブルに載せる

タイムコードバイナルの注意点:
- Rekordbox用、Serato用、Traktor用は別のタイムコード → 互換性なし
- タイムコードバイナルは消耗品。溝が摩耗するとトラッキング性能が落ちる
- 新品のタイムコードバイナルは年に1回程度交換するのが理想
- 針圧は推奨値より若干重めに設定すると安定する
```

## 5. カートリッジの選び方 ― 用途別の最適解

### カートリッジとは

カートリッジ（フォノカートリッジ）は、レコードの溝に刻まれた物理的な振動を電気信号に変換するトランスデューサーです。音質、トラッキング性能、耐久性など、DJ体験に直結する最も重要なパーツの一つです。

**カートリッジの構造:**
```
┌─────────────────────────────┐
│  カートリッジ本体             │
│  ┌───────────────────────┐ │
│  │ マグネット / コイル      │ │ ← 電気信号を生成する部分
│  │ カンチレバー（支持棒）   │ │ ← スタイラスの振動を伝達
│  │ ダンパー（制振ゴム）     │ │ ← 振動を適切に制御
│  └───────┤              ├─┘
│          │ スタイラス     │     ← レコード溝に接触する針先
│          │ （交換針）    │     ← 消耗品。定期的に交換が必要
│          └──────────────┘
└─────────────────────────────┘
```

**スタイラス（針先）の種類:**
```
丸針（Spherical/Conical）:
- DJ用の標準
- レコード溝との接触面積が広い
- スクラッチ時のスキップが少ない
- 音質はやや劣るが、耐久性が高い
- DJ用カートリッジのほとんどがこの形状

楕円針（Elliptical）:
- リスニング・ミックス用
- レコード溝のより細かい情報を読み取れる
- 高音域の再現性に優れる
- スクラッチには不向き（溝へのダメージが大きい）

ラインコンタクト/シバタ針:
- ハイファイオーディオ用
- DJ用途には完全に不向き
- 針先が非常に繊細で、スクラッチは厳禁
```

### MM型とMC型の違い

カートリッジには大きく分けてMM型とMC型の2種類があります。DJ用途ではMM型一択ですが、知識として両方を理解しておくことは有益です。

**MM型（Moving Magnet）:**
```
仕組み:
- カンチレバーの後端にマグネットが取り付けられている
- レコード溝の振動でマグネットが動き、コイルに電気信号が誘導される

DJ向きの理由:
✓ 高出力（3-10mV） → Phonoプリアンプの増幅が少なくて済む → S/N比が良い
✓ 交換針が安価 → 消耗品コストが低い
✓ 堅牢な設計 → スクラッチの衝撃に耐える
✓ ほぼすべてのDJミキサーのPhono入力に対応

DJ用カートリッジは100% MM型
```

**MC型（Moving Coil）:**
```
仕組み:
- カンチレバーの後端にコイルが取り付けられている
- コイルが動くことで電気信号を生成

DJ用途に不向きな理由:
✗ 低出力（0.1-0.5mV） → 専用のMC対応Phonoプリアンプが必要
✗ 針交換ができない（本体ごと交換 or メーカー修理）
✗ 繊細な設計 → スクラッチの衝撃で壊れる
✗ 高価（数万〜数十万円）
✗ DJ用のMC型カートリッジは存在しない

MC型はオーディオファン向けのリスニング専用
```
