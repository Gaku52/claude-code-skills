# Trance

感情的な高揚感と壮大なビルドアップが特徴的なトランス。クラブからフェスまで、観客を感動的な音楽の旅へと導くジャンルです。

## 1. Trance の歴史

### 起源（1990年代初期 ドイツ/ゴア）

**1992-1994: Trance の誕生**

```
場所: フランクフルト、ベルリン（ドイツ）

起源:
Techno + Acid House + New Age
= Trance

特徴:
- 反復的なシンセメロディ
- エモーショナルなビルドアップ
- 130-140 BPM
- トライバルなリズム
```

**キーパーソン:**
- Sven Väth（Frankfurt）
- Paul van Dyk（Berlin）
- Jam & Spoon
- Cosmic Baby

**初期の名曲:**
- Age of Love - The Age of Love (1990)
- Energy 52 - Café Del Mar (1993)
- Jam & Spoon - Stella (1992)
- Paul van Dyk - For An Angel (1994)

### 1995-1999: ゴアトランスとプログレッシブ

**ゴアトランス（Goa Trance）:**

```
場所: ゴア、インド

特徴:
- サイケデリックなサウンド
- 複雑なシンセライン
- スピリチュアルな雰囲気
- 138-145 BPM

代表的アーティスト:
- Astral Projection
- Juno Reactor
- Infected Mushroom (初期)
- Hallucinogen
```

**プログレッシブトランス台頭:**
- より洗練されたサウンド
- ハウス的な要素
- 長いミックス展開
- Sasha、John Digweed の影響

### 2000-2004: 黄金時代

## 2. 音楽的特徴

### BPM とリズム

**典型的なBPM:**
- Progressive Trance: 128-132 BPM
- Uplifting Trance: 136-140 BPM
- Psytrance: 138-145 BPM
- Tech Trance: 135-138 BPM

**リズムパターン:**

```
キックドラム:
4/4 ビート（4つ打ち）
|ドン|　　|ドン|　　|ドン|　　|ドン|　　|
 1    2    3    4

特徴:
- Deep でパワフルなキック
- オープンハイハット（16分音符）
- ドライブ感のあるベースライン
```

### メロディとハーモニー

**トランス特有のメロディ:**

```
典型的な展開:

1. Intro（イントロ）
   └─ ミニマルなビート
      時間: 0:00-1:30

2. Build-up（ビルドアップ）
   └─ シンセが徐々に追加
      時間: 1:30-3:00

3. Breakdown（ブレイクダウン）
   └─ 感情的な展開
      ボーカルやメロディ
      時間: 3:00-4:30

4. Climax（クライマックス）
   └─ 全要素が爆発
      最高潮の瞬間
      時間: 4:30-6:00

5. Outro（アウトロ）
   └─ 徐々に減衰
      次の曲へ
      時間: 6:00-8:00
```

**コードプログレッション:**
- マイナーキーが多い
- エモーショナルな展開
- アルペジオシンセ
- パッドサウンド

### サウンドデザイン

**典型的な音色:**

```
1. シンセリード:
   - TB-303風アシッドライン
   - Supersaw（壮大なサウンド）
   - アルペジエーター

2. パッド:
   - アンビエントな雰囲気
   - リバーブたっぷり
   - ストリングス的

3. プラック:
   - 短いスタッカート音
   - リズミカルな要素
   - 16分音符パターン

4. エフェクト:
   - ホワイトノイズスウィープ
   - フィルタースウィープ
   - リバースシンバル
```

## 6. DJ技法とミキシング

### 基本ミックス技法

**ロングミックス（2-4分）:**

```
トランスの基本:

理由:
- 長いイントロ/アウトロ構造
- ビルドアップを活かす
- ジャーニー重視
- 感情的な展開

推奨ミックス時間:
- Progressive: 3-4分
- Uplifting: 2-3分
- Psytrance: 1-2分
- Tech Trance: 2-3分
```

**EQミックス:**

```
DDJ-FLX4 での設定:

Track A（プレイ中）:
├─ HI: 12時（0）
├─ MID: 12時（0）
└─ LOW: 12時（0）

Track B（次の曲、イントロ）:
├─ HI: 9時（-∞）
├─ MID: 9時（-∞）
└─ LOW: 12時（0） ← ベースは残す

徐々に:
1. B の HI を上げる（30秒）
2. B の MID を上げる（30秒）
3. A の HI を下げる（30秒）
4. A の MID を下げる（30秒）
5. A の LOW を下げる（30秒）
6. スムーズにトランジション完了
```

### ブレイクダウンミックス

**感情的なトランジション:**

```
タイミング:

Track A: ブレイクダウン開始
       ↓
      (キックが抜ける)
       ↓
Track B: イントロ開始
       ↓
      (ベースラインのみ)
       ↓
Track A: フェードアウト
       ↓
Track B: ビルドアップ開始
       ↓
      (クライマックスへ)

時間: 約2-3分
```

**具体例:**

```
曲: Above & Beyond - Sun & Moon

0:00-1:30: イントロ
          → 前の曲とミックス開始

3:00-4:30: ブレイクダウン
          → 次の曲のイントロ開始
          → 感情的な瞬間

4:30-6:00: クライマックス
          → 前の曲完全フェードアウト
```

### キーミキシング（ハーモニック）

**トランスでの重要性:**

```
理由:
- メロディが豊富
- ハーモニーが重要
- 感情的な展開

推奨トランジション:
Perfect 5th: +7 semitones
Perfect 4th: +5 semitones
Minor 3rd: +3 semitones

例:
Track A: 8A (A minor)
       ↓
Track B: 8B (C major) ← Perfect!
       または
Track B: 9A (E minor) ← Perfect!
```

**DDJ-FLX4 での確認:**

```
Rekordbox:
1. [Collection] で曲を選択
2. [Key] カラムで確認
   例: 8A, 5B, 12A

3. ソート機能:
   - [Key] をクリックしてソート
   - 同じキーの曲が集まる

4. 次の曲選択:
   - ±1 の番号（例: 8A → 9A）
   - 同じ番号で A↔B（例: 8A → 8B）
```

### エフェクト活用

**DDJ-FLX4 エフェクト:**

**1. Echo（エコー）:**

```
使用タイミング:
- ブレイクダウン前
- ビルドアップ中
- トランジション

設定:
[BEAT] ボタン: 1/2（2分音符）
[LEVEL/DEPTH] ノブ: 12-1時

手順:
1. ブレイクダウン8小節前
2. Echo ON
3. LEVEL を徐々に上げる
4. ブレイクダウン突入でOFF
```

**2. Reverb（リバーブ）:**

```
使用タイミング:
- ボーカルパート
- ブレイクダウン
- 感情的な瞬間

設定:
[BEAT] ボタン: 2/1（2小節）
[LEVEL/DEPTH] ノブ: 10-12時

効果:
- 空間的な広がり
- 感情的な深み
```

**3. Filter（フィルター）:**

```
High Pass Filter:
- イントロ開始時
- 徐々に曲を導入

Low Pass Filter:
- ブレイクダウン前
- トランジション
- 雰囲気作り

Smart CFX (DDJ-FLX4):
[SMART CFX] ノブを回す:
- 左: Low Pass Filter
- 右: High Pass Filter
```

### エナジーマネジメント

**90分セットの構築:**

```
0-15分: Warm Up（導入）
├─ BPM: 128-132
├─ スタイル: Progressive
└─ 目的: 雰囲気作り

15-30分: Build Up（上昇）
├─ BPM: 132-136
├─ スタイル: Progressive → Uplifting
└─ 目的: エナジー上昇

30-60分: Peak Time（ピーク）
├─ BPM: 136-140
├─ スタイル: Uplifting/Anthem
└─ 目的: クライマックス、感動

60-75分: Mid Energy（中盤）
├─ BPM: 136-138
├─ スタイル: Tech Trance/Vocal
└─ 目的: 一息、でもエナジー維持

75-90分: Closing（クロージング）
├─ BPM: 138-140
├─ スタイル: Anthem/Classic
└─ 目的: 最高の瞬間、記憶に残る
```

## 11. サブジャンル詳細ガイド

トランスの各サブジャンルには独自の文化、サウンド、歴史があります。ここでは主要サブジャンルをさらに深掘りし、DJとしての理解を深めます。

### Psytrance（サイトランス）詳細

**歴史と文化的背景:**

```
起源: 1960年代ヒッピームーブメント → 1980年代ゴア、インド

タイムライン:
1960s: ヒッピーがインド・ゴアに集まる
1970s: フリーパーティー文化の形成
1980s: 電子音楽とサイケデリック文化の融合
1990s: Goa Trance として確立
2000s: Psytrance へと進化、世界的に拡散
2010s: メインストリームフェスへの進出（Vini Vici等）
2020s: ハイブリッドサウンドの発展

文化的特徴:
- サイケデリックアート（フラクタル、蛍光色）
- スピリチュアルな要素
- アウトドアフェスティバル文化
- デコレーション（UV装飾、マンダラ）
- コミュニティ重視の非商業的精神
```

**Psytrance の主要サブカテゴリー詳細:**

**1. Full-On Psytrance:**
```
BPM: 138-145
特徴:
- メロディックで明るいサウンド
- 壮大なビルドアップ
- 複雑なシンセレイヤー
- 踊りやすいグルーヴ

代表的アーティスト:
- Astrix（イスラエル）: Deep Jungle Walk, Type 1

  → メインストリーム成功、Dimitri Vegas & Like Mikeとのコラボ
- Coming Soon!!!（イスラエル）: Drugs
  → キャッチーなメロディライン
- Captain Hook（南アフリカ）: Bungee Jump
  → プログレッシブとFull-Onのクロスオーバー

代表的レーベル:
- HOMmega Productions（イスラエル）
```

### Progressive Trance 詳細

**Progressive Trance vs Progressive House の違い:**

```
Progressive Trance:
- BPM: 128-134
- メロディが前面に出る
- トランス的なブレイクダウン
- シンセリードが特徴的
- 感情的な展開
- レーベル: Anjunadeep, Colorize

Progressive House:
- BPM: 120-128
- グルーヴ重視
- ミニマルな展開
- テクスチャー中心
- 深いベースライン
- レーベル: Sudbeat, Hernan's Picks

クロスオーバーゾーン（128-130 BPM）:
- 両方の要素を持つ
- Guy J, Yotto が代表
- DJセットで両方のジャンルを行き来可能
```

**Progressive Trance の選曲戦略:**

```
ジャーニー構築の原則:

1. グルーヴの統一性
   - ベースラインのパターンを揃える
   - キックの質感を合わせる
   - 16小節以上のロングミックス

2. テクスチャーの変化
   - パッドの質感を徐々に変える
   - アンビエント要素の増減
   - 空間の広がりをコントロール

3. エモーションの波
   - 3-4曲ごとにピークを作る
   - ブレイクダウンの配置を計画
   - 感情の起伏を設計

4. ハーモニックフロー
   - 同一キーを3-4曲続ける
   - 5度上/下への移動を活用
   - 急激なキー変化を避ける
```

### Tech Trance 詳細

**Tech Trance の音楽的DNA:**

```
Techno からの影響:
- ミニマルな反復
- 硬質なキック
- 工業的なサウンドデザイン
- グルーヴの深さ

Trance からの影響:
- メロディ要素
- ビルドアップ構造
- エモーショナルな瞬間
- 高BPM（135-140）

Tech Trance = Techno の骨格 + Trance の魂

サウンドデザインの特徴:
1. ディストーションベース
   - 歪んだ低音
   - アシッドライン（TB-303風）

2. インダストリアルなパーカッション
   - 金属的なハイハット
   - ノイズ系スネア

3. テンションビルダー
   - ライザー（上昇音）
   - ダウンリフター（下降音）
   - FX スウィープ
```

**代表的レーベル詳細:**

## 14. ロングミックスの高等テクニック

### ハーモニックミキシングの実践

**Camelot Wheel（カメロットホイール）を使った選曲:**

```
トランスで特に有効な移動パターン:

1. 同一キー（Same Key）
   例: 8A → 8A
   効果: 完全にハーモニーが一致
   使用場面: セットの安定区間

2. 隣接キー（Adjacent Key）
   例: 8A → 9A（5度上）
   効果: 自然な上昇感
   使用場面: ビルドアップ区間

3. パラレルキー（Major/Minor Switch）
   例: 8A → 8B（Am → C）
   効果: 明暗の変化（暗→明）
   使用場面: ブレイクダウン後のドロップ

4. エナジーブースト（+2 Steps）
   例: 8A → 10A
   効果: 明確なキー変化による高揚感
   使用場面: ピークタイムへの突入

5. ドラマチックシフト（Relative Minor/Major）
   例: 8A → 5B
   効果: 感情的な変化
   使用場面: セット中盤の転換点

避けるべき組み合わせ:
× 半音違い（8A → 8♯A）= 不協和音
× 3度以上離れた無関係なキー
```

### ダブルドロップテクニック

**2曲のクライマックスを重ねる高等技:**

```
前提条件:
- 2曲が同じキーまたは完全5度の関係
- BPMが同じ（±1以内）
- ビートグリッドが完全に合っている

手順:

Track A: ビルドアップ中
Track B: ビルドアップ中（Track Aの少し前をプレイ）

Step 1: Track A のブレイクダウン開始
        Track B もブレイクダウンに合わせる
        → 2曲のブレイクダウンが同時進行

Step 2: Track A の LOW をカット
        Track B の LOW を残す
        → ベースはTrack Bから

Step 3: 両方のビルドアップが進行
        → テンションが2倍に

Step 4: ドロップ！
        Track A: メロディ+HI
        Track B: ベース+キック
        → 壮大なダブルドロップ

Step 5: 8-16小節後
        Track A をフェードアウト
        Track B が単独に

注意点:
- 事前リハーサル必須
- 位相の確認（打ち消しに注意）
- EQでの周波数帯の棲み分けが重要
- 失敗リスクが高いので練習を積むこと
```

### ブレイクダウンの重ね合わせ

**感情的なトランジションの極意:**

```
テクニック: ブレイクダウン・レイヤリング

Track A: 通常再生中（メインセクション）
Track B: ブレイクダウンのボーカル/メロディ部分をキュー

手順:
1. Track A のアウトロ開始時
2. Track B をブレイクダウンの先頭からスタート
   - HI: 9時, MID: 10時, LOW: 9時
3. Track A の音量を下げつつ
4. Track B のボーカル/メロディだけを聴かせる
   - HI: 11時, MID: 12時に徐々に
5. Track B のビルドアップが始まったら
6. Track A を完全フェードアウト
7. Track B のドロップで新曲スタート

効果:
- 感情的な瞬間を演出
- 観客の注目を集める
- ストーリーテリング性が向上
- プロフェッショナルな印象
```

### 演習5: エフェクト実践ドリル

```
課題: DDJ-FLX4 のエフェクトを使った演出を練習

ドリル1: Echo ビルドアップ（5分間反復練習）
──────────────────────────────────
1. 任意のトランス曲を再生
2. ブレイクダウン8小節前で Echo ON
3. BEAT = 1/2
4. LEVEL を 10時 → 2時 に徐々に上げる
5. ブレイクダウン突入で Echo OFF
6. 繰り返し × 10回

ドリル2: Filter トランジション（10分間反復練習）
──────────────────────────────────
1. Track A を再生中
2. Track B をビートマッチ
3. Track B の SMART CFX を右いっぱい（HPF）
4. Track B のフェーダーを上げる
5. SMART CFX を徐々にセンターへ戻す
6. 同時に Track A の SMART CFX を左に回す（LPF）
7. Track A をフェードアウト
8. 繰り返し × 10回

ドリル3: Reverb ブレイクダウン演出（5分間反復練習）
──────────────────────────────────
1. ボーカルトランス曲を再生
2. ブレイクダウンのボーカル部分で Reverb ON
3. BEAT = 2/1（2小節）
4. LEVEL = 12時
5. ボーカルに空間的な広がりを追加
6. ビルドアップ開始で Reverb OFF
7. 繰り返し × 10回

目標:
- 各ドリルを無意識にできるまで反復
- エフェクトのON/OFFタイミングを体に覚えさせる
- 曲の構造を聴きながら自然にエフェクトを操作
```

## 17. トランスの未来とトレンド

### 現在のトレンド（2020年代）

```
1. ジャンルの融合
   - Trance × Techno（テックトランスの進化）
   - Trance × Progressive House（ボーダーレス化）
   - Trance × Psytrance（メインストリームPsy）
   - Trance × Melodic Techno（Anyma 等の影響）

2. テクノロジーの活用
   - AI によるマスタリング（LANDR等）
   - ステムプレイ（Native Instruments Traktor）
   - クラウドベースの楽曲管理
   - VR/AR ライブイベント

3. クラシックリバイバル
   - 90年代/00年代サウンドの復権
   - リマスターリリース
   - クラシックDJセットの人気
   - レトロフューチャリスティックな新曲

4. 新しいマーケット
   - アジア（中国、インド）での人気拡大
   - 南米でのトランスフェスティバル増加
   - 中東でのシーン成長
   - オンラインフェスティバルの定着

5. サステナビリティ
   - エコフレンドリーなフェスティバル運営
   - デジタルチケット
   - カーボンオフセット
   - 地域コミュニティとの共存
```

### トランスDJとしてのキャリアパス

```
ステップ1: 基礎固め（0-6ヶ月）
├─ 週3回以上のミックス練習
├─ 500曲以上のライブラリ構築
├─ 30分セットを完璧にプレイ
└─ オンラインでミックスを公開

ステップ2: コミュニティ参加（6-12ヶ月）
├─ ローカルトランスイベントに参加
├─ オンラインコミュニティで活動
├─ 60分セットの録音・公開
└─ 他のDJとのネットワーキング

ステップ3: ギグ獲得（1-2年）
├─ オープンデッキイベントに参加
├─ バーやクラブでのレジデント
├─ 90分セットの安定的なプレイ
└─ 自分のイベント企画

ステップ4: プロダクション開始（2-3年）
├─ DAW（Ableton/FL Studio）の習得
├─ オリジナルトラック制作
├─ レーベルへのデモ送付
└─ リミックス制作

ステップ5: プロフェッショナル（3年以上）
├─ レーベルからのリリース
├─ フェスティバルへの出演
├─ 国際的な活動
└─ 自身のブランド確立
```
