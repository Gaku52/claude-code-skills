## なぜ House が重要か

**House = 電子音楽の基礎:**
```
事実:
- 世界で最も人気のジャンル
- クラブギグの70%が House
- DJ人口の50%が House DJ
- 学びやすく、奥が深い

House をマスター = DJ の基礎
```

## 1. House の歴史

### 起源（1980年代シカゴ）

**1977-1984: 誕生**
```
場所: シカゴ

キーパーソン:
- Frankie Knuckles（"The Godfather of House"）
- Ron Hardy
- Larry Levan（ニューヨーク）

The Warehouse（クラブ）:
- Frankie Knuckles が DJ
- Disco、Soul、Funk をミックス
- ドラムマシン（Roland TR-808）追加
- "House Music" の名前の由来

1984年: 最初の House 曲
「On and On」by Jesse Saunders
```

### ゴールデンエイジ（1985-1995）

**シカゴからヨーロッパへ:**
```
1985-1987: シカゴ House 確立
- Trax Records、DJ International
- Marshall Jefferson「Move Your Body」
- Fingers Inc.「Can You Feel It」

1987-1990: UK、ヨーロッパへ
- Acid House ブーム
- Summer of Love（1988）
- イビサ島が中心地に

1990-1995: グローバル化
- Deep House、Tech House登場
- ニューヨーク、デトロイト、ベルリン
- 世界中に拡散
```

### 現代（1995-現在）

**進化と多様化:**
```
1995-2005: サブジャンル爆発
- Progressive House
- Tribal House
- Electro House

2005-2015: EDM ブーム
- David Guetta、Calvin Harris
- メインストリーム化
- フェスティバル中心

2015-現在: アンダーグラウンド回帰
- Tech House 人気（Carl Cox、Jamie Jones）
- Deep House（Black Coffee、Kerri Chandler）
- Minimal House

House は常に進化
```

## 2. House の特徴

### 音楽的特徴

**基本構造:**
```
BPM: 120-130（標準は 125）

リズム:
- 4つ打ち（Four on the floor）
- キックが4分音符で連打
- ドン・ドン・ドン・ドン

楽器:
- ドラムマシン（Roland TR-808、909）
- ベースライン（深い、グルーヴィー）
- ハイハット（オープン/クローズ）
- クラップ/スネア

ボーカル:
- ソウルフル（Deep House）
- サンプリング（Tech House）
- アカペラ

雰囲気:
- グルーヴ重視
- ダンサブル
- ポジティブ
```

## 5. DJ技法とミキシング

### House のミキシング原則

**ロングミックス:**
```
特徴:
- 1-2分のミックス
- スムーズな移行
- グルーヴを壊さない

手順:
1. 次の曲をキュー（イントロから）
2. ビートマッチング
3. ゆっくりフェードイン（32-64ビート）
4. EQ調整（低音から）
5. 両曲が混ざる（32ビート）
6. ゆっくりフェードアウト

合計: 1-2分

House = ロングミックスの芸術
```

### EQ の使い方

**3バンド EQ:**
```
LOW（ベース）:
- 最も重要
- 両曲のベースは被らない
- 片方を完全にカット
- 徐々に切り替え

MID（ボーカル、メロディ）:
- 少し下げる程度
- 完全カットしない
- バランス調整

HIGH（ハイハット、シンバル）:
- 微調整
- 明るさを調整

EQ = House ミックスの鍵
```

### トランジション技法

**Echo Out:**
```
使用タイミング:
- ピークから次の曲へ
- エネルギーを保つ

手順:
1. 現在の曲にEcho（DDJ-FLX4のBeat FX）
2. Echo時間を伸ばす
3. 次の曲をミックスイン
4. Echoを切る

効果: スムーズで音楽的
```

**Filter Sweep:**
```
使用タイミング:
- ビルドアップからドロップ
- エネルギー変化

手順:
1. 現在の曲にLow Pass Filter
2. 徐々にフィルター閉じる
3. 次の曲をミックスイン
4. フィルター開く

効果: ダイナミック
```

## 7. セット構築

### 60分 Tech House セット例

**フェーズ1: ウォームアップ（0-15分）**
```
BPM: 122-125
エネルギー: 3-4 / 10

選曲:
- グルーヴィーだが落ち着いている
- ボーカル少なめ
- ベースライン重視

例:
1. Patrick Topping「Voicemail」122 BPM
2. Solardo「Tribesmen」124 BPM
3. CamelPhat「Breathe」125 BPM
```

**フェーズ2: エネルギー構築（15-35分）**
```
BPM: 125-128
エネルギー: 5-7 / 10

選曲:
- より Energy 高い
- パーカッシブ
- ボーカルサンプル

例:
4. Fisher「Crowd Control」126 BPM
5. Chris Lake「Turn Off the Lights」127 BPM
6. Jamie Jones「Hungry for the Power」128 BPM
7. Green Velvet「La La Land」128 BPM
```

**フェーズ3: ピーク（35-50分）**
```
BPM: 128-130
エネルギー: 8-9 / 10

選曲:
- 最もエネルギッシュ
- ドロップあり
- 観客が知っている曲

例:
8. Fisher「Losing It」128 BPM - 観客大喜び
9. Chris Lake「Operator」129 BPM
10. Carl Cox「I Want You」130 BPM
```

**フェーズ4: クールダウン（50-60分）**
```
BPM: 128-126
エネルギー: 6-7 / 10

選曲:
- 少しエネルギー下げる
- グルーヴ重視
- 記憶に残る曲

例:
11. Solardo「XTC」127 BPM
12. Patrick Topping「Be Sharp」126 BPM
13. CamelPhat & Elderbrook「Cola」126 BPM

最後は印象的な曲で締める
```

## 10. ハウスミュージックの詳細な歴史

### 10.1 プレハウス時代（1970年代）: ディスコからの系譜

ハウスミュージックを理解するためには、その直接的な先祖であるディスコ文化を知る必要がある。1970年代のニューヨークを中心に花開いたディスコは、ハウスの音楽的・文化的基盤を形成した。

**ディスコの黄金期と崩壊:**
```
1970年代前半:
- David Mancuso の「The Loft」が原点
- Nicky Siano の「The Gallery」
- フィラデルフィアソウルの影響
- O'Jays, MFSB, Salsoul Orchestra

1977年: ディスコ最盛期
- Studio 54 オープン
- Saturday Night Fever 公開
- Bee Gees「Stayin' Alive」が世界的ヒット
- ディスコ産業は40億ドル規模に

1979年7月12日: 「Disco Demolition Night」
- シカゴ・コミスキーパーク
- ラジオDJ Steve Dahl が主導
- ディスコレコード破壊イベント
- 人種差別・同性愛嫌悪の側面も
- ディスコバブル崩壊の象徴的事件

1980-1982: ポストディスコ
- ディスコは「死んだ」と宣言される
- しかしアンダーグラウンドでは進化継続
- Larry Levan の Paradise Garage（NY）
- Frankie Knuckles の The Warehouse（Chicago）
- ディスコの精神がハウスへ受け継がれる
```

**Paradise Garage とLarry Levan:**
```
場所: ニューヨーク、84 King Street
期間: 1977-1987
DJ: Larry Levan（レジデント）

特徴:
- 世界最高峰のサウンドシステム（Richard Long設計）
- 金曜土曜のオールナイトパーティー
- ゲイコミュニティが中心だが多様な客層
- 「DJ = アーティスト」という概念の確立
- アルコール販売なし（ジュースバー）

Larry Levan のDJスタイル:
- ジャンルを超えた選曲（ディスコ、ファンク、ロック、レゲエ）
- 曲をリアルタイムで再構築（EQ、エフェクト駆使）
- 3日間の準備で1晩のプレイ
- 「ミキシング」より「プログラミング」と呼んだ
- 観客との一体感を最重視

影響:
- ハウスミュージックの精神的原型
- 「ガラージ」サブジャンルの名前の由来
- 世界中のクラブカルチャーに影響
```

### 10.2 シカゴハウスの誕生（1982-1987）

**The Warehouse と Frankie Knuckles:**
```
The Warehouse:
- 場所: シカゴ、206 South Jefferson Street
- オープン: 1977年
- 主にアフリカ系アメリカ人ゲイコミュニティ
- Frankie Knuckles がNYから招聘される

Frankie Knuckles の革新:
- ディスコレコードにドラムマシンを重ねる
- Roland TR-808、TR-909 を使用
- テープ編集でロングミックスを作成
- Erasmo Riviera と共にリエディット制作
- 「ポストディスコ」を「ハウス」へ変容させた

名前の由来:
- 「The Warehouse で流れる音楽」
- レコード店で「As played at The Warehouse」と表記
- 短縮されて「House Music」に
- 1982年頃からこの呼称が定着
```

**Ron Hardy と The Music Box:**
```
The Music Box:
- 場所: シカゴ、326 North Michigan Avenue
- 期間: 1983-1987
- Frankie Knuckles と双璧をなす存在

Ron Hardy のスタイル:
- Frankie よりアグレッシブ
- ドラッグの影響を受けた荒々しいプレイ
- 新人プロデューサーの楽曲をテスト
- テープレコーダーでリアルタイムエディット
- 「Music Box ミックス」が多数のアーティストを輩出

Music Box から生まれたアーティスト:
- Phuture（Acid House の創始者）
- Adonis
- Farley "Jackmaster" Funk
- Chip E.
- Steve "Silk" Hurley
```

**シカゴハウスの重要レーベル（1984-1990）:**
```
Trax Records:
- 設立: 1984年、Larry Sherman
- ハウスミュージック最初の専門レーベル
- 代表作: Frankie Knuckles「Your Love」
- Phuture「Acid Tracks」
- Marshall Jefferson「Move Your Body」
- アーティストへの支払い問題で悪名高い

DJ International:
- 設立: 1985年、Rocky Jones
- Trax と並ぶ二大レーベル
- 代表作: Chip E.「Time to Jack」
- Fast Eddie「Acid Thunder」
- より商業的な成功を追求

Dance Mania:
- 設立: 1985年、Jesse Saunders & Ray Barney
- ゲットーハウス、ジュークの聖地
- DJ Deeon、DJ Funk、Parris Mitchell
- シカゴの生々しいストリート感覚
- 2012年に再始動

Relief Records:
- 設立: 1991年、Green Velvet (Cajmere)
- よりハードなシカゴハウス
- Green Velvet「Flash」「Percolator」
```

### 10.3 Acid House革命（1987-1989）

**Acid Houseの誕生:**
```
1987年: Phuture「Acid Tracks」
- DJ Pierre, Spanky, Herb J の3人組
- Roland TB-303 ベースシンセの偶然の発見
- ツマミをランダムに回した結果の独特なサウンド
- Ron Hardy が Music Box で初プレイ
- 「Acid」の名は LSD ではなく TB-303 の音色から

TB-303 の革命:
- 元々はベースギターの代用品として開発
- 商業的失敗で投げ売り状態（50-100ドル）
- シカゴのプロデューサーが安価で入手
- 意図されていない使い方が新サウンドを生む
- レゾナンスとカットオフの操作が核心

主要 Acid House トラック:
- Phuture「Acid Tracks」(1987)
- Sleezy D「I've Lost Control」(1986)
- Adonis「No Way Back」(1986)
- Armando「Land of Confusion」(1988)
- Fast Eddie「Acid Thunder」(1988)
```

## 13. ハウスミュージック プロダクションの基礎

DJ だけでなく、プロデューサーとしてのスキルを持つことで、ハウスミュージックへの理解が深まる。

### 13.1 必要な機材とソフトウェア

**DAW（Digital Audio Workstation）**
```
推奨DAW:
1. Ableton Live（ハウス界の標準）
   - 直感的なワークフロー
   - ライブパフォーマンス機能
   - 豊富なテンプレート
   - 価格: $99-$749

2. FL Studio（初心者に優しい）
   - ステップシーケンサー
   - ビートメイキングに最適
   - 価格: $99-$499

3. Logic Pro（Mac限定、コスパ最高）
   - プロフェッショナル機能
   - 膨大な音源付属
   - 価格: $199（買い切り）

4. Native Instruments Maschine（ハードウェア統合）
   - グルーヴボックス的ワークフロー
   - ハウス向けライブラリ豊富
```

```
ドラムマシン:

ベースシンセ:
```

### 13.2 ハウストラックの基本構造

**標準的なアレンジメント（128小節 = 約4分）**
```
セクション1: イントロ（0-32小節）
- キック
- ハイハット
- パーカッション徐々に追加
- ベースヒント（サブベース）
- 目的: DJがミックスしやすいように

セクション2: ビルドアップ1（32-48小節）
- メインメロディ導入
- ボーカルサンプル（あれば）
- フィルター開く
- エネルギー上昇

セクション3: ドロップ1（48-64小節）
- フルアレンジメント
- ベースライン完全
- すべての要素が揃う
- 最高エネルギー

セクション4: ブレイクダウン（64-80小節）
- キック抜き
- メロディック要素残す
- リバーブ/ディレイ増加
- 緊張感を作る

セクション5: ビルドアップ2（80-88小節）
- ドロップに向けた上昇
- フィルタースイープ
- FXライザー
- エネルギー最大化

セクション6: ドロップ2（88-112小節）
- 再度フルアレンジメント
- 新しい要素追加（変化）
- トラックのクライマックス

セクション7: アウトロ（112-128小節）
- 徐々に要素削除
- ベース、メロディ抜く
- キック、ハイハット、パーカッションのみ
- 目的: 次のDJへスムーズに渡す
```

### 13.3 各楽器の作り方

**キックドラム（ハウスの心臓）**
```
サウンド特性:
- 周波数: 40-60Hz（サブベース）
- パンチ: 100-200Hz（ボディ）
- クリック: 2-5kHz（アタック）

作成手順:
1. 808/909サンプル選択
2. EQ でローカット（30Hz以下）
3. コンプレッサーでパンチ追加
4. サチュレーションで倍音付加
5. サイドチェーンコンプのキー信号に設定

Tech House キック:
- より短いディケイ
- パンチ強調
- 909系サウンド

Deep House キック:
- 長いディケイ
- 温かみのある808系
- サブベース重視
```

**ハイハット**
```
オープンハイハット:
- 裏拍に配置（8分音符）
- ベロシティ変化でグルーヴ
- リバーブ少量
- 周波数: 8-12kHz

クローズドハイハット:
- 16分音符パターン
- ベロシティランダム化（人間味）
- ハイパスフィルター 5kHz

シェイカー:
- 16分音符裏拍
- ステレオワイド化
- グルーヴの接着剤
```

**クラップ/スネア**
```
配置: 2拍目と4拍目
サウンド:
- 909クラップ（Tech House）
- 生スネア（Deep House）
- 重ねる: クラップ + スネア

処理:
- リバーブ（短めのルーム）
- EQ で 200Hz カット（キックとの分離）
- コンプレッション軽め
```

**ベースライン**
```
Deep House ベース:
- サブベース（サイン波 40-80Hz）
- 長い音符（全音符〜2分音符）
- フィルターエンベロープでうねり
- サイドチェーンでキックに譲る

Tech House ベース:
- よりパーカッシブ
- 8分音符〜16分音符
- フィルター変化でリズム
- グライド（ポルタメント）使用

Acid ベース:
- TB-303エミュレーター（ABL3, Phoscyon）
- レゾナンス高め
- カットオフをオートメーション
- アクセントパターン重要
```

**ピアノ/コード**
```
Deep House ピアノ:
- Rhodes、Wurlitzer
- ジャズコード（7th, 9th, 13th）
- リバーブたっぷり
- ベロシティ変化で人間味

コード進行例:
- Ⅱm7 - Ⅴ7 - Ⅰmaj7（ジャズ的）
- Ⅵm - Ⅳ - Ⅰ - Ⅴ（ポップ的）
- Ⅰ - Ⅵm - Ⅳ - Ⅴ（クラシック）

Tech House シンセ:
- シンプルなスタブ
- フィルター開閉でリズム
- サイドチェーンでパンピング
```

**ボーカル処理**
```
サンプル選択:
- Splice Sounds
- Loopcloud
- 自分で録音（Acapella）

処理チェーン:
1. ピッチ補正（Melodyne、Auto-Tune）
2. EQ（ローカット 100Hz、プレゼンス 3-5kHz ブースト）
3. コンプレッサー（4:1、ファストアタック）
4. ディエッサー（歯擦音削減）
5. リバーブ（Plate、2-3秒ディケイ）
6. ディレイ（1/8 or 1/4音符）

ボーカルチョップ:
- 1-2単語の短いフレーズ
- リズミックに配置
- ピッチシフト実験
- Tech House の定番技法
```

### 13.4 ミキシングとマスタリング

**ミキシングの基本原則**
```
周波数分離:
- キック: 40-100Hz
- ベース: 80-200Hz
- ボーカル: 200Hz-5kHz
- ハイハット/シンバル: 8kHz以上

各トラックに:
- ハイパスフィルター（不要な低域カット）
- コンプレッサー（ダイナミクス制御）
- EQ（周波数バランス調整）

ステレオイメージング:
- キック: モノラル（センター）
- ベース: モノラル（センター）
- パーカッション: ステレオワイド
- パッド/ストリングス: ステレオワイド
- ボーカル: センター（ディレイ/リバーブはステレオ）

サイドチェーンコンプレッション:
- キックをキーに、ベースとパッドを圧縮
- パンピング効果
- グルーヴ強化
```

**マスタリングの基礎**
```
マスターチェーン（順番重要）:
1. EQ（全体のトーナルバランス）
2. マルチバンドコンプ（周波数帯別の圧縮）
3. コンプレッサー（グルー、軽め）
4. イメージャー（ステレオワイド化）
5. リミッター（最終音量調整）

目標レベル:
- ピーク: -0.1dB（True Peak）
- ラウドネス: -8 to -6 LUFS（ストリーミング）
- ダイナミックレンジ: 6-8 DR

参照トラック:
- 同ジャンルのプロトラック
- A/B 比較
- スペクトラムアナライザーで周波数比較
```
