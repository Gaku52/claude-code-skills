# ベース・メロディ完全ガイド

**楽曲制作における音楽の核心を完全マスター**

このセクションでは、楽曲の「心臓部」となるベースライン、メロディ、コード進行、リードシンセ、パッドなど、音楽の核となる要素を完全にマスターします。ドラムがリズムの土台なら、ベース・メロディは楽曲の魂です。

---

## 📚 この章で学ぶこと

このセクションを完了すると、以下のスキルを習得できます：

### ベース関連
- **サブベース**: 20-60 Hzの超低域を正確にコントロール
- **ベースライン**: グルーヴを生み出す効果的なベースライン作成
- **ベース音色**: Wavetable/Operatorでの多彩なベース音色デザイン

### メロディ関連
- **メロディ作成**: 記憶に残るメロディの作曲技法
- **コード進行**: 定番進行からオリジナルまで
- **リードシンセ**: 楽曲の主役となるリード音色設計
- **パッド**: 空間を満たすアンビエントパッド

### 高度なテクニック
- **カウンターメロディ**: メインメロディを補完する対旋律
- **ハーモニー**: 音楽理論を実践に応用
- **レイヤリング**: 複数の音色を重ねて厚みを作る

## 🎯 なぜベース・メロディが重要なのか

### DJから制作者への視点

**DJとして**:
- ベースラインで**フロアを揺らす**経験をしてきた
- メロディで**観客の感情を動かす**瞬間を知っている
- キー（調）の違いで**雰囲気が変わる**ことを体感している

**プロデューサーとして**:
- その「魔法の瞬間」を**自分で作り出す**ことができる
- ベース・メロディは**楽曲の個性**を決定づける
- 音楽理論を理解することで**無限の可能性**が開ける

### 数字で見る重要性

| 要素 | 楽曲への影響 | プロデューサーの使用率 |
|------|------------|-------------------|
| **サブベース** | クラブでの体感、重低音 | 95%+ (EDM全般) |
| **ベースライン** | グルーヴ、ダンサビリティ | 100% |
| **メロディ** | 記憶に残る、口ずさめる | 80% (ジャンルによる) |
| **コード進行** | 感情、雰囲気 | 90%+ |
| **リードシンセ** | 楽曲の顔、フック | 70% (EDM) |
| **パッド** | 空間、深み | 85%+ |

### プロの制作時間配分

**楽曲制作20時間のうち**:
- ドラム・リズム: 4-5時間 (20-25%)
- **ベース: 3-4時間 (15-20%)**
- **メロディ・コード: 4-5時間 (20-25%)**
- ミキシング: 4-5時間 (20-25%)
- アレンジメント: 3-4時間 (15-20%)

> **ベース・メロディ合計で約40-45%**の時間を費やす

## 🎵 音楽理論の基礎（必須知識）

### 音階とキー

```
メジャースケール (明るい):
C Major: C - D - E - F - G - A - B - C
        全  全 半  全  全  全 半

マイナースケール (暗い):
A Minor: A - B - C - D - E - F - G - A
        全 半  全  全 半  全  全
```

### 相対キー（Relative Key）

**メジャーとマイナーは兄弟関係**:
- C Major = A Minor (同じ音を使用)
- F Major = D Minor
- G Major = E Minor

> DJのハーモニックミキシングと同じ概念

### 音程（Interval）

```
C からの音程:
C → C: 完全1度 (Root)
C → E: 長3度 (Major 3rd) - 明るい
C → E♭: 短3度 (Minor 3rd) - 暗い
C → G: 完全5度 (Perfect 5th) - 安定
C → B♭: 短7度 (Minor 7th) - テンション
```

### 基本コード

```
メジャートライアド (明るい):
C Major: C - E - G (Root, M3, P5)

マイナートライアド (暗い):
C Minor: C - E♭ - G (Root, m3, P5)

7thコード (ジャズ的):
C Major 7: C - E - G - B
C Minor 7: C - E♭ - G - B♭
```

### モード（教会旋法）の理解

モードはスケールのバリエーションで、EDM制作でも頻繁に使用されます。各モードはCメジャースケールの音を出発点を変えて弾くことで得られます。

```
イオニアン (Ionian) = メジャースケール:
C - D - E - F - G - A - B - C
雰囲気: 明るい、ハッピー
使用ジャンル: House、Progressive House

ドリアン (Dorian):
D - E - F - G - A - B - C - D
雰囲気: マイナーだが明るめ、ファンキー
使用ジャンル: Tech House、Funk、Deep House
特徴: ナチュラルマイナーの6度が長6度

フリジアン (Phrygian):
E - F - G - A - B - C - D - E
雰囲気: ダーク、エキゾチック、スパニッシュ
使用ジャンル: Psytrance、Dark Techno
特徴: 2度がフラット（半音の緊張感）

リディアン (Lydian):
F - G - A - B - C - D - E - F
雰囲気: 浮遊感、ドリーミー、神秘的
使用ジャンル: Ambient、Progressive Trance
特徴: 4度がシャープ（#4）

ミクソリディアン (Mixolydian):
G - A - B - C - D - E - F - G
雰囲気: メジャーだがブルージー
使用ジャンル: Garage、UK Funky
特徴: 7度がフラット（b7）

エオリアン (Aeolian) = ナチュラルマイナースケール:
A - B - C - D - E - F - G - A
雰囲気: 悲しい、メランコリック
使用ジャンル: Melodic Techno、Trance

ロクリアン (Locrian):
B - C - D - E - F - G - A - B
雰囲気: 不安定、不気味
使用ジャンル: Experimental、Industrial
特徴: 5度がフラット（非常に不安定）
```

### モードのベースライン制作への応用

```
ドリアンモードのベースライン例（D Dorian）:
|D2----|A1----|G1----|F1-B1-|
→ 6度の音（B）がマイナーキーに明るさを加える
→ Deep House、Tech Houseで頻繁に使用

フリジアンモードのベースライン例（E Phrygian）:
|E1----|F1----|E1----|F1-G1-|
→ ルートと短2度（E-F）の半音の緊張感
→ Dark Techno、Psytranceの定番

ミクソリディアンモードのベースライン例（G Mixolydian）:
|G1----|F1----|C2----|D2----|
→ b7（F）がブルージーな雰囲気
→ ファンキーなHouse、Garage向き
```

### ハーモニックマイナースケール

ナチュラルマイナーの7度を半音上げたスケール。Tranceやメロディック系ジャンルで非常に重要です。

```
A ハーモニックマイナー:
A - B - C - D - E - F - G# - A
        全 半 全 全 半 増2 半

特徴:
- G → G# に変化（導音を作る）
- F → G# 間が「増2度」（エキゾチックな響き）
- V7コード（E7: E-G#-B-D）が使用可能に

ベースライン応用:
|Am----|E7----|Am----|Dm-E7-|
→ クラシカルでドラマチックな響き
→ Uplifting Trance、Euro Tranceの定番

全キーのハーモニックマイナー:
A HMin: A  - B  - C  - D  - E  - F  - G# - A
E HMin: E  - F# - G  - A  - B  - C  - D# - E
B HMin: B  - C# - D  - E  - F# - G  - A# - B
D HMin: D  - E  - F  - G  - A  - Bb - C# - D
G HMin: G  - A  - Bb - C  - D  - Eb - F# - G
C HMin: C  - D  - Eb - F  - G  - Ab - B  - C
```

### メロディックマイナースケール

上行時に6度と7度を半音上げるスケール。ジャズやフュージョン系EDMで使用されます。

```
A メロディックマイナー (上行):
A - B - C - D - E - F# - G# - A
→ 6度と7度がシャープ

A メロディックマイナー (下行 = ナチュラルマイナー):
A - G - F - E - D - C - B - A
→ 下行時はナチュラルマイナーに戻す（クラシック理論）
→ EDM制作では上行形のみを使うことが多い

特徴:
- ナチュラルマイナーより明るい
- メジャーとマイナーの中間的な雰囲気
- Melodic Techno、Progressive Houseで効果的
```

### 音程の完全ガイド

ベースラインとメロディの両方で、音程の理解は不可欠です。

```
完全系:
完全1度 (P1/Unison):  0半音  → 同じ音
完全4度 (P4):          5半音  → 安定、浮遊感
完全5度 (P5):          7半音  → 最も安定（パワーコード）
完全8度 (P8/Octave):  12半音  → オクターブ

長音程:
長2度 (M2):  2半音  → 全音、メロディックな動き
長3度 (M3):  4半音  → メジャー感、明るい
長6度 (M6):  9半音  → 甘い、ロマンチック
長7度 (M7): 11半音  → テンション、ジャズ的

短音程:
短2度 (m2):  1半音  → 緊張、不協和
短3度 (m3):  3半音  → マイナー感、暗い
短6度 (m6):  8半音  → ドラマチック
短7度 (m7): 10半音  → ブルージー、ファンキー

増減:
増4度 (Aug4)/減5度 (Dim5): 6半音 → トライトーン（悪魔の音程）
→ 最も不安定、テンションの源
→ ドミナント7thコードに含まれる
```

### 音程のベースライン・メロディへの活用

```
ベースラインでよく使う音程:
- 完全1度（ルート）: 安定感、繰り返し
- 完全5度: パワフル、Rock/Metal/EDM全般
- 完全8度（オクターブ）: エネルギー、Techno
- 短3度/長3度: コード感、House/Garage
- 短7度: ファンキー、Tech House

メロディでよく使う音程:
- 長2度（全音）: スムーズな上行/下行
- 短2度（半音）: テンション、解決
- 長3度: 跳躍、明るい
- 完全4度: 力強い跳躍
- 完全5度: ドラマチックな跳躍
- 完全8度: クライマックス感
```

### ダイアトニックコード完全ガイド

スケール上の各音をルートとして構築されるコード群が「ダイアトニックコード」です。

```
Cメジャースケールのダイアトニックコード:

度数    コード名    構成音         機能          響き
I       C Major    C - E - G      トニック      安定、始まり
ii      D minor    D - F - A      サブドミナント 柔らかい、移行
iii     E minor    E - G - B      トニック代理   繊細、内省的
IV      F Major    F - A - C      サブドミナント 開放感、浮遊
V       G Major    G - B - D      ドミナント    緊張、解決要求
vi      A minor    A - C - E      トニック代理   哀愁、切ない
vii°    B dim      B - D - F      ドミナント代理 不安定、テンション

7thコードに拡張:
I7      Cmaj7      C - E - G - B      ジャジー、洗練
ii7     Dm7        D - F - A - C      ソウルフル
iii7    Em7        E - G - B - D      浮遊感
IV7     Fmaj7      F - A - C - E      ドリーミー
V7      G7         G - B - D - F      強い解決要求
vi7     Am7        A - C - E - G      メランコリック
vii°7   Bm7b5      B - D - F - A      テンション
```

```
Aマイナースケール（ナチュラル）のダイアトニックコード:

度数    コード名    構成音         機能          響き
i       A minor    A - C - E      トニック      暗い安定
ii°     B dim      B - D - F      サブドミナント 不安定
III     C Major    C - E - G      トニック代理   相対メジャー
iv      D minor    D - F - A      サブドミナント 哀愁
v       E minor    E - G - B      ドミナント    弱い解決
VI      F Major    F - A - C      サブドミナント 明るさ
VII     G Major    G - B - D      サブトニック   浮遊感

※ ハーモニックマイナーではV = E Major (E-G#-B) となり
   強いドミナント機能を持つ
```

## 🎹 Ableton Live での実践

### 主要音源の使い分け

| 音源 | 最適用途 | 使用頻度 |
|------|---------|---------|
| **Wavetable** | サブベース、Technoベース、リード | 70% |
| **Operator** | FMベース、ブラス、ベル | 20% |
| **Analog** | アナログベース、ビンテージパッド | 10% |
| **Sampler** | 808、サンプルベース | 30% |

### ベース・メロディ用テンプレート

```
トラック構成例（Techno）:

1. Sub Bass (Wavetable)
   - 20-60 Hz純粋なサイン波
   - Mono必須

2. Mid Bass (Wavetable)
   - 60-250 Hz
   - フィルターモジュレーション

3. Lead Synth (Wavetable)
   - 500 Hz - 4 kHz
   - ステレオ幅60-80%

4. Pad (Analog)
   - 200 Hz - 8 kHz
   - ステレオ幅80-100%
   - Reverb Send 30-40%

Return Tracks:
A. Reverb (Hall, 2.5s)
B. Delay (1/8 Dotted)
C. Chorus (Pads用)
```

### Wavetableでのベース音色設計（詳細手順）

Wavetableはベース制作の中核となる音源です。以下に各種ベースの作成手順を詳細に解説します。

#### サブベース（Pure Sub）

```
Step 1: Oscillator 1 設定
  - Wavetable: Basic Shapes
  - Position: 0% (Pure Sine)
  - Voices: 1 (Mono)
  - Detune: 0%

Step 2: Oscillator 2
  - OFF（サブベースは単一オシレーター）

Step 3: Filter
  - Type: Low Pass 12dB (LP12)
  - Cutoff: 80-120 Hz
  - Resonance: 0%
  - Drive: 0%

Step 4: Amp Envelope
  - Attack: 5-10 ms（クリックを避ける）
  - Decay: 0 ms
  - Sustain: 100%
  - Release: 50-100 ms（テールを短めに）

Step 5: Post-Processing
  - Utility: Width 0%（完全Mono）
  - Utility: Bass Mono ON (120 Hz)
  - Saturator: OFF（クリーンなサブ）
  - EQ: High Cut 100 Hz（余分な倍音を除去）

注意点:
- サブベースは必ずMonoにする
- ステレオ情報があるとフェーズキャンセルが起きる
- クラブのサブウーファーはモノ再生が多い
```

#### Technoベース（Driving Bass）

```
Step 1: Oscillator 1 設定
  - Wavetable: Basic Shapes
  - Position: 20-40% (Saw寄り)
  - Voices: 1
  - Detune: 0%

Step 2: Oscillator 2
  - Wavetable: Basic Shapes
  - Position: 0% (Sine)
  - Level: -6 dB（サブ補強用）
  - Tune: -12 (1オクターブ下)

Step 3: Filter
  - Type: Low Pass 24dB (LP24)
  - Cutoff: 200-500 Hz
  - Resonance: 20-40%
  - Drive: 10-30%
  - Key Tracking: 50-70%

Step 4: Filter Envelope
  - Attack: 0 ms
  - Decay: 200-500 ms
  - Sustain: 20-40%
  - Release: 100-200 ms
  - Amount: +40-60%

Step 5: LFO 1 → Filter Cutoff
  - Shape: Sine or Triangle
  - Rate: 1/4 or 1/8 (sync)
  - Amount: 20-40%
  - → フィルターが周期的に開閉 = 動きが生まれる

Step 6: Post-Processing
  - Saturator: Soft Clip, Drive 3-6 dB
  - EQ Eight: Low Cut 30 Hz, High Cut 5 kHz
  - Compressor: Ratio 4:1, Attack 10ms, Release 50ms
  - Utility: Width 30-50%
```

#### Acid Bass（TB-303スタイル）

```
Step 1: Oscillator 1 設定
  - Wavetable: Basic Shapes
  - Position: 100% (Pure Saw)
  - Voices: 1
  - Detune: 0%

Step 2: Filter（最重要）
  - Type: Low Pass 24dB (LP24) ← 必ず24dB
  - Cutoff: 300-800 Hz
  - Resonance: 60-80% ← 高めのレゾナンス！
  - Drive: 20-40%
  - Key Tracking: 100%

Step 3: Filter Envelope
  - Attack: 0 ms
  - Decay: 100-300 ms ← アクセントの鍵
  - Sustain: 0-10%
  - Release: 50-100 ms
  - Amount: +70-100% ← 大きくかける

Step 4: アクセントの再現
  - ベロシティ → Filter Cutoff: 50-70%
  - ベロシティ → Filter Envelope Amount: 30-50%
  - → 強く弾いた音だけフィルターが大きく開く

Step 5: スライド/グライドの再現
  - Glide: ON
  - Glide Time: 50-100 ms
  - Glide Mode: Legato
  - → 音と音がつながるようにMIDIノートを重ねる

Step 6: Post-Processing
  - Overdrive: Drive 30-50%
  - Delay: 1/16 Dotted, Feedback 30-40%
  - Reverb: Small Room, Dry/Wet 10-15%
```

## 💡 よくある質問（FAQ）

### Q1: 音楽理論がゼロでも大丈夫ですか？

**A**: はい、このセクションは音楽理論初心者でも理解できるように設計されています。

**推奨アプローチ**:
1. まず**サブベース**（理論不要）から始める
2. **ベースライン作成**でルート音の概念を学ぶ
3. **コード進行**で音楽理論の基礎を習得
4. **メロディ作成**で理論を実践に応用

**DJとしての強み**:
- キー（調）の概念は**ハーモニックミキシング**で既に理解
- BPMとリズムは**ビートマッチング**で習得済み
- 音楽構造は**選曲とセットリスト**で体感している

> DJスキルが音楽理論の土台になる！

---

### Q2: サブベースが聞こえません（ヘッドフォン）

**A**: これは正常です。20-60 Hzの超低域はヘッドフォンでは聞き取りにくいです。

**解決方法**:

1. **スペクトラムアナライザーで確認**
   ```
   Utility → Spectrum On
   → 50 Hz付近にピークが見える
   ```

2. **Mid Bassとレイヤリング**
   ```
   Sub Bass: 20-60 Hz (聞こえにくい)
   Mid Bass: 60-250 Hz (聞こえる)
   → 2つを重ねる
   ```

3. **クラブシステムでテスト**
   - サブウーファー（18インチ以上）で初めて体感
   - 家のスピーカーでは50 Hzは出ない

4. **サイン波テスト**
   ```
   Test Tone Generator:
   30 Hz → ほぼ聞こえない
   50 Hz → かすかに聞こえる
   80 Hz → はっきり聞こえる
   ```

---

### Q3: ベースとキックが喧嘩します

**A**: これは**周波数マスキング**の問題です。適切なEQ処理で解決します。

**解決方法**:

1. **周波数帯域の分離**
   ```
   Kick: 50-60 Hz (Sub)、80-100 Hz (Punch)
   Sub Bass: 40-50 Hz (Kick より低い)
   Mid Bass: 100-250 Hz (Kick の上)
   ```

2. **サイドチェインコンプレッション**
   ```
   Bass Track:
   → Compressor (Sidechain: Kick)
   → Attack 0 ms、Release 100 ms
   → Ratio 4:1、Threshold -20 dB

   → Kickが鳴る瞬間だけBassが下がる
   ```

3. **EQ でのスペース作り**
   ```
   Kick: 60 Hz を +3 dB ブースト
   Bass: 60 Hz を -2 dB カット
   → お互いに干渉しない
   ```

**詳細**: [05-mixing/frequency-balance.md](../05-mixing/frequency-balance.md)

---

### Q4: メロディが思いつきません

**A**: メロディ作成は練習で習得できるスキルです。以下のテクニックを試してください。

**即効性のある方法**:

1. **スケール制約を使う**
   ```
   Ableton Scale ツール:
   → C Major または A Minor
   → スケール外の音は自動除外
   → 間違った音を弾けない
   ```

2. **モチーフの繰り返し**
   ```
   2小節のシンプルなメロディ:
   C - E - G - E (1小節目)
   D - F - A - F (2小節目)
   → これを4回繰り返す
   → 3回目で少し変化
   ```

3. **リファレンストラックのコピー**
   ```
   好きな曲のメロディを耳コピ:
   → MIDIで打ち込む
   → 音程を少し変える
   → リズムを変える
   → オリジナルメロディの完成
   ```

4. **ランダム生成 + 編集**
   ```
   MIDI Clip → 右クリック:
   → "Generate Random Notes"
   → 不要な音を削除
   → リズムを整える
   ```

**詳細**: [melody-creation.md](./melody-creation.md)

### Q5: WavetableとOperatorの使い分けは？

**A**: 音色の特性で使い分けます。

| 音色 | Wavetable | Operator | Analog |
|------|----------|----------|--------|
| **サブベース** | ◎ 最適 | △ 可能 | ○ 良い |
| **Technoベース** | ◎ 最適 | ○ FMベース | △ アナログ |
| **808ベース** | ○ 可能 | ◎ FM最適 | △ 可能 |
| **リードシンセ** | ◎ Supersaw | △ ブラス | ○ ビンテージ |
| **パッド** | ○ モダン | △ ベル系 | ◎ 温かい |
| **Pluckシンセ** | ◎ 最適 | ○ FM Piano | △ 可能 |

**使用頻度（プロデューサー調査）**:
- Wavetable: 70% (最も汎用的)
- Operator: 20% (FM、ベル、ブラス)
- Analog: 10% (ビンテージ、温かい音)

**推奨アプローチ**:
1. **まずWavetableから始める**（最も簡単）
2. FMサウンドが必要な時だけOperator
3. ビンテージ感が欲しい時だけAnalog

### Q6: コード進行が単調に聞こえます

**A**: **ボイシング**と**リズム**を工夫することで解決します。

**解決方法**:

1. **コードボイシング（音の配置）**
   ```
   基本形（退屈）:
   C Major: C3 - E3 - G3 (全て近い)

   転回形（面白い）:
   C Major: C3 - G3 - E4 (広がり)
   C Major: E3 - C4 - G4 (高音域)
   ```

2. **リズムバリエーション**
   ```
   パターン1（4つ打ち）:
   |C---|C---|C---|C---|

   パターン2（Offbeat）:
   |-C-|-C-|-C-|-C-|

   パターン3（シンコペーション）:
   |C-C|--C|C-C|--C|
   ```

3. **7thコード、テンション**
   ```
   基本（トライアド）:
   C Major: C - E - G

   拡張（7th）:
   C Major 7: C - E - G - B (ジャジー)

   さらに拡張（9th）:
   C Major 9: C - E - G - B - D (複雑)
   ```

**詳細**: [chord-progressions.md](./chord-progressions.md)

---

### Q7: リードシンセが埋もれてしまいます

**A**: **ミキシング**と**音色デザイン**の両面で対処します。

**音色デザイン**:

1. **高域を強調**
   ```
   Wavetable:
   → Filter Cutoff 8000 Hz
   → Resonance 20-30%
   → High Shelf +3 dB at 4 kHz
   ```

2. **Unisonで厚みを作る**
   ```
   Wavetable Oscillator:
   → Unison 4-8 voices
   → Detune 30-50%
   → ステレオ幅60-80%
   ```

3. **フィルターエンベロープで動きを作る**
   ```
   Filter Envelope:
   → Attack 10 ms
   → Decay 300 ms
   → Sustain 60%
   → Release 500 ms
   → Envelope Amount +30%
   ```

**ミキシング**:

1. **EQ で他のトラックとスペースを作る**
   ```
   Lead: 1-4 kHz をブースト +3 dB
   Pad: 1-4 kHz をカット -3 dB
   → お互いに干渉しない
   ```

2. **コンプレッションで前に出す**
   ```
   Compressor:
   → Ratio 3:1
   → Threshold -15 dB
   → Attack 10 ms、Release 100 ms
   → Makeup Gain +3 dB
   ```

3. **ステレオ幅を活用**
   ```
   Lead: 60-80% Width (中央寄り)
   Pad: 100% Width (広い)
   → リードが中央で際立つ
   ```

**詳細**: [lead-synth.md](./lead-synth.md)、[05-mixing/](../05-mixing/)

## 🚀 今日から始める

### 最初の1時間

**目標**: Wavetableでサブベースを作成

1. **新規MIDIトラック作成**
   - Cmd+Shift+T (新規MIDIトラック)
   - Wavetableをロード

2. **サイン波サブベース設定**
   - Oscillator 1: Basic Shapes → Sine
   - Oscillator 2: Off
   - Filter: Low Pass 100 Hz

3. **C1のルート音を打ち込み**
   - C1 (MIDI Note 36)
   - 4小節ループ、4つ打ち

4. **Utility でMono化**
   - Utility → Bass Mono: On (120 Hz)

5. **スペクトラムで確認**
   - Utility → Spectrum: On
   - 約33 Hz (C1) にピークが見える

**完成！** 最初のサブベースが作成できました。

## 🎓 まとめ

### このセクションで習得する核心

1. **ベース**: 楽曲の土台、グルーヴの源
2. **メロディ**: 楽曲の魂、記憶に残る要素
3. **コード進行**: 感情、雰囲気を決定
4. **シンセデザイン**: 個性的な音色を作る技術

### DJから制作者へ

**DJとして**:
- ベースラインでフロアを揺らす経験
- メロディで観客を感動させる瞬間
- キーの重要性を体感

**プロデューサーとして**:
- その「魔法」を自分で作り出す
- 音楽理論を実践に応用
- 無限の創造性を解放

### プロへの道

**初心者（Month 1-2）**:
- プリセット活用でベース・メロディ作成
- シンプルなコード進行（I-V-vi-IV）
- リファレンストラックのコピー

**中級者（Month 3-6）**:
- オリジナルベース音色デザイン
- 独自のメロディ作成
- 複雑なコード進行、カウンターメロディ

**上級者（Month 7-12）**:
- ジャンル別ベース・メロディの使い分け
- 完全オリジナル楽曲制作
- ハーモニーの完全理解

**プロ（Year 2+）**:
- 独自のサウンド確立
- リリース可能な楽曲を量産
- 他のプロデューサーにも認められる
