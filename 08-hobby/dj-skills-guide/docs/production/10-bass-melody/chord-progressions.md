# コード進行

**楽曲の感情とハーモニーを決定する**

コード進行は楽曲の「感情の設計図」です。同じメロディでも、背後のコード進行次第で明るくも暗くも聞こえます。このガイドでは、音楽理論の基礎から定番コード進行10選、実践的な作成テクニックまで、プロレベルのハーモニー構築を完全マスターします。

## なぜコード進行が重要なのか

### DJの視点から

**DJとして**:
- コード進行の変化で**楽曲のセクションが切り替わる**
- ハーモニックミキシングで**キー（調）の重要性**を理解
- マイナーキーとメジャーキーで**雰囲気が変わる**ことを体感

**プロデューサーとして**:
- その「感情」を**自分で設計**できる
- コード進行だけで**楽曲の80%が決まる**
- ハーモニーで**独自性**を確立できる

### プロの意見

> "コード進行は楽曲の感情。I-V-vi-IVは永遠に機能する。"
> — **Max Martin**（ヒット量産プロデューサー）

> "Progressive Houseでは、コード進行がすべて。メロディは後から付ければいい。"
> — **Deadmau5**

> "同じ4つのコードで1000曲作れる。重要なのはボイシングとリズム。"
> — **Calvin Harris**

## 1. コード理論の基礎

### 1.1 トライアド（3和音）

**メジャーコード（明るい）**:
```
C Major:
  C - E - G
  Root - Major 3rd - Perfect 5th
  半音数: 0 - 4 - 7

特徴: 明るい、楽しい、安定
```

**マイナーコード（暗い）**:
```
A Minor:
  A - C - E
  Root - Minor 3rd - Perfect 5th
  半音数: 0 - 3 - 7

特徴: 暗い、悲しい、メランコリック
```

**違いは3rdだけ**:
```
C Major: C - E (Major 3rd, 4半音)
C Minor: C - Eb (Minor 3rd, 3半音)

→ たった1半音の違いで雰囲気が変わる
```

### 1.2 ダイアトニックコード

**C Major Scaleから作られる7つのコード**:

```
Scale: C - D - E - F - G - A - B - C

Chord I:   C - E - G  = C Major (CM)
Chord ii:  D - F - A  = D minor (Dm)
Chord iii: E - G - B  = E minor (Em)
Chord IV:  F - A - C  = F Major (FM)
Chord V:   G - B - D  = G Major (GM)
Chord vi:  A - C - E  = A minor (Am)
Chord vii: B - D - F  = B diminished (Bdim)
```

**パターン**:
```
Major Scale:
  I   ii  iii IV  V   vi  vii°
  Maj min min Maj Maj min dim

覚え方: 大小小大大小減
```

**機能**:
```
Tonic (安定):     I, iii, vi
Subdominant (準備): II, IV
Dominant (緊張):   V, vii°

進行の原則:
  Tonic → Subdominant → Dominant → Tonic
  (安定 → 準備 → 緊張 → 解決)
```

### 1.3 7thコード

**トライアド + 7th**:

```
C Major 7 (CMaj7):
  C - E - G - B
  Root - M3 - P5 - Major 7th

特徴: ジャジー、洗練、Progressive House
```

**その他の7thコード**:
```
C Dominant 7 (C7):    C - E - G - Bb (ブルース的)
C Minor 7 (Cm7):      C - Eb - G - Bb (ジャズ的)
C Minor Major 7 (CmM7): C - Eb - G - B (ドラマティック)
```

**使用頻度**:
```
EDM: Triadが90%、7thは10%
Jazz/Soul House: 7thが60%
```

## 2. 定番コード進行10選

### 2.1 I-V-vi-IV（最強の進行）

**C Majorの場合**:
```
I:  C Major  (C - E - G)
V:  G Major  (G - B - D)
vi: A minor  (A - C - E)
IV: F Major  (F - A - C)

進行: C → G → Am → F
```

**特徴**:
- ポップミュージック史上最も使用される
- 明るく、キャッチー
- ループしやすい

**使用例**:
- "Let It Be" - The Beatles
- "Don't Stop Believin'" - Journey
- "Someone Like You" - Adele
- "Wake Me Up" - Avicii

**EDMでの使用**:
- Progressive House 70%
- Pop EDM 80%
- Future Bass 60%

### 2.2 vi-IV-I-V（悲しげ）

**C Majorの場合**:
```
vi: A minor
IV: F Major
I:  C Major
V:  G Major

進行: Am → F → C → G
```

**特徴**:
- マイナーから始まる（暗い）
- 徐々に明るくなる（希望）
- エモーショナル

**使用例**:
- "All of Me" - John Legend
- "Faded" - Alan Walker

### 2.3 ii-V-I（ジャズの王道）

**C Majorの場合**:
```
ii: D minor  (D - F - A)
V:  G Major  (G - B - D)
I:  C Major  (C - E - G)

進行: Dm → G → C
```

**特徴**:
- ジャズで最も重要
- 強い解決感
- Sophisticatedな印象

**使用例**:
- Jazz House
- Nu-Disco
- Soulful House

### 2.4 I-IV-V（ロックの基本）

**C Majorの場合**:
```
I:  C Major
IV: F Major
V:  G Major

進行: C → F → G
```

**特徴**:
- シンプル、力強い
- ロック、ブルース
- 3コードだけ

**EDMでの使用**:
- Big Room House
- Electro House

### 2.5 i-VII-VI-V（マイナー進行）

**A Minorの場合**:
```
i:   A minor  (A - C - E)
VII: G Major  (G - B - D)
VI:  F Major  (F - A - C)
V:   E Major  (E - G# - B)

進行: Am → G → F → E
```

**特徴**:
- ダーク、ドラマティック
- ハードスタイル、メタル的
- Deep House、Techno

**使用例**:
- Deep House 50%
- Melodic Techno 60%

### 2.6 I-vi-IV-V（50's進行）

**C Majorの場合**:
```
I:  C Major
vi: A minor
IV: F Major
V:  G Major

進行: C → Am → F → G
```

**特徴**:
- 1950年代ポップ
- ノスタルジック
- 安定感

### 2.7 I-V-vi-iii-IV-I-IV-V（Canon進行）

**C Majorの場合**:
```
C → G → Am → Em → F → C → F → G

パッヘルベルのカノン
```

**特徴**:
- クラシック由来
- 非常に美しい
- Progressive House、Trance

### 2.8 i-VI-III-VII（Andalusian）

**A Minorの場合**:
```
i:   A minor
VI:  F Major
III: C Major
VII: G Major

進行: Am → F → C → G
```

**特徴**:
- スペイン、フラメンコ的
- エキゾチック
- Techno、Progressive House

### 2.9 I-bVII-IV（モーダル）

**C Majorの場合**:
```
I:    C Major
bVII: Bb Major（借用和音）
IV:   F Major

進行: C → Bb → F
```

**特徴**:
- ロック的
- パワフル
- Big Room House

### 2.10 i-iv-v（マイナー3コード）

**A Minorの場合**:
```
i:  A minor
iv: D minor
v:  E minor

進行: Am → Dm → Em
```

**特徴**:
- ダーク、ミニマル
- Techno、Deep House

## 3. Ableton Live 実践

### 3.1 MIDI打ち込み基礎

**新規トラックでコード作成**:
```
1. Cmd+Shift+T（新規MIDIトラック）
2. 音源: Wavetable, Analog, または Samplerでピアノ音色
3. 空のMIDI Clipを作成
```

**C Majorコードの打ち込み**:
```
1. ピアノロールで以下の3音を同時に配置:
   - C3
   - E3
   - G3

2. 長さ: 1小節（4拍）

3. Velocity: 100（全音同じ）
```

### 3.2 コード進行の打ち込み（I-V-vi-IV）

```
Bar 1: C Major
  - C3, E3, G3

Bar 2: G Major
  - G3, B3, D4

Bar 3: A minor
  - A3, C4, E4

Bar 4: F Major
  - F3, A3, C4

→ 4小節のループ
```

### 3.3 Scaleツール活用

**スケール設定**:
```
1. MIDI Clip選択
2. Clip View → Notes → Scale
3. Root: C
4. Scale: Major
5. Fold: On

→ C, D, E, F, G, A, Bだけ表示
```

**コード作成が簡単に**:
```
Fold Onの状態:
  - 縦に3音選択 → 自動的にコード
  - 間違った音が弾けない
```

### 3.4 Chord MIDIエフェクト

**コードを自動生成**:

```
1. MIDI Effect → Chord
2. Shift 1: +4 (Major 3rd)
3. Shift 2: +7 (Perfect 5th)

→ C を弾くと C-E-G (C Major) が鳴る
```

**設定例**:
```
Major Chord:
  Shift 1: +4
  Shift 2: +7

Minor Chord:
  Shift 1: +3 (Minor 3rd)
  Shift 2: +7

7th Chord:
  Shift 1: +4
  Shift 2: +7
  Shift 3: +11 (Major 7th)

→ 1音打ち込むだけでコード完成
```

## 4. コードボイシング

### 4.1 基本形vs転回形

**基本形（Root Position）**:
```
C Major: C3 - E3 - G3
         Root 3rd 5th

特徴: 安定、標準
```

**第1転回形（1st Inversion）**:
```
C Major: E3 - G3 - C4
         3rd 5th Root

ベース音: E
特徴: 軽やか、動きがある
```

**第2転回形（2nd Inversion）**:
```
C Major: G3 - C4 - E4
         5th Root 3rd

ベース音: G
特徴: 不安定、トランジションに使用
```

### 4.2 スムーズな声部進行

**近い音程で移動（Voice Leading）**:

**悪い例（ジャンプが大きい）**:
```
C Major: C3 - E3 - G3
F Major: F3 - A3 - C4  ← 全音が大きく移動
```

**良い例（最小限の移動）**:
```
C Major: C3 - E3 - G3
F Major: C3 - F3 - A3  ← C3は共通、E→F（1音）、G→A（2音）

→ スムーズ、自然な流れ
```

**原則**:
```
1. 共通音はそのまま維持
2. 他の音は最小限の移動
3. 2度（1-2音）以内の移動が理想
```

### 4.3 オープン vs クローズドボイシング

**クローズドボイシング（狭い）**:
```
C Major: C3 - E3 - G3
         1オクターブ以内

特徴: コンパクト、明瞭
用途: House、Pop
```

**オープンボイシング（広い）**:
```
C Major: C3 - G3 - E4
         1オクターブ以上

特徴: 広がり、エアリー
用途: Jazz、Progressive House
```

### 4.4 ダブリング

**ルート音を倍にする**:
```
C Major: C2 - C3 - E3 - G3 - C4
         ↑       ↑           ↑
         Root音を3回（低音、中音、高音）

特徴: 厚み、力強さ
用途: Big Room House、Anthemicなトラック
```

## 5. ジャンル別コード進行

### 5.1 Progressive House

**定番進行**:
```
Am - F - C - G (vi-IV-I-V)

特徴:
  - エモーショナル
  - マイナーから始まる
  - ビルドアップで盛り上がる

音色: Saw Pad, Pluck

BPM: 125-128

例: "Language" - Porter Robinson
```

### 5.2 Trance

**定番進行**:
```
C - Am - F - G (I-vi-IV-V)

または:

Am - F - C - G (vi-IV-I-V)

特徴:
  - エピック、感動的
  - 長いビルドアップ（16-32小節）
  - メジャー7thを追加

音色: Supersaw, Epic Pad

BPM: 138-140
```

### 5.3 Deep House

**定番進行**:
```
Am - Dm - G - C (i-iv-VII-III)

または:

Dm7 - G7 - CMaj7 - Am7 (ii7-V7-IMaj7-vi7)

特徴:
  - ジャジー
  - 7thコード必須
  - 温かみのある音色

音色: Rhodes, Warm Pad

BPM: 120-125
```

### 5.4 Techno

**定番進行**:
```
Am - Am - Am - Am (i-i-i-i)

または:

Am - F - Am - F (i-VI-i-VI)

特徴:
  - ミニマル
  - 変化が少ない
  - ダーク、催眠的

音色: Dark Pad, Acid

BPM: 125-135
```

### 5.5 Future Bass

**定番進行**:
```
C - Am - F - G (I-vi-IV-V)

または:

vi - IV - I - V（暗→明）

特徴:
  - 明るく、感動的
  - 7thコード、テンション
  - ボーカルとの相性◎

音色: Supersaw, Vocal Chops

BPM: 140-170
```

## 6. リズムとボイシング

### 6.1 4つ打ちコード（House）

```
Kick:  |X---|X---|X---|X---|
Chord: |C---|G---|Am--|F---|

各拍でコードチェンジ
リズム: 4分音符
```

### 6.2 Offbeat コード（Funky House）

```
Kick:  |X---|X---|X---|X---|
Chord: |-C--|-G--|-Am-|-F--|

Kickの直後にコード
リズム: 8分音符（裏拍）
```

### 6.3 シンコペーション

```
Chord: |C-C-|--G-|Am--|--F-|
       1 & 2 & 3 & 4 &

不規則なリズム
グルーヴィー
```

### 6.4 Pluck パターン（Progressive House）

```
C Majorコード（C-E-G）を分解:

|C-E-G-E-|C-E-G-C5-|

Arpeggioのように1音ずつ
音色: Pluck Synth
リズム: 16分音符
```

## 7. ミキシング

### 7.1 EQ処理

**コードの基本EQ**:
```
EQ Eight:

1. High Pass 100-200 Hz
   （ベースとの分離）

2. Low-Mid Cut 250-400 Hz -2 dB
   （マッディネス除去）

3. Presence Boost 1-2 kHz +1 dB
   （明瞭さ）

4. Air Boost 8-10 kHz +1 dB
   （輝き）
```

**ジャンル別**:
```
Deep House: 400 Hz周辺を温かくブースト
Trance: 4 kHz以上を明るくブースト
Techno: 2 kHz以下をダークにカット
```

### 7.2 サイドチェイン（必須）

**Kickとの共存**:
```
Chord Track:
→ Compressor
→ Audio From: 1-Kick

設定:
  - Ratio: 4:1
  - Threshold: -20 dB
  - Attack: 10 ms
  - Release: 150 ms

効果:
  - Kickが鳴る瞬間、コードが下がる
  - グルーヴ、ダンサビリティ
  - Progressive House必須
```

### 7.3 ステレオ幅

**コードの最適幅**:
```
Pad Chord: 100% (広く)
Pluck Chord: 60-80% (中央寄り)

Utility:
  - Width: 100% (Pad)
  - Width: 70% (Pluck)
```

### 7.4 Reverb/Chorus

**空間と厚み**:
```
Return Track A: Reverb
  - Hall 2.5s
  - Pre-Delay: 30 ms
  - Dry/Wet: 100%

Chord Send A: 30-40%

Return Track B: Chorus
  - Rate: 0.5 Hz
  - Depth: 30%

Chord Send B: 20% (温かみ)
```

## 9. よくある失敗と対処法

### 失敗1: コードが合わない音がする

**対処法**:
```
1. Scaleツールで In Key 確認
2. ダイアトニックコードのみ使用
3. 3rd（3度）の音を確認
   - Majorなら4半音（例: C→E）
   - Minorなら3半音（例: A→C）
```

---

### 失敗2: コード進行が単調

**対処法**:
```
1. リズムバリエーション（Offbeat、シンコペーション）
2. ボイシング変化（転回形）
3. 7thコード追加
4. 一部のコードをマイナーに（モーダル・インターチェンジ）
```

---

### 失敗3: ベースラインと合わない

**対処法**:
```
1. ベースはコードのRoot音を演奏
   C Major → ベースはC
2. または5th（G）を使用
3. 転回形の場合、ベース音を別トラックで指定
```

---

### 失敗4: コードが埋もれる

**対処法**:
```
1. EQ: 1-2 kHz +2 dB（明瞭さ）
2. サイドチェイン（Kickとの共存）
3. ステレオ幅100%（Pad）
4. Reverb/Chorus で空間を作る
```

## 10. テンションコードとエクステンデッドハーモニー

### 10.1 テンションとは何か

テンション（Tension）とは、トライアドや7thコードの上に積み重ねる追加の音です。9th、11th、13thなどが代表的で、コードに複雑さ、色彩、深みを与えます。EDM制作ではDeep House、Future Bass、Lo-Fi Houseなどで特に重要です。

**テンションの種類と半音数**:
```
9th (2nd の1オクターブ上):
  Major 9th:  14半音 (例: C → D5)
  Minor 9th:  13半音 (例: C → Db5)  ※不協和が強い
  Sharp 9th:  15半音 (例: C → D#5)  ※ブルージー

11th (4th の1オクターブ上):
  Perfect 11th: 17半音 (例: C → F5)
  Sharp 11th:   18半音 (例: C → F#5) ※リディアン的

13th (6th の1オクターブ上):
  Major 13th:  21半音 (例: C → A5)
  Minor 13th:  20半音 (例: C → Ab5) ※ダーク
```

### 10.2 9thコード

9thコードは7thコードにさらに9度の音を加えたもので、モダンで洗練されたサウンドを提供します。

**Major 9th (CMaj9)**:
```
構成音: C - E - G - B - D
         R   M3  P5  M7  M9

特徴:
  - 非常に美しく、柔らかい
  - Lo-Fi House、Chillout に最適
  - ジャズ的な雰囲気

Ableton MIDI:
  C3 - E3 - G3 - B3 - D4
```

**Minor 9th (Cm9)**:
```
構成音: C - Eb - G - Bb - D
         R   m3   P5  m7   M9

特徴:
  - 暗いが洗練されている
  - Deep House のコードワークの要
  - Neo-Soul との親和性が高い

Ableton MIDI:
  C3 - Eb3 - G3 - Bb3 - D4
```

**Dominant 9th (C9)**:
```
構成音: C - E - G - Bb - D
         R   M3  P5  m7   M9

特徴:
  - ファンキー、ブルージー
  - Disco House、Funk House に最適
  - リズミカルなカッティングと相性抜群

使用例:
  Dm9 → G9 → CMaj9 (ii9-V9-IMaj9)
  → Soulful House の定番
```

### 10.3 11thコード

11thコードは9thコードにさらに11度を加えたもので、浮遊感のあるサウンドが特徴です。

**Major 11th (CMaj7#11)**:
```
構成音: C - E - G - B - D - F#
         R   M3  P5  M7  M9  #11

特徴:
  - リディアン的な響き
  - 非常に浮遊感がある
  - Ambient、Progressive House のビルドアップ

注意: Perfect 11th (F) はMajor 3rd (E) と
      半音でぶつかるため、#11 (F#) を使用
```

**Minor 11th (Cm11)**:
```
構成音: C - Eb - G - Bb - D - F
         R   m3   P5  m7   M9  P11

特徴:
  - ダークだが開放的
  - Techno のパッドに最適
  - ミニマルな進行でも色彩が豊か

Ableton MIDI:
  C3 - Eb3 - Bb3 - D4 - F4
  (G3 を省略して軽くする)
```

### 10.4 13thコード

13thコードはテンションの最高峰で、6音以上の構成音を持ちます。

**Dominant 13th (C13)**:
```
構成音: C - E - G - Bb - D - A
         R   M3  P5  m7   M9  M13

特徴:
  - フルオーケストラ的な厚み
  - Soulful House、Gospel House に頻出
  - ボイシングの工夫が必須

実用的なボイシング（全音入れると厚すぎるため省略する）:
  C3 - E3 - Bb3 - D4 - A4
  (P5のGを省略)
```

### 10.5 テンションコードのEDM実践ガイド

**ジャンル別テンション使用頻度**:

| ジャンル | 9th | 11th | 13th | 推奨度 |
|---------|-----|------|------|-------|
| Deep House | 多用 | 時々 | 時々 | 必須 |
| Lo-Fi House | 多用 | 多用 | 少ない | 必須 |
| Progressive House | 時々 | 少ない | 少ない | 推奨 |
| Future Bass | 多用 | 多用 | 多用 | 必須 |
| Techno | 少ない | 時々 | 少ない | 任意 |
| Trance | 少ない | 少ない | 少ない | 任意 |
| Big Room | 使わない | 使わない | 使わない | 不要 |

**テンションの追加手順（Ableton）**:
```
Step 1: まずトライアドを打ち込む
  C3 - E3 - G3

Step 2: 7thを追加
  C3 - E3 - G3 - B3

Step 3: 9thを追加
  C3 - E3 - G3 - B3 - D4

Step 4: 必要に応じて11th/13thを追加
  C3 - E3 - G3 - B3 - D4 - F#4

ポイント:
  - 1音ずつ追加して響きを確認する
  - 不協和が強すぎたらボイシングを変える
  - P5（5th）は省略しても問題ない
```

### 10.6 Sus（サス）コードとAdd（アド）コード

テンションコード以外にも、トライアドを変形させたコードがEDMで頻繁に使用されます。

**Sus2 コード (Csus2)**:
```
構成音: C - D - G
         R   M2  P5

特徴:
  - 3rd がないため、メジャー/マイナーの区別がない
  - 開放的、浮遊的
  - Ambient、Chillout に最適
  - エレクトロニカの定番
```

**Sus4 コード (Csus4)**:
```
構成音: C - F - G
         R   P4  P5

特徴:
  - 緊張感がある
  - Majorコードへの解決を期待させる
  - ビルドアップの直前に配置すると効果的

使用例:
  Csus4 → C → Am → F
  （サスペンスから解決）
```

**Add9 コード (Cadd9)**:
```
構成音: C - E - G - D
         R   M3  P5  M9

特徴:
  - 7th を含まないため、CMaj9 より軽い
  - ポップ、EDMで最も使いやすいテンション
  - 明るく、きらびやか

Cadd9 vs CMaj9:
  Cadd9: C - E - G - D     (7th なし)
  CMaj9: C - E - G - B - D (7th あり)
  → Cadd9 の方がシンプルで扱いやすい
```

## 11. モーダルインターチェンジ（借用和音の体系）

### 11.1 モーダルインターチェンジとは

モーダルインターチェンジ（Modal Interchange）とは、同主調の別のモード（旋法）からコードを借用するテクニックです。例えば、CメジャーキーでCナチュラルマイナー（Cエオリアン）のコードを使用するなどです。これにより、ダイアトニック内だけでは得られない色彩豊かなハーモニーが実現できます。

**基本概念**:
```
Key: C Major (Ionian)
ダイアトニック: C  Dm  Em  F  G  Am  Bdim

同主短調 C Natural Minor (Aeolian) から借用:
借用可能: Cm  Ddim  Eb  Fm  Gm  Ab  Bb

→ CメジャーキーでAb、Bb、Fm、Ebなどが使える
```

### 11.2 各モードからの借用コード一覧

**C Major（Ionian）が基準の場合**:

**Cドリアン（Dorian）からの借用**:
```
元のスケール: C - D - Eb - F - G - A - Bb
借用コード: Cm, Dm, Eb, F, Gm, Adim, Bb

最も使われる借用:
  IV → iv (F → Fm)  ← マイナーサブドミナント
  bVII (Bb)

効果: 「甘い切なさ」を追加
用途: R&B、Neo-Soul、Deep House
```

**Cフリジアン（Phrygian）からの借用**:
```
元のスケール: C - Db - Eb - F - G - Ab - Bb
借用コード: Cm, Db, Eb, Fm, Gdim, Ab, Bbm

最も使われる借用:
  bII (Db)  ← ナポリタンコード
  bVI (Ab)

効果: エキゾチック、スペイン風
用途: Melodic Techno、Ethnic House
```

**Cリディアン（Lydian）からの借用**:
```
元のスケール: C - D - E - F# - G - A - B
借用コード: C, D, Em, F#dim, G, Am, Bm

最も使われる借用:
  II (D)  ← 通常はDm(ii)がDメジャーに
  #IV dim (F#dim)

効果: 浮遊感、夢のような響き
用途: Ambient、Progressive House のBreakdown
```

**Cミクソリディアン（Mixolydian）からの借用**:
```
元のスケール: C - D - E - F - G - A - Bb
借用コード: C, Dm, Edim, F, Gm, Am, Bb

最も使われる借用:
  bVII (Bb)  ← 最も一般的な借用和音
  v (Gm)

効果: ロック的、パワフル
用途: Big Room House、Electro House
```

**Cエオリアン（Aeolian = Natural Minor）からの借用**:
```
元のスケール: C - D - Eb - F - G - Ab - Bb
借用コード: Cm, Ddim, Eb, Fm, Gm, Ab, Bb

最も使われる借用:
  bVI (Ab)
  bIII (Eb)
  iv (Fm)
  bVII (Bb)

効果: ダーク、ドラマティック
用途: ほぼ全ジャンルで使用
```

### 11.3 EDMで頻出するモーダルインターチェンジのパターン

**パターン1: IV → iv（マイナーサブドミナント）**:
```
Key: C Major
進行: C - G - F - Fm

I → V → IV → iv

効果:
  - F Major (明るい) → Fm (暗い) のコントラスト
  - 「切ない感じ」を追加
  - ポップEDMの定番テクニック

使用例:
  "Something Just Like This" - The Chainsmokers
  多くのProgressive Houseトラック
```

**パターン2: bVI-bVII-I（マリオ・カデンツ）**:
```
Key: C Major
進行: Ab - Bb - C

bVI → bVII → I

効果:
  - 力強い到達感
  - エピックなドロップに最適
  - ビルドアップの最終段階

使用例:
  Trance のクライマックス
  Epic EDM のドロップ直前
```

**パターン3: I-bIII-bVII-IV**:
```
Key: C Major
進行: C - Eb - Bb - F

効果:
  - ロック的な力強さ
  - Mixolydian + Aeolian の融合
  - Big Room、Festival House に最適
```

**パターン4: i-bVI-bIII-bVII**:
```
Key: C Minor
進行: Cm - Ab - Eb - Bb

効果:
  - メジャーキーのコードが多く、暗すぎない
  - Melodic Techno の定番
  - Progressive Trance でも頻出
```

### 11.4 モーダルインターチェンジの実践手順

```
Step 1: まずダイアトニック進行を作る
  例: C - Am - F - G (I-vi-IV-V)

Step 2: 1つのコードを借用和音に置換
  例: C - Am - Fm - G (IV→iv)

Step 3: 響きを確認
  - 良ければ採用
  - 違和感があればボイシングを調整

Step 4: さらに実験
  例: C - Ab - Fm - G (vi→bVI, IV→iv)

注意点:
  - 一度に2つ以上の借用は注意（カオスになる）
  - 借用コードの前後でスムーズな声部進行を意識
  - ベースラインが半音移動するとスムーズ
```

## 12. 転調テクニック

### 12.1 転調とは

転調（Modulation）は楽曲の途中でキー（調）を変更するテクニックです。EDMではビルドアップやBreakdownで転調を使い、エネルギーの変化や感動の演出に活用します。

**転調の種類**:
```
1. 直接転調（Direct Modulation）
   → 何の準備もなくいきなりキーが変わる

2. ピボットコード転調（Pivot Chord Modulation）
   → 両方のキーに共通するコードを経由して転調

3. クロマティック転調（Chromatic Modulation）
   → 半音進行で滑らかに移行

4. エンハーモニック転調（Enharmonic Modulation）
   → 異名同音を利用した転調
```

### 12.2 直接転調

最もシンプルな転調方法で、EDMでは最も多用されます。

**半音上げ転調（+1）**:
```
元のキー: C Major
新しいキー: Db Major（半音上）

進行:
  C - G - Am - F | Db - Ab - Bbm - Gb
  ↑ C Major      | ↑ Db Major

効果:
  - エネルギーが一気に上がる
  - ビルドアップ後のドロップで使用
  - 最も簡単で効果的

使用例:
  Trance では16小節ごとに半音上げが定番
  ビルドアップのラスト4小節で転調
```

**全音上げ転調（+2）**:
```
元のキー: C Major
新しいキー: D Major（全音上）

進行:
  Am - F - C - G | Bm - G - D - A

効果:
  - 自然な高揚感
  - 半音上げより柔らかい
  - Progressive House のBreakdown → Drop
```

**4度上げ転調（+5）**:
```
元のキー: C Major
新しいキー: F Major（4度上）

進行:
  C - G - Am - F | F - C - Dm - Bb

効果:
  - サブドミナントへの自然な移行
  - 楽曲の中盤での雰囲気転換
  - スムーズで違和感が少ない
```

**平行調への転調（Relative Key）**:
```
C Major → A Minor（平行短調）

進行:
  C - G - Am - F | Am - Em - F - G

効果:
  - 明るい → 暗い の自然な変化
  - 共通の音が多く、スムーズ
  - Breakdownで暗くし、ドロップで明るく戻す
```

### 12.3 ピボットコード転調

両方のキーに共通するコード（ピボットコード）を経由して、滑らかに転調する技法です。

**CメジャーからGメジャーへの転調**:
```
C Major のダイアトニック: C  Dm  Em  F  G  Am  Bdim
G Major のダイアトニック: G  Am  Bm  C  D  Em  F#dim

共通コード（ピボット候補）:
  C  = C Major の I   = G Major の IV
  Em = C Major の iii = G Major の vi
  Am = C Major の vi  = G Major の ii
  G  = C Major の V   = G Major の I

転調例:
  C - Am - F - G | Em - Am - D - G
  ↑ C Major (V=G) | ↑ Emをピボットに G Majorへ

解説:
  - Em は C Major では iii、G Major では vi
  - Am は C Major では vi、G Major では ii
  - D (G Majorの V) が出た瞬間、転調が確定
```

**ピボットコード早見表（CメジャーからX Majorへの転調）**:

| 転調先 | ピボットコード | 関係 |
|-------|-------------|------|
| G Major | Em, Am, C, G | 5度上（最も自然） |
| F Major | Dm, Am, F, C | 4度上（自然） |
| D Major | Em, G | やや遠い |
| A Major | Am → A（変化させる） | 遠い |
| Eb Major | なし（直接転調推奨） | 遠い |
| Ab Major | なし（直接転調推奨） | 遠い |

### 12.4 クロマティック転調

半音階的な動きで滑らかに転調する高度な手法です。

**ベースラインの半音進行**:
```
Key: C Major → Db Major

進行:
  C - Am - F - G | G - Ab - | Db - Bbm - Gb - Ab
  ↑ C Major              ↑ 半音上昇   ↑ Db Major

ベースライン:
  C  A  F  G  G  Ab  Db  Bb  Gb  Ab
                ↑ G→Ab で半音上昇、新キーへ

効果:
  - 非常にスムーズ
  - リスナーが気づかないうちに転調
  - Ambient、Chillout で効果的
```

**ドミナントの半音接近**:
```
Key: C Major → Db Major

進行:
  C - G - Am - Ab7 | Db - Ab - Bbm - Gb

解説:
  Ab7 は Db Major のドミナント（V7）
  C Major の文脈では唐突に聞こえるが、
  直後に Db Major が来ることで解決される
  → セカンダリードミナントの応用
```

### 12.5 EDMにおける転調のタイミング

**ビルドアップ → ドロップ（最も一般的）**:
```
構成:
  Breakdown (16 bars): C Major
  Buildup (8 bars):    C Major → 最後の1-2 barで上げ
  Drop (16 bars):      Db Major（半音上）

効果:
  - ドロップのエネルギーが劇的に増加
  - リスナーに「何か変わった」感覚を与える
  - Trance、Progressive House の定番
```

**2番ドロップで転調**:
```
構成:
  Drop 1 (16 bars):    A Minor
  Breakdown 2 (16 bars): A Minor → B Minor への準備
  Drop 2 (16 bars):    B Minor（全音上）

効果:
  - 2番ドロップが1番より高揚感がある
  - フェスティバルで盛り上がる
  - Martin Garrix、Avicii などが多用
```

**Breakdown で平行調に転調**:
```
構成:
  Drop (16 bars):       A Minor (暗い)
  Breakdown (16 bars):  C Major (明るい、平行長調)
  Buildup (8 bars):     C Major → A Minor に戻す
  Drop 2 (16 bars):     A Minor

効果:
  - Breakdown の「浮遊感」「希望感」を演出
  - ドロップとのコントラストが鮮明
```

### 12.6 Ableton Liveでの転調実装

**方法1: MIDIノートを全選択して移動**:
```
1. 転調したいセクションのMIDIノートを全選択
2. Shift + 上矢印キーで半音ずつ移動
3. 半音上げ転調 = 1回上矢印
4. 全音上げ転調 = 2回上矢印

注意: スケール設定は新しいキーに合わせて変更
```

**方法2: Pitch MIDIエフェクト**:
```
1. MIDI Effect → Pitch
2. Pitch: +1（半音上げ）
3. トラック全体に適用

利点: 元のMIDIデータを変更しない
欠点: トラック全体に影響
→ セクションごとにオートメーションで on/off
```

**方法3: 新しいMIDI Clipを作成**:
```
1. 転調後のセクション用に新しいClipを作成
2. 新しいキーでコード進行を打ち込み
3. ボイシングを最適化

利点: 最もフレキシブル
欠点: 手間がかかる
→ プロの制作では最も一般的
```

## 13. ボイシング発展テクニック

### 13.1 ドロップ2ボイシング

ジャズで多用されるボイシングで、クローズドボイシングの上から2番目の音を1オクターブ下げたものです。Deep House、Nu-Disco で美しいサウンドを生み出します。

**CMaj7のドロップ2**:
```
クローズド: C4 - E4 - G4 - B4
                      ↑ 上から2番目

ドロップ2:  G3 - C4 - E4 - B4
            ↑ G を1オクターブ下へ

特徴:
  - 広がりのあるサウンド
  - ベースラインとの分離が良い
  - ギター的なボイシング
```

**Dm7のドロップ2**:
```
クローズド: D4 - F4 - A4 - C5
ドロップ2:  A3 - D4 - F4 - C5

→ ii-V-I をドロップ2で:
  A3-D4-F4-C5 | B3-D4-F4-B4 | G3-C4-E4-B4
  (Dm7)         (G7)           (CMaj7)
```

### 13.2 クラスターボイシング

隣接する音程で密集させたボイシングで、テンション感と現代的な響きを生み出します。

**Cm9 クラスター**:
```
通常:   C3 - Eb3 - G3 - Bb3 - D4
クラスター: Bb3 - C4 - D4 - Eb4

特徴:
  - 2度音程が隣接（密集）
  - 強いテンション感
  - Ambient Techno、IDM で効果的
  - ルート音はベーストラックに任せる
```

### 13.3 4度堆積ボイシング（Quartal Voicing）

3度ではなく4度で音を積み重ねるモダンなボイシングです。

**C Quartal**:
```
構成: C - F - Bb - Eb
      4度  4度  4度

特徴:
  - メジャー/マイナーの区別が曖昧
  - モダンジャズ的
  - Ambient、Downtempo に最適
  - Wayne Shorter、Herbie Hancock が多用

EDM応用:
  パッドに4度堆積 → 浮遊感のあるテクスチャー
  Melodic Techno のバックグラウンドハーモニー
```

### 13.4 パワーコード（5度のみ）

3rdを省略し、RootとPerfect 5thだけのコードです。

**C5 (C Power Chord)**:
```
構成: C - G
      R   P5

特徴:
  - メジャー/マイナーの区別なし
  - 力強い、シンプル
  - Distortion との相性が良い
  - Big Room House、Hardstyle で多用

ダブリング:
  C2 - G2 - C3 - G3
  → 極太のサウンド
```

### 13.5 スプレッドボイシング（Wide Voicing）

音域を極端に広げたボイシングで、エピックなサウンドスケープを作ります。

**CMaj7 スプレッド**:
```
構成: C2 - G3 - E4 - B5
      ↑3オクターブ以上に渡る

特徴:
  - シネマティック
  - Trance のBreakdown
  - Epic Progressive House
  - Hans Zimmer的なサウンド

Ableton実装:
  - 低音（C2）: サブベースのシンセ
  - 中音（G3）: パッド
  - 高音（E4, B5）: ベルやシマー系シンセ
  → 複数トラックで1つのコードを表現
```

## 14. コードのオートメーションとモーフィング

### 14.1 フィルターオートメーション

コード自体は変えずに、フィルターのカットオフ周波数をオートメーションすることで、エネルギーの変化を演出できます。

**Low-Pass Filter オートメーション**:
```
Breakdown:
  カットオフ: 200 Hz → こもった音、遠い印象
  8小節かけて徐々に開く

Buildup:
  カットオフ: 200 Hz → 20 kHz
  8小節でフルオープンへ

Drop:
  カットオフ: 20 kHz（フルオープン）

Ableton設定:
  1. Auto Filter をコードトラックに追加
  2. Frequency のオートメーションレーンを表示
  3. ブレイクポイントを描画

カーブの種類:
  - リニア: 均一な変化
  - エクスポネンシャル: 後半で急激に開く（推奨）
  - S字: 滑らかな変化
```

### 14.2 レゾナンスオートメーション

フィルターのレゾナンスをオートメーションすることで、ビルドアップの緊張感を高めます。

```
Buildup:
  Resonance: 0% → 60%
  最後の2小節で急上昇

Drop直前:
  Resonance: 60% → 0%（瞬間的にリセット）

効果:
  - フィルタースイープの「キュイーン」音
  - テンションの蓄積
  - ドロップ時の解放感が増す
```

### 14.3 ボイシングモーフィング

同じコード進行でもボイシングをセクションごとに変化させることで、楽曲に動きを与えます。

**セクション別ボイシング計画**:
```
Intro (8 bars):
  - クローズドボイシング
  - 低い音域（C3-G3）
  - 音数: 3音（トライアド）

Verse (16 bars):
  - 第1転回形
  - 中音域（E3-C4）
  - 音数: 3-4音

Buildup (8 bars):
  - 徐々にオープンボイシングへ
  - 音域拡大（C3-E4 → C2-B5）
  - 音数: 3 → 5音（テンション追加）

Drop (16 bars):
  - スプレッドボイシング
  - 広い音域（C2-B5）
  - 音数: 5-6音（フル）

Breakdown (16 bars):
  - ドロップ2ボイシング
  - 中音域
  - 音数: 4音（7th含む）
```

### 14.4 パッドモーフィング（Wavetable使用）

Ableton の Wavetable シンセを使い、コードパッドの音色をリアルタイムで変化させます。

**設定手順**:
```
1. Wavetable でパッド音色を作成
   Oscillator 1: Saw → Sine のウェーブテーブル
   Oscillator 2: Square → Triangle のウェーブテーブル

2. Wavetable Position をオートメーション
   Breakdown: Position = 0 (Saw/Square = 明るい)
   → Drop に向けて
   Drop: Position = 50% (中間 = 温かい)
   → Breakdown 2 に向けて
   Breakdown 2: Position = 100% (Sine/Triangle = 柔らかい)

3. Sub Oscillator を追加
   Drop セクションでのみ ON
   → 低域の厚みを追加

効果:
  - 同じMIDIでも音色が変化
  - セクションごとの表情が変わる
  - リスナーを飽きさせない
```

### 14.5 コードリズムのオートメーション

コードのリズムパターンをセクションごとに変化させます。

```
Intro:
  |C---|----|----|----| (全音符、1小節に1コード)
  → 穏やか、ゆったり

Verse:
  |C-C-|C-C-|----|----| (2分音符)
  → やや動きがある

Pre-chorus:
  |C-C-C-C-|C-C-C-C-| (4分音符)
  → エネルギー上昇

Buildup:
  |CCCCCCCC|CCCCCCCC| (8分音符)
  → 緊張感MAX

Drop:
  |C---|-C--|----|C---| (シンコペーション)
  → グルーヴィー、踊れる

Breakdown:
  |C---|----|----|----| (全音符に戻る)
  → リラックス
```

### 14.6 Velocity オートメーション

MIDIのVelocity（強弱）を変化させることで、コードの表情を細かくコントロールします。

```
ダイナミクスパターン:

パターン1（クレッシェンド）:
  Bar 1: Velocity 60
  Bar 2: Velocity 75
  Bar 3: Velocity 90
  Bar 4: Velocity 110
  → ビルドアップ向き

パターン2（アクセント）:
  Beat 1: Velocity 120 (強)
  Beat 2: Velocity 80  (弱)
  Beat 3: Velocity 100 (中)
  Beat 4: Velocity 80  (弱)
  → グルーヴ生成

パターン3（ゴーストノート）:
  On-beat: Velocity 110
  Off-beat: Velocity 40-60
  → 裏拍を弱く弾いて繊細さを出す

パターン4（ランダマイズ）:
  基準: Velocity 90
  ランダム幅: +/- 15
  → 人間らしい揺らぎ（機械的にならない）

Ableton設定:
  1. MIDI Clip → Velocity レーンを表示
  2. 個別ノートのVelocityをドラッグで変更
  3. Cmd+A で全選択 → Randomize で揺らぎ追加
```

## 15. ジャンル別コード進行パターン拡張

### 15.1 Melodic Techno

Melodic Techno はダークでありながらメロディックな要素を持ち、コード進行が雰囲気の鍵を握ります。

**パターン1: ミニマルループ**:
```
Am - Am - Am - Am
(単一コードのループ)

テクニック:
  - コード自体は変えない
  - フィルター、LFO でテクスチャーを変化
  - パッドの音量を徐々に変化
  - アルペジオで1音ずつ分解

アーティスト参考:
  Stephan Bodzin, Tale Of Us, Recondite
```

**パターン2: 2コードシャトル**:
```
Am - F - Am - F (i-bVI)
または
Am - Em - Am - Em (i-v)

テクニック:
  - 2つのコードだけで8-16小節
  - ベースラインで変化をつける
  - パッドは非常にゆっくり変化
  - Reverb を深めに（Decay 4-6秒）
```

**パターン3: ダーク進行**:
```
Am - Bdim - Dm - E (i-ii°-iv-V)

テクニック:
  - ディミニッシュコードで不安感
  - V（E Major）のG#がマイナーキーに緊張を追加
  - Slow Attack のパッド（500ms+）
  - 低い音域（C2-C4）で暗さを強調
```

### 15.2 Drum & Bass / Liquid DnB

高速BPM（170-180）でもコード進行は感動の核心です。

**パターン1: Liquid 進行**:
```
Fm - Db - Ab - Eb (iv-bII-bVI-bIII)
Key: C Minor 系

特徴:
  - 美しく、切ない
  - ピアノやストリングスパッド
  - 高速アルペジオとの組み合わせ

BPM: 174
```

**パターン2: ジャジーDnB**:
```
Dm9 - G13 - CMaj9 - A7b9 (ii9-V13-IMaj9-VI7b9)

特徴:
  - テンションコード多用
  - Rhodes、エレクトリックピアノ音色
  - カリバー、ロンドン・エレクトリシティ的

BPM: 174
```

### 15.3 Lo-Fi House

暖かく、ノスタルジックなサウンドが特徴のジャンルです。

**パターン1: Vinyl House**:
```
CMaj7 - Am7 - Dm7 - G7 (IMaj7-vi7-ii7-V7)

テクニック:
  - 全てのコードに7thを追加
  - Lo-Fi エフェクト（Bitcrusher、Vinyl Crackle）
  - デチューンを少し加える（+/- 5-10 cents）
  - テープサチュレーション
```

**パターン2: サンプルベース**:
```
古いソウル/ファンクのサンプルからコードを抽出:

1. サンプルのキーを分析（Mixed In Key等）
2. コード進行を耳コピ
3. 自分のシンセで再構築
4. Lo-Fi処理を追加

よく使われるサンプル元:
  - 70-80年代のソウル
  - ジャパニーズシティポップ
  - ブラジリアンミュージック
```

### 15.4 Hardstyle / Hard Dance

高エネルギーで分かりやすいコード進行が求められます。

**パターン1: エピック Hardstyle**:
```
Am - F - C - G (vi-IV-I-V)
Euphoric Hardstyle の王道

テクニック:
  - Supersaw（8-16ボイス、デチューン広め）
  - オクターブユニゾン（C3 + C4 + C5）
  - 非常にコンプレッション強め
  - BPM: 150-155
```

**パターン2: ダーク Hardstyle**:
```
Am - Bdim - E - Am (i-ii°-V-i)

テクニック:
  - ディミニッシュでダークさを強調
  - ディストーションリード
  - ベースがスクリーム音色
  - BPM: 150
```

### 15.5 Synthwave / Retrowave

80年代のシンセポップを現代に復活させたジャンルです。

**パターン1: ノスタルジック進行**:
```
Fm - Cm - Db - Ab (i-v-bVI-bIII)
Key: F Minor

テクニック:
  - Juno-106 風のポリシンセパッド
  - コーラスエフェクト（Ensemble）
  - ゲートリバーブ
  - BPM: 100-120
```

**パターン2: ドライブ進行**:
```
Am - F - C - E (i-VI-III-V)
Key: A Minor

テクニック:
  - アルペジオをメインに
  - ベースは FM シンセ（DX7的）
  - パッドはゆっくりしたアタック
  - サイドチェインは軽め
```

## 17. コード進行の高度な分析手法

### 17.1 機能和声分析（Functional Harmony Analysis）

コード進行を分析する際、各コードの「機能」に着目することで、なぜその進行が効果的なのかを理解できます。

**3つの基本機能**:
```
Tonic（T）: 安定、休息、帰着点
  → I, iii, vi

Subdominant（S）: 準備、移行、方向性
  → ii, IV

Dominant（D）: 緊張、解決への期待
  → V, vii°
```

**分析の手順**:
```
進行: Am - F - C - G

機能分析:
  Am (vi) = T  → 安定からスタート
  F  (IV) = S  → 移行
  C  (I)  = T  → 帰着
  G  (V)  = D  → 緊張で終わり、ループの動機

解釈:
  T → S → T → D → (T に戻る)
  安定→移行→帰着→緊張→（ループで安定に戻る）
  → だから気持ちよくループする
```

**Progressive House の分析例**:
```
進行: Am - F - C - G → Am - F - Fm - G

機能分析:
  Am (vi) = T
  F  (IV) = S
  C  (I)  = T
  G  (V)  = D
  Am (vi) = T
  F  (IV) = S
  Fm (iv) = S（借用、暗い色彩）
  G  (V)  = D

解釈:
  2周目で Fm（モーダルインターチェンジ）が入ることで
  同じS機能でも色彩が変わり、2周目に新鮮さを追加
```

### 17.2 ベースラインの動き分析

コード進行のベースラインの動きを分析することで、進行の滑らかさやエネルギーを評価できます。

**ベースラインの動き分類**:
```
Step Motion（段階的移動）:
  C → D → E → F（2度ずつ）
  → 非常にスムーズ

Skip（跳躍）:
  C → E → G（3度以上）
  → やや動きがある

Leap（大跳躍）:
  C → G（5度以上）
  → エネルギッシュ

Pedal（ペダル）:
  C → C → C → C（同一音）
  → 安定、催眠的
```

**進行のベースライン分析例**:
```
I-V-vi-IV (C-G-Am-F):
  ベース: C → G → A → F
  動き:   5度↑ → 2度↑ → 3度↓
  → 最初にLeap、その後は比較的スムーズ

ii-V-I (Dm-G-C):
  ベース: D → G → C
  動き:   4度↑ → 4度↑
  → 4度上行の連鎖（Circle of 5ths）
  → 最も自然な解決
```

### 17.3 テンションの解決分析

テンションノートがどのように解決されるかを分析します。

```
CMaj9 → Dm7 での 9th の解決:

CMaj9: C - E - G - B - D
                        ↑ 9th (D)
Dm7:   D - F - A - C
       ↑ D が Root に解決

→ 9th が次のコードの Root として解決
→ 非常にスムーズな接続
```

```
G7 → CMaj7 でのトライトーンの解決:

G7:     G - B - D - F
              ↑       ↑
              B と F はトライトーン（増4度）
CMaj7:  C - E - G - B
            ↑
            B は維持、F → E に半音下降

→ トライトーンが解決して安定
→ ii-V-I の V→I が気持ちよい理由
```

## まとめ

### コード進行の核心

1. **ダイアトニックコード**: I, ii, iii, IV, V, vi, vii°
2. **定番進行**: I-V-vi-IV が最強
3. **ボイシング**: スムーズな声部進行
4. **リズム**: 4つ打ち、Offbeat、シンコペーション
5. **ミキシング**: サイドチェイン必須
6. **テンション**: 9th、11th、13th で色彩を追加
7. **モーダルインターチェンジ**: 借用和音で意外性を演出
8. **転調**: ビルドアップ→ドロップで半音/全音上げ
9. **オートメーション**: フィルター、Velocity、ボイシングの動的変化

### DJから制作者へ

**DJスキル**:
- ハーモニックミキシングでキーの理解
- ジャンル別のコード進行を体感
- マイナー/メジャーの雰囲気を知っている

**プロデューサーとして**:
- 感情を設計できる
- 無限のハーモニーを作成
- 音楽理論で差別化
